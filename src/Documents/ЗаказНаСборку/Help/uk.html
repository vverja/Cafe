Замовлення на збирання (розбирання)

Документ призначений для планування здійснення наступних видів операцій:

  • Збирання з комплектуючих.
  • Розбирання на комплектуючі.

В залежності від обраної операції документ фіксує потреби в комплектуючих (для збирання), або потреби в комплектах (для розбирання).

Можливість використання в системі замовлень на збирання (розбирання) визначається функціональною опцією Замовлення на збирання (розбирання) в розділі НДІ і адміністрування - Склад і доставка - Внутрішній рух товарів.

Замовлення на збирання (розбирання) може бути створене вручну. Замовлення на збирання комплектів також може бути сформоване автоматично в робочому місці Формування замовлень за потребами. Необхідність в забезпеченні потреб у номенклатурі на складі за допомогою збирання комплектів визначається в робочому місці Параметри забезпечення потреб з використанням способу забезпечення потреб з типом забезпечення - Збирання.

Замовлення на збирання (розбирання) може бути оформлене в рамках конкретної угоди, а також по конкретному підрозділу (або деталізуватися по менеджерам підрозділу). Дана можливість регулюється функціональними опціями: Відокремлений облік собівартості товарів за видами запасів, По угодам, По підрозділах або менеджерам в розділі НДІ і адміністрування- Фінансовий результат і контролінг- Облік товарів.

Виконання замовлення

Рішення про необхідність забезпечити, зарезервувати або відвантажити комплектуючі для виконання збирання приймається шляхом вказівки Дії для окремих комплектуючих замовлення. Для виконання розбирання аналогічно приймається рішення про необхідність забезпечити, зарезервувати або відвантажити розібрані комплекти шляхом вказівки Дії.

Виконанням замовлення на збирання (розбирання) є документ Збирання товарів. Відвантаження комплектуючих для збирання (комплектів для розбирання) по замовленню на збирання (розбирання) з ордерного складу здійснюється документом Видатковий ордер на товари. Приймання зібраних комплектів (комплектуючих при розбиранні) по замовленню на збирання (розбирання) на ордерний склад здійснюється документом Прибутковий ордер на товари.

Використання статусів

Контроль процесу обробки документів може здійснюватися за допомогою статусів.
Використання статусів визначається функціональною опцією Статуси замовлень на збирання (розбирання) в розділі НДІ і адміністрування - Склад і доставка - Статуси документів.
Її установка робить доступними наступні статуси:

  • До забезпечення дозволяє почати забезпечення замовлення а також зарезервувати комплектуючі для збирання, або комплекти для розбирання на складі.
  • До виконання дозволяє почати відвантаження комплектуючих для збирання на замовлення, або комплектів для розбирання по замовленню.
  • Закритий дозволяє проконтролювати завершення робіт по замовленню.

Встановити статус Закритий в замовленні на збирання (розбирання) можна при дотриманні наступних умов:

  • повністю оформлені документи збирання товарів по замовленню,
  • комплектуючі для збирання (комплекти для розбирання) повністю відвантажені при використанні ордерної схеми при відвантаженні на складі (оформлені всі видаткові ордери на товари за замовленням),
  • зібрані комплекти (комплектуючі при розбиранні) повністю прийняті при використанні ордерної схеми при надходженні на складі (оформлені всі прибуткові ордери на товари за замовленням).

Можливість використання статусу Закритий у документі визначається функціональною опцією Не закривати частково зібрані (розібрані) замовлення на збирання (розбирання) в розділі НДІ і адміністрування - Склад і доставка - Внутрішній рух товарів.

Стани замовлення

Замовлення на збирання (розбирання) може перебувати в одному з наступних станів:

  • Готовий до забезпечення - в документі встановлені дії До забезпечення або Не забезпечувати, вважається що замовлення не забезпечене комплектуючими для збирання (комплектами для розбирання) на складі;
  • Очікується забезпечення - в документі встановлені дії Резервувати до дати (майбутньої дати) або Забезпечувати відокремлено (але товар ще не надійшов), вважається що замовлення не забезпечене наявністю комплектуючих для збирання (комплектів для розбирання) на складі;
  • Готовий до збирання (розбирання) - у документі встановлені дії Резервувати на складі, Резервувати до дати (і дата настала), Забезпечувати відокремлено (і товар надійшов), вважається що розпорядження на відвантаження комплектуючих для збирання (комплектів для розбирання) по замовленню зі складу не видано;
  • У процесі збирання (розбирання) - у документі встановлені дії Відвантажити або Відвантажити відокремлено, статус До виконання (якщо статуси використовуються) та по замовленню не повністю оформлені видаткові ордери, прибуткові ордери і накладні на збирання (розбирання), вважається що збирання (розбирання) за замовленням не завершено;
  • Готовий до закриття- накладні та ордери на замовлення повністю оформлені, використовується контроль закриття замовлень, в документі встановлений статус До виконання, вважається що виконання замовлення не завершено;
  • Закритий- виконання замовлення завершено.
