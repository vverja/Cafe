Для просмотра показателей и отчетов на мобильных устройствах предназначено приложение Монитор ERP. Поддерживается подключение к конфигурациям "BAS ERP", "BAS Комплексное управление предприятием", "BAS Управление торговлей".

Возможности приложения

  • быстрый просмотр состояния целевых показателей в виде диаграмм и детальных отчетов;
  • расшифровка целевых показателей с помощью отчетов-расшифровок;
  • просмотр отчетов;
  • управление контактной информацией партнеров;
  • просмотр досье партнеров.

Инструкция по установке приложения Монитор ERP

Ссылки на приложение будут доступны позже.

Для того, чтобы приложение могло подключиться к информационной базе, администратору (пользователю с полными правами) необходимо:

  1. Опубликовать web-сервис ERPMonitor, входящий в состав конфигурации.
     
  2. Создать в справочнике Группы доступа новую группу для поставляемого профиля Пользователь мобильного приложения "Монитор ERP" (дополнительный);
  3. Добавить пользователей, желающих использовать мобильное приложение Монитор ERP в созданную группу доступа.

При первом запуске приложения на мобильном устройстве, пользователю будет предложено ознакомиться с приложением на демонстрационных данных. Для работы с данными информационной базы необходимо отключить режим ознакомления, если такой режим был включен, и указать реквизиты синхронизации:

  • Пользователь - имя пользователя, используемое для работы с информационной базой;
  • Адрес сервера - необходимо запросить у администратора информационной базы;
  • Каталог - необходимо запросить у администратора информационной базы; 
  • При начале синхронизации приложение запросит пароль - нужно указать пароль, используемый для работы с информационной базой. 

Приложение готово к работе!

Внимание! Для первой синхронизации рекомендуется WiFi соединение.
