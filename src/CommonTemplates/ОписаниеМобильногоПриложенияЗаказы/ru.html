Мобильное приложение Заказы предназначено для менеджеров по продажам или торговых представителей, которым необходимо принимать заказы от клиентов мобильно, вне офиса. Поддерживается подключение к конфигурациям "BAS ERP", "BAS Комплексное управление предприятием", "BAS Управление торговлей".

Возможности приложения

  • прием заказов на товары и услуги от клиентов;
  • регистрация клиентов и информации о них;
  • регистрация оплаты от клиентов;
  • просмотр контактной информации и досье клиентов.

Инструкция по установке приложения Заказы

Ссылки на приложение будут доступны позже.

Для подключения приложения к информационной базе администратору (пользователю с полными правами) нужно:

  1. Опубликовать web-сервис CustomerOrdersExchange, входящий в состав конфигурации.
     
  2. Если пользователь, желающий использовать мобильное приложение Заказы, не входит ни в одну из групп пользователей с профилем доступа:
    - торговый представитель;
    - менеджер по продажам;
    - супервайзер торговых представителей,
    необходимо создать профиль и группу доступа.
    Для этого надо создать в справочнике Группы доступа новую группу для профиля, созданного по роли Использование мобильного приложения "Заказы". Добавить пользователя в созданную группу доступа.
     
  3. Настроить синхронизацию для пользователей в разделе НСИ и администрирование — Продажи — Настройки использования мобильного приложения "Заказы"

При первом запуске приложения на мобильном устройстве нужно выбрать режим работы:

  • Режим синхронизации — в этом режиме  потребуется указать информацию для подключения:
    • Адрес сервиса обмена — необходимо запросить у администратора информационной базы;
    • Пользователь — имя пользователя, используемого для работы с информационной базой;
    • При начале синхронизации приложение запросит пароль — нужно указать пароль, используемый для работы с информационной базой.
  • Автономный режим — в этом режиме работа приложения не поддерживает синхронизацию данных с информационной базой, приложение является полностью автономным.

Приложение готово к работе!

Внимание! Для первой синхронизации рекомендуется Wi-Fi соединение.
