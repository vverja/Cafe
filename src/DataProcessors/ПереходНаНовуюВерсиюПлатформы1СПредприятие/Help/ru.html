Обработка представляет собой помощник для автоматического получения из сервиса Интернет-обновлений и установки на компьютере пользователя требуемой версии платформы.

Информация о доступной версии платформы

Отображается при открытии и содержит информацию о:

  • Номере актуальной версии платформы для этой версии программы;
  • Размере дистрибутива платформы, который будет загружен на компьютер;
  • Информацию об особенностях перехода на новую версию платформы;

Также может отображаться сообщение "Рекомендуется перейти на новую версию платформы", если разработчик программы рекомендует использовать для работы с программой более новую версию платформы по причине обнаружения ошибки критичной в платформе или подготовки к обновлению на новую версию программы;

На этом шаге необходимо ознакомиться с указанной информацией и нажать "Далее >".

Подключение к порталу ИТС

Получение дистрибутивов платформы доступно только зарегистрированным пользователям портала ИТС.

  • Логин - логин пользователя портала ИТС;
  • Пароль - пароль пользователя портала ИТС;
  • Запомнить пароль - если флаг установлен, тогда логин и пароль будут сохранены в программе и их повторный не потребуется при следующем получении обновления.


Введите логин и пароль пользователя портала ИТС и нажмите "Далее >".

Режим установки платформы

Определяет режим установки дистрибутива платформы.

  • Установить с настройками по умолчанию - установить новую версию платформы в "тихом" режиме без отображения программы установки платформы;
  • Установить с ручными настройками - установить новую версию платформы в полном интерактивном режиме. Отображается программа установки платформы, доступен выбор всех необходимых параметров установки. Необходимо следовать инструкциям программы установки платформы, после завершения установки выполнение будет автоматически продолжено.

     

    Каталог хранения дистрибутивов платформы

    Рекомендуется копировать полученные дистрибутивы платформы в каталог на локальном компьютере или сетевом окружении, если в дальнейшем планируется также установить новую версию платформы на другой компьютер.

    • Сохранять полученные дистрибутивы платформы в каталог - если флаг установлен, тогда полученные дистрибутивы платформы будут автоматически сохраняться в выбранном каталоге - в подкаталогах с именем, соответствующим номеру новой версии платформы; если флаг не установлен, тогда на компьютере сохраняется только последний полученный дистрибутив платформы.


    Выберите необходимый вариант копирования дистрибутивов платформы и нажмите "Далее >".

    Получение и установка новой версии платформы

    После выбора каталога для сохранения дистрибутивов и нажатия "Далее >" отображается предупреждающая надпись "Пожалуйста, подождите...", начинается получение файла дистрибутива новой версии платформы. Получение файла может занять от нескольких минут до нескольких часов в зависимости от размера обновления, скорости подключения к Интернету и производительности компьютера.
    После успешного получения файла дистрибутива открывается окно программы установки. Следуйте инструкциям программы установки в открывшемся окне. После завершения установки выполнение будет автоматически продолжено.

    Завершение установки платформы

    После успешного завершения установки новой версии платформы отображается информация об установленной версии платформы:

    • Номер установленной версии;
    • Каталог, в которую скопирован полученный дистрибутив платформы;
    • Флажок "Автоматически создать резервную копию и открыть программу на новой версии платформы" - если флажок установлен, тогда при нажатии "Готово" откроется механизм для автоматического создания резервной копии информационной базы, при этом после создания резервной копии программа автоматически откроется на новой версии платформы.

     

    Перед открытием программы на новой версии платформы необходимо:

    • завершить работу всех активных пользователей программы;
    • завершить текущий сеанс работы с программой;
    • установить новую версию платформы на компьютеры других пользователей программы;
    • открыть программу на новой версии платформы;
