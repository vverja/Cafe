Відкривається за допомогою кнопки Додати з вкладки Оформлення настройок варіанту звіту.

Для підтвердження настройок умовного оформлення натисніть кнопку ОК.

Оформлення

  • На вкладці введіть Заголовок елемента оформлення.
  • За допомогою прапорців можна вмикати або вимикати різні параметри оформлення.
  • Параметри оформлення можна змінювати за допомогою відповідних інструментів, наприклад, Колір фону можна змінити за допомогою кнопки , відкривши палітру кольорів і вибравши потрібний колір замість чорного, який проставлений по умовчанню.

Умова

  • На вкладці можна задати відбір, при якому дане умовне оформлення буде застосовуватися.
  • За допомогою відповідної кнопки можна Додати відбір.
  • Для зміни настройок відборів звіту застосовується таблиця, що складається з колонок:
  • Поле - перерахування полів, за якими проводиться відбір;
  • Умова - вибір операції для відбору по кожному полю;
  • Операції Дорівнює, Не дорівнює, В групі, Не в групі припускають вибір одного значення (одиничний відбір).
  • Операції В списку, В групі зі списку, Не в списку, Не в групі зі списку припускають вибір декількох значень (множинний відбір).
  • Для операцій Заповнено і Не заповнено не потрібно вибір значення.
  • Значення - можна вказати одне або кілька значень кожного поля для відбору в залежності від операції.
    • Для введення значення можна двічі клацнути мишею по полю.
    • Введіть значення параметра за допомогою кнопки Вибрати. Увімкніть прапорці поруч з параметрами, за якими необхідно провести відбір. Можна задіяти всі параметри для відбору або частину, включаючи або вимикаючи прапорці.
    • Для вибору декількох значень передбачена кнопка Підбір. Відкрийте список, виберіть кілька значень зі списку. Увімкніть прапорці поруч з потрібними для відбору параметрами. Введіть значення параметрів для відбору або виберіть зі списку.

Оформлювані поля

  • На вкладці з допомогою кнопки Додати виберіть одне або кілька полів, для яких буде застосовуватися умовне оформлення.

Додатково

На вкладці за допомогою прапорців вкажіть, де Показувати умовне оформлення:

  • В групуванні;
  • В ієрархічному групуванні;
  • У загальному підсумку;
  • У заголовку полів;
  • У заголовку;
  • У параметрах;
  • У відборах.

Див. також:

  • Робота зі звітами.
