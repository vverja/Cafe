Призначений для ведення користувачів, які працюють з програмою.

Відкрити список користувачів можна з розділу Адміністрування - Настройки користувачів і прав або Управління доступом з допомогою команди Користувачі.

Ведення списку користувачів може здійснювати як адміністратор, так і відповідальний за список зовнішніх користувачів.

Вхід в програму дозволяє тільки адміністратор.

Відповідальний за список (з правом додавання нових користувачів) може виконувати обмежене ведення списку користувачів і настройку користувача до дозволу входу в програму адміністратором. Відповідальний за список може редагувати списки Користувачі і Групи користувачів.

Всім користувачам доступний тільки перегляд відомостей про себе, а також зміна ряду властивостей, таких як: пароль, мова програми та контактна інформація.

Користувач, якого ввів відповідальний за список, отримує статус Підготовлений. Адміністратор програми може відібрати таких користувачів за цією ознакою, перевірити правильність заповнення відомостей і дозволити вхід в програму.

Після того як адміністратор дозволить вхід в програму, відповідальний за список більше не зможе змінити настройки для входу, крім відключення дозволу на вхід в програму.

У розподіленій інформаційній базі вхід в програму настроюється окремо для кожної з інформаційних баз програми.

У разі великої кількості користувачів список може бути згрупований. Наприклад, групи користувачів можна використовувати для ведення структури підрозділів компанії.

Команди для роботи зі списком користувачів також доступні в контекстному меню по правій кнопці миші.

Введення користувача

  • Натисніть Створити, заповніть необхідні поля.

Перегляд недійсних користувачів

  • Користувачі можуть бути відмічені як недійсні. Для того щоб переглянути таких користувачів, увімкніть прапорець Показувати недійсних користувачів внизу списку.

Перегляд користувачів нижчестоящих груп

  • Для того щоб побачити всіх користувачів, включених в дочірні групи з урахуванням ієрархії, необхідно встановити прапорець Показувати користувачів нижчестоящих груп в нижній частині списку. По умовчанню прапорець включений.

Установка пароля

  • Натисніть Ще - Встановити пароль, напишіть пароль.

Перегляд списку користувачів інформаційної бази

  • Для контролю правильності введення або у випадку некоректного введення користувачів по команді Ще - Користувачі інформаційної бази можна перейти до контролю записів і синхронізації списку користувачів зі списком користувачів інформаційної бази.

Групування користувачів

  • У разі якщо в програмі дозволено використовувати групи користувачів, список складається з двох частин: у лівій частині необхідну кількість груп, в правій - список користувачів виділеної групи. По умовчанню поточною є група Всі користувачі, тобто видно всіх користувачів програми.
  • Один і той же користувач може одночасно входити в різні групи. Включити користувача в групу можна різними способами:
    • При введенні користувача у списку можна вибрати групу, потім натиснути кнопку Створити.
    • Для додавання в групу великого числа користувачів можна відкрити вікно групи і натиснути Підібрати. При цьому для підбору використовується список Користувачі, що складається з трьох частин.
    • Виділіть користувача в списку, виконайте команду меню Ще - Призначити групи, за допомогою прапорців увімкніть його в одну або кілька груп. Також можна знайти цю команду в контекстному меню по правій кнопці миші.
    • Перетягніть одного або декількох користувачів мишею в потрібну групу. Підтвердіть свій вибір, натиснувши кнопку Так. Для використання цієї можливості прапорець Показувати користувачів нижчестоящих груп необхідно відключити.

Введення групи користувачів

  • Для введення групи натисніть Створити групу.
  • Також можна скористатися командою Створити контекстного меню по правій кнопці миші в лівій частині списку.

Включення групи користувачів в іншу групу

  • Виберіть групу в лівій частині списку, скористайтеся командою контекстного меню по правій кнопці миші Перемістити до групи.
  • Виберіть групу для переміщення.

Підбір учасників групи користувачів

  • У разі якщо зі списку користувачів проводиться підбір учасників групи, він складається з трьох частин. Зліва у верхній частині списку відображаються групи, в нижній частині користувачі обраної групи. У правій частині списку відображаються вибрані користувачі.
  • Для зручності підбору у верхній лівій частині списку показані вже наявні групи користувачів. Вибравши потрібну групу (або групу Всі користувачі) , можна переглянути користувачів цієї групи в нижній лівій частині списку.
  • Виділіть одного або декількох користувачів в лівій нижній частині списку, за допомогою стрілки перемістіть в праву частину Вибрані користувачі. Також можна перенести користувача з лівої нижньої частини в праву за допомогою подвійного натиснення кнопки миші.
  • Для того щоб скасувати свій вибір, приберіть одного або декількох непотрібних користувачів з правої частини списку аналогічними діями.
  • За допомогою подвійної стрілки < < можна прибрати з Обраних відразу всіх користувачів.
  • Програма веде підрахунок обраних користувачів.
  • Після закінчення підбору учасників групи користувачів натисніть кнопку Завершити і закрити.

Перегляд звіту Відомості про користувачів

  • Звіт дозволяє виконати аналіз параметрів для входу (властивостей користувача ІБ) спільно з іншими властивостями користувача в довіднику. Включає 3 варіанти звіту - про користувачів, зовнішніх користувачів і всіх користувачів програми.
  • Натисніть Ще - Відомості про користувачів, сформуйте звіт.
