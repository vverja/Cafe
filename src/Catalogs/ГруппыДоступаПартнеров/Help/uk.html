Групи доступу партнерів

Довідник призначений для опису доступу до елементів довідника Партнери по виду доступу Групи партнерів. Довідник використовується в тому випадку, якщо необхідно обмежити доступ користувачам (або групі користувачів) до конкретних партнерів.

Приклад. Менеджери з продажу на підприємстві працюють з різними партнерами підприємства. Одні менеджери працюють з вітчизняним партнерами, а інші із зарубіжними партнерами. Менеджерам повинна бути доступна інформація тільки по тим партнерам, з якими вони працюють і не повинна бути доступна інформація по партнерам інших менеджерів. У довідник Групи доступу партнерів вводиться інформація про всі ті групи партнерів, з якими працюють конкретні менеджери: Вітчизняні партнери та Зарубіжні партнери. У довідник групи доступу користувачів додається інформація про ті групи менеджерів, які працюють з певною групою партнерів: Менеджери по роботі з вітчизняними партнерами і Менеджери по роботі із зарубіжними партнерами. В якості дозволеної групи доступу партнерів вказується відповідна група партнерів: Вітчизняні партнери та Зарубіжні партнери.

Для менеджерів буде обмежений список доступних їм партнерів:

  • Менеджери по роботі з українськими партнерами будуть бачити інформацію тільки по тим партнерам, для яких визначена група доступу партнерів Українські партнери
  • Менеджери по роботі із зарубіжними партнерами будуть бачити інформацію тільки по тим партнерам, для яких визначена група доступу партнерів Зарубіжні партнери.

Встановити обмеження за групами доступу партнерів можливо тільки в тому випадку, якщо в розділі НДІ і адміністрування - Настройки користувачів і прав - Групи доступу встановлені прапори Обмежувати доступ на рівні записів і Групи доступу партнерів. В профілі груп доступу користувачів (наприклад, менеджерів з продажу) має бути доданий вид доступу Група доступу партнерів.
