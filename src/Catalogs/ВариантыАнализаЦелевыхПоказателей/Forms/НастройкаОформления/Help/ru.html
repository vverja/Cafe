Настройка оформления

Настройки закладки Цвета определяют цветовое оформление ключевых элементов диаграмм, таких как линии значений показателя, линии целевых значений, границ отклонений и прогноза.

Настройки закладки Цвета покомпонентного сравнения:

  • Только основной цвет - при оформлении гистограмм сравнения будет использоваться цвет, заданный в Значение показателя (на закладке Цвет). Если настройка отключена, то будет использоваться оформление диаграмм с умолчаниями платформы.
  • Градиент основного цвета - при оформлении гистограмм светлость цвета будет изменяться в зависимости от номера столбика.
  • Выделять максимальное значение - максимальное значение будет выделено отличающимся от основного цветом.


Настройки закладки Дополнительно (состав зависит от типа анализа):

  • Отображать легенду - будет выводиться легенда в нижней части диаграммы.
  • Выводить подписи к диаграммам - названия серий данных будут выводиться рядом со столбцами.
  • Окантовка диаграмм - серии данных будут обведены тонкой черной линией.
  • Сглаживание графика - линии графика будут выведены в виде гладкой кривой.
  • Маркировать точки прогноза - точки прогноза будут маркированы круглым маркером.
