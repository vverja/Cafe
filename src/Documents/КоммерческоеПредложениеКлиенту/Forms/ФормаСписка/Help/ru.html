Коммерческие предложения клиентам

Документ предназначен для регистрации в системе формального предложения (условий продаж) клиенту о покупке товаров.
Коммерческое предложение может быть создано на основе анализа первичного спроса, который регистрируется при первичном контакте с клиентом. Первичный спрос клиента может быть зарегистрирован в сделке с клиентом.

Использование коммерческого предложения регулируется функциональной опцией Коммерческое предложение в разделе НСИ и администирирование - Продажи - Оптовые продажи. Коммерческое предложение обязательно используется при включенной функциональной опции Управление сделками.

Групповое изменение статуса
  • В списке коммерческих предложений отметьте те предложения, по которым необходимо изменить статус, используя стандартные возможности платформы.
  • Нажмите на кнопку Установить статус.
  • В появившемся списке выберите один из статусов: Не согласовано, Действует, Отменено.
  • У отмеченных коммерческих предложений будет установлен выбранный статус.
