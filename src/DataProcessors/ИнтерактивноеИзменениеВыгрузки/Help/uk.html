Призначена для відбору даних, які будуть відправлені при синхронізації даних в іншу програму.

Відкрити список можна, натиснувши на посилання в полі З відбором для додавання даних на сторінці Дані для відправлення Помічника інтерактивної синхронізації даних.

Дані, що відправляються

  • Натисніть Підібрати, для того щоб вибрати дані для відправлення і заповнити список. Об'єкти програми для вибору виводяться у вигляді дерева. Виділіть натисніть Вибрати. Дані розміщуються в списку для відправлення.
  • Для вилучення даних зі списку натисніть Ще - Вилучити.
  • Список даних, що відправляються, складається з колонок:
    • Дані для відправлення - вибрані для відправлення об'єкти програми перераховуються в цій колонці;
    • Відбір - по умовчанню обрані Всі об'єкти (посилання). Натисніть посилання, для того щоб відібрати потрібні елементи обраних списків.
    • До відправлення/всього - підраховується кількість обраних елементів і загальна кількість елементів списків. Натисніть на посилання Буде відправлено внизу списку, для того щоб перерахувати вибрані елементи.
  • Натисніть OK для підтвердження свого вибору і повернення у Помічник для продовження настройки синхронізації даних.

Перегляд складу даних вивантаження

  • Натисніть Склад даних, для того щоб переглянути Відомості про склад даних, що відправляються.

Збереження настройок вивантаження та їх використання

  • За допомогою кнопки Настройки можна зберегти введені настройки для відбору даних або використовувати раніше збережені. Програма пропонує використовувати попередні настройки, наприклад збережену автоматично, для цього виберіть із запропонованого списку значення Останнє відправлення. Можна вибрати зі списку раніше збережену настройку. Можна зберегти свої настройки. Для цього натисніть кнопку Зберегти поточну настройку. Напишіть найменування настройки, що зберігається, натисніть Зберегти.

Підрахунок кількості об'єктів, що відправляються

  • У нижній частині списку в поле Буде відправлено натисніть на посилання, для того щоб розрахувати кількість об'єктів, що відправляються. Після підрахунку в колонці списку До відправлення / Всього відповідно проставляється кількість елементів і посилання приймає значення підсумкової суми елементів, що відправляються.
