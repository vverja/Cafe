Програма надає можливість зберігання файлів довільного формату. На відміну від зберігання файлів на дисках або в корпоративних програмах документообігу, файли зв'язуються з даними програми. Так зручно зберігати файли, безпосередньо пов'язані з даними. Об'єктами-зберігачами файлів в програмі можуть бути різні списки, документи, бізнес-процеси і т. д. Робота з файлами, прикріпленими до об'єкта, здійснюється в списку Приєднані файли. Список можна відкрити відповідною командою на панелі навігації документа або іншого об'єкта програми. Існує також можливість зберігати файли без прив'язки до конкретного об'єкта у списку файлів.

Для роботи з реквізитами файлу (автор, назва, опис і т. д.) використовується картка файлу, а для роботи з самим файлом - відповідний додаток, встановлений на комп'ютері. При редагуванні файлу його попередня версія автоматично зберігається в інформаційній базі.

Робота з файлами

Для роботи з файлами використовуйте команди, розташовані на командній панелі чи в меню Ще в списку файлів, в контекстному меню по правій кнопці миші або картці файлу:

Команда Опис
Створити (Insert) Додає новий файл в програму. Можливі способи створення файлу: з шаблону, з файлу на диску, зі сканера.
Відкрити (Переглянути) Відкриває файл зовнішнім додатком для перегляду або редагування, залежно від того, позначений файл як зайнятий поточним користувачем для редагування.
Редагувати (F4) Позначає файл як зайнятий для редагування і відкриває його зовнішнім додатком.
Клавіша Enter (подвійне натиснення кнопки миші) По умовчанню необхідно вибрати, як відкрити файл - для перегляду або редагування. Дія настроюється в персональних настройках програми на закладці Робота з файлами.
Закінчити редагування Оновлює файл в сховище і знімає з нього позначку зайнятості для редагування.
Друк За допомогою кнопки можна роздрукувати файл.
Відправити Можна відправити поштове повідомлення, обраний файл при цьому автоматично буде його додатком.
Зберегти зміни Зберігає версію редагованого файлу в сховище, але залишає його зайнятим поточним користувачем для редагування
Зайняти Позначає файл в сховище як зайнятий поточним користувачем для редагування, але не відкриває зовнішній додаток. Приступити безпосередньо до редагування можна буде пізніше.
Скасувати редагування Знімає з файлу позначку зайнятості для редагування.
Відкрити картку Відкрити картку файлу, в якій зберігаються відомості про файл.
Попередній перегляд Увімкніть прапорець, для того щоб переглянути зображення у списку файлів.
Зберегти як Зберігає файл у вказаний каталог на жорсткому або мережевому диску.
Відкрити каталог файлу Відкрити папку в Провіднику Windows, в якій знаходиться файл.
Оновити з файлу на диску Оновлює файл в інформаційній базі обраним файлом на жорсткому диску. При цьому програма перевіряє дати файлів, при спробі замінити файл старою версією виводиться попередження, дія не виконується.
Підписати Створює електронний підпис для файлу. Перед створенням електронного підпису програма пропонує вибрати сертифікат електронного підпису з наявних на комп'ютері. Вибір сертифіката можна зробити і під час персональних настройок програми. Підписаний файл стає недоступним для редагування і відображається у списках зі значком . Підписаний файл можна скопіювати, при цьому програма задасть питання -копіювати файл з підписом або без.
Додати електронний підпис з файлу Додає вибраний файл підпису до підписів файлу. Команду можна використовувати, наприклад, якщо підпис (файл підпису) був отриманий пізніше, ніж підписаний файл.
Зберегти разом з електронним підписом Зберігає файл у вказаний каталог на жорсткому або мережевому диску, а також зберігає підписи файлу - всі або вибрані, залежно від настройки дій При збереженні даних в файл разом з електронним підписом (у персональних настройках програми, закладка Додатково) Команду можна використовувати, наприклад, якщо необхідно передати підписаний файл іншій особі.
Зашифрувавши Шифрування файлу. На відміну від підписання можуть використовуватися кілька сертифікатів, наприклад, якщо необхідно зашифрувати файл для декількох користувачів. Виберіть потрібні зі списку за допомогою прапорців. Зашифрований файл у списках відображається зі значком . На відміну від підписання пароль до закритого ключа в процесі шифрування вводити не потрібно. Зашифрований файл доступний для редагування. Шифруються всі версії файлу. Для зашифрованих файлів не працює повнотекстовий пошук. Зашифрований файл можна скопіювати. При цьому новий, скопійований файл залишиться зашифрованим. Зашифрований файл не можна підписати. Підписаний файл можна зашифрувати, при цьому перед перевіркою електронного підпису у зашифрованого файлу програма запросить пароль для попередньої розшифровки файлу.
Розшифрувати За командою відбувається розшифровка файлу. Для розшифровки потрібно ввести пароль закритого ключа.

Примітка: Якщо файл зашифрований, повнотекстовий пошук за його вмістом не працює.

Додавання файлів

Для додавання нового файлу в програму:

  • Натисніть Створити (клавішу Insert);
  • Виберіть спосіб створення:
    • З файлу на диску
      • Для того щоб прикріпити до об'єкту або помістити в папку вже наявний на комп'ютері файл, вкажіть шлях до файлу на диску, натисніть кнопку Відкрити.
      • Після цього програма збереже файл, автоматично створить і відкриє Картку файлу.
    • З шаблону
      • Виділіть шаблон файлу в списку вибору, потім натисніть кнопку Вибрати.
      • Програмою буде автоматично створена і відкрита Картка файлу.
    • Зі сканера 
      • Варіант створення файлу Зі сканера стає доступним, якщо до комп'ютера підключено сканер (більш докладно див. документацію до сканера) і на комп'ютері підключена компонента для роботи зі сканером. Також необхідно настроїти сканування.
      • Якщо всі умови дотримані і настройки виконані, то можна отримати файл зі сканера.
      • Для цього необхідно слідувати інструкціям Помічника сканування.
      • Після того як відсканований документ, виконайте команду Прийняти, після цього файлу автоматично присвоюється код і створюється картка файлу, яка виводиться на екран. У полі Найменування також автоматично проставляється код. Відредагуйте потрібні поля, натисніть Записати і закрити, для того щоб зберегти картку файлу.
