Заполнение этапов оплат

Этапы оплаты заполняются на основании данных о тех этапах платежей по заказу, которые зарегистрированы в указанном в заказе соглашении. Даты и суммы платежей рассчитываются на основании даты заказа и его суммы. В заказе заполняются все этапы оплаты: как авансовые платежи, так и этапы, которые касаются оплаты товаров в кредит (после поставки товаров по заказу).

Заполнение этапов оплаты по заказу необходимо производить только после того, как заказ с поставщиком будет согласован и будут определены суммы и товарный состав заказа. Для заполнения этапов оплаты необходимо нажать на кнопку Заполнить. Расчет дней отсрочки платежа производится с учетом или нет выходных дней в зависимости от настроек в графике оплаты (этапах оплаты).

Предусмотрена также возможность заполнения этапов оплаты в соответствии с теми этапами оплаты, которые были указаны в предыдущих заказах данного поставщика. Для этих целей используется команда Заполнить по предыдущим заказам.

Заполнение информации об этапах оплаты вручную

Заполнение этапов оплаты вручную рекомендуется проводить после полного согласования суммы заказа с клиентом.

  • Перейдите по ссылке Этапы оплаты.
  • Нажмите на кнопку Добавить.
  • В появившейся строке выберите вариант оплаты: Аванс (до подтверждения), Предоплата (до поступления) , Кредит (после поступления).
  • Укажите дату предполагаемого платежа, процент платежа.
  • Сумма платежа будет рассчитана автоматически в соответствии с суммой заказа и процентом платежа.
  • Аналогичным образом укажите другие этапы оплаты по заказу клиента.

Если информация об этапах оплаты в заказе не заполнена, то при проведении документа она будет заполнена автоматически или скорректирована в соответствии с указанным графиком (этапами оплаты в соглашении) и суммой документа. Если график не указан, то будет предложено в качестве этапа оплаты заполнить информацию о 100% кредите по заказу.

Редактирование порядка расчетов по документу.


Можно уточнить порядок расчетов по документу, если он отличается от указанного в соглашении или договоре.
Возможные варианты изменения:

  • по договорам --> по заказам или по накладным
  • по заказам --> по накладным
  • по накладным --> не изменяется

Редактирование порядка расчетов не разрешается в накладных, введеных по заказу. В этом случае порядок расчетов можно изменить в самом заказе.
