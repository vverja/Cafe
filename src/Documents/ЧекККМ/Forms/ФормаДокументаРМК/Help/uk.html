Оформлення ручних знижок


Після підбору товарів в документі касир може вказати для товару ручну знижку. Для цього призначена кнопка Ручні знижки.

Ручна знижка в чеку може бути введена по всіх позиціях в чеку або по конкретному товару. Ручна знижка може бути введена як процентна, або як сума.

Доступність ручних знижок визначається функціональною опцією Ручні знижки в продажах в розділі НДІ і адміністрування- CRM і маркетинг- Маркетинг.

Ручна знижка по всіх позиціях документа вводиться в окремому діалоговому вікні, яке відкривається при натисканні на кнопку Призначити ручну знижку. Для скасування ручних знижок використовується команда Скасувати ручні знижки.

Оформлення автоматичних знижок


Знижки, передбачені для роздрібного складу розраховуються автоматично. Для перегляду інформації про те, за якими умовами була зареєстрована та чи інша знижка, натисніть на гіперпосилання сумою розрахованої знижки на інформаційній панелі або виберіть відповідну команду в контекстному меню табличній частині Товари. Буде показаний список тих знижок які застосувалися для конкретного товару.

Розрахунок автоматичних знижок необхідно проводити обов'язково. Якщо знижки не розраховані, то чек не буде проведений.

Пробиття чека


Пробиття чека слід проводити після оформлення всіх операцій з оплати чека. Для пробиття чека на фіскальному реєстраторі натисніть на кнопку Пробити чек.

Друк товарного чека


Для друку товарного чека виконайте наступні дії:

  • Виберіть команду Товарний чек в меню, яке викликається по кнопці Інші операції.
  • З'явиться форма друку товарного чека.
  • Вкажіть кількість копій друку товарного чека.
  • Натисніть на кнопку Друк.
