Завантаження даних з файлу в списки і документи

Відкривається з документів і елементів списків за допомогою кнопки Завантажити з файлу.

З допомогою команди Завантаження даних з файлу для адміністратора є можливість завантаження даних в довільний список програми.

За допомогою кнопки виберіть, як будуть завантажуватися дані:

  • заповнення таблиці в програмі;
  • завантаження з зовнішнього файлу.

Заповнення таблиці

  • З допомогою кнопки Заповнення таблиці (включена по умовчанню) можна перейти до введення даних в таблицю для подальшого завантаження в документ безпосередньо в програмі.
  • Таблицю можна заповнити, скопіювавши дані в таблицю з зовнішнього файлу через буфер обміну.
  • В таблиці є обов'язкові для заповнення колонки. Вони перераховуються над таблицею.
  • За допомогою посилання Змінити бланк можна відкрити список полів таблиці для введення і за допомогою прапорців змінити її структуру. Необхідно враховувати, що поля, виділені в таблиці жирним шрифтом, є обов'язковими для заповнення і чіпати їх не можна.
    • Вкажіть за допомогою прапорців необхідні колонки. Можна настроїти найменування колонок, їх відображення і порядок проходження для спрощення завантаження.
  • Після заповнення таблиці для переходу до зіставлення даних натисніть Далі.

Завантаження з зовнішнього файлу

  • Натиснувши кнопку Із зовнішнього файлу, можна завантажити дані з раніше підготовленого файлу, збереженого на комп'ютері, або зберегти бланк для заповнення і заповнити дані пізніше.
  • З допомогою кнопки Зберегти бланк для заповнення можна зберегти таблицю для введення даних у файлі обраного формату. Необхідно враховувати, що в даному файлі є колонки, обов'язкові для заповнення. Виберіть шлях до папки на комп'ютері. Програма присвоює ім'я файлу по умовчанню.
  • Можна заповнити зовнішні файли для завантаження в документи. Заповніть файл за рекомендованими правилами (обов'язкові для заповнення поля виділені в таблиці жирним шрифтом).
  • З допомогою кнопки Завантажити таблицю з файлу можна завантажити дані з зовнішнього файлу. Виберіть шлях до папки на комп'ютері.

Зіставлення даних

  • Після завантаження даних програма здійснює автоматичне зіставлення даних. Якщо автоматичне зіставлення не відбулося, то дані виводяться для зіставлення вручну.
  • Для того щоб переглянути зіставлені або незіставлені дані, можна використовувати відбір за допомогою поля Показувати. Незіставлені дані позначаються червоним. За допомогою відповідної кнопки можна Дозволити неоднозначність.
  • Для того щоб зіставити дані, відкрийте відповідний список програми, знайдіть в ньому відповідне значення або введіть нове, потім натисніть Вибрати.
  • Крім обов'язкових полів, зіставлятися можуть також і необов'язкові. При завантаженні поле можна зіставити і включити в документ за допомогою кнопки Вибрати зі списку. Якщо потрібного елемента в списку немає, то його можна Створити за допомогою відповідної кнопки.
  • Після автоматичного або ручного зіставлення даних, у разі, якщо всі дані зіставлені, натисніть кнопку Завантажити дані. Після цього дані будуть завантажені в документ або список. Залишається лише заповнити відсутні поля.

Завантаження даних в довільний список програми (тільки для адміністратора)

  • Завантаження даних з файлу в довільний список адміністратору можна зробити за відповідною командою.
  • У цьому випадку для завантаження даних можна вибрати один зі списків програми. Можна завантажувати в списки контактну інформацію, а також додаткові реквізити та відомості.
    • Після вибору списку для завантаження даних натисніть кнопку Далі. Виберіть за допомогою кнопок, як будуть завантажуватися дані:
      • Програма пропонує таблицю для заповнення.
      • Також можна завантажити дані з зовнішнього файлу.

Зіставлення даних (тільки для адміністратора)

  • Заповнивши таблицю або завантаживши дані з файлу, натисніть кнопку Далі. Необхідно провести зіставлення даних.
  • Поведінку програми при зіставленні даних можна настроїти:
    • За допомогою перемикача вибрати, як чинити з Зіставленими елементами: Оновлювати або Пропускати (по умовчанню).
    • За допомогою перемикача вибрати, як чинити з Незіставленими елементами: Створювати (по умовчанню) або Пропускати.
  • Дані будуть автоматично зіставлені, якщо відповідності не буде знайдено, таблиця виводиться з пропозицією зіставити дані.
    • У деяких списках існує можливість автоматичного створення незіставлених даних. Такі дані позначаються написом <Новий>. Для автоматичного створення в списку виберіть Створювати за допомогою перемикача (вибрано по умовчанню).
    • Після автоматичного або ручного зіставлення даних, у разі, якщо всі дані зіставлені, натисніть кнопку Завантажити дані.
  • Після цього дані будуть завантажені в документ або список. Залишається лише заповнити відсутні поля. Результати завантаження виводяться на перегляд.
    • За допомогою поля Показувати можна вивести Всі, Нові, Оновлені і Пропущені елементи.
    • Кілька виділених рядків Зберегти в файл, підкоригувати і пізніше завантажити повторно.
    • У списку можна змінити введені елементи, якщо вони не влаштовують. Для зміни декількох виділених елементів натисніть кнопку Змінити реквізити.
    • В разі потреби увімкніть прапорець Відкрити довідник по кнопці "Готово".
  • Якщо в таблиці всі дані коректні, натисніть кнопку Готово для підтвердження. Якщо прапорець увімкнено, відкривається список із введеними елементами.
