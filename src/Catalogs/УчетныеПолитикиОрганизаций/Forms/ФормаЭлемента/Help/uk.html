Метод оцінки вартості товарів

Для вибору доступні наступні методи оцінки:

  • Середня (зважена) за місяць;
  • ФІФО (зважена оцінка);
  • ФІФО (змінна оцінка).

Метод ФІФО (змінна оцінка) доступний для вибору у випадку, якщо включено використання партійного обліку (розділ НДІ та адміністрування - Фінансовий результат та контролінг - Облік товарів)

Система оподаткування

Обирається одна із напередвизначених Систем оподаткування, яка визначає, платником яких основних податків є організація.
