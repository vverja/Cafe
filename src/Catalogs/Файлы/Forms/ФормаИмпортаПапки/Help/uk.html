Призначена для завантаження папки з диска на комп'ютері.

Відкривається зі списку файлів за командою Завантажити папку контекстного меню по правій кнопці миші після вибору папки у вікні Провідника Windows. У провіднику вкажіть шлях до папки, натисніть Вибір папки.

Заповніть необхідні для завантаження папки відомості:

  • Поле Каталог на диску заповнюється автоматично, при необхідності можна вибрати іншу папку.
  • У полі Папка автоматично проставляється обрана папка в списку файлів, в яку будуть завантажені файли з диска на комп'ютері. При необхідності можна вибрати іншу папку;
  • Для економії місця на диску увімкніть прапорець Вилучати файли після додавання (по умовчанню вимкнено);
  • Напишіть Опис для файлів, що створюються;
  • Увімкніть прапорець, для того щоб Зберігати версії (рекомендується, по умовчанню прапорець включений);
  • Для текстових файлів можна Вибрати кодування зі списку.
  • Для підтвердження натисніть Завантажити папку. Якщо вибрана для завантаження папка порожня, то програма виводить повідомлення "Немає файлів для додавання".
Також можна використовувати перетягування мишею для завантаження папки з жорсткого або мережевого диска - просто перетягніть папку мишкою з Провідника Windows в потрібну папку списку файлів програми.
