Набори упаковок

Довідник призначений для ведення списку упаковок, які використовуються для обліку при надходженні і відвантаженні товарів.

Доступність довідника визначається функціональною опцією Упаковки номенклатури в розділі НДІ та адміністрування - Номенклатура - Розрізи обліку.
Упаковки обов'язкові до використання, якщо на складі (приміщенні) використовується адресне зберігання товарів у комірках.

Приклад. Підприємство торгує цукерками (в коробках) і молочними продуктами (йогуртами). Цукерки можуть надходити в упаковках по 10 коробок або в ящиках по 25 коробок. Йогурти можуть надходити в упаковках по 4 штуки або в ящиках по 20 штук. У довіднику Набори упаковок слід ввести два набори упаковок Цукерки (в коробках) і Йогурти. Для кожного набору упаковок слід визначити свій список упаковок.

Для кожного набору упаковок вводиться та одиниця вимірювання, щодо якої буде зазначатися кількість для упаковки. У наведеному прикладі такою одиницею вимірювання для набору упаковок Цукерки (в коробках) буде коробка, а для набору упаковок Йогурти такою одиницею вимірювання буде штуки. Після введення інформації про назву набору упаковки та одиниці вимірювання необхідно записати елемент і тільки потім перейти до заповнення списку упаковок, що входять в набір (Упаковки). Список упаковок заповнюється в довіднику Упаковки. Для кожної упаковки вказується кількість в одиницях вимірювання, зазначених для набору упаковки, вага і розмір упаковки.

Передбачена можливість використання вкладеної структури упаковок. У цьому випадку для кожної упаковки вказується з якої кількості дрібніших упаковок вона може складатися. Причому упаковки слід створювати у виді дерева упаковок і для кожної вказувати з яких упаковок вона може складатися.

Для кожної позиції номенклатури визначається набір упаковок. Якщо для позиції номенклатури використовуються індивідуальні упаковки, то з довідника Набори упаковок вибирається напередвизначений елемент Індивідуальний набір упаковок. При використанні індивідуального набору упаковок, список використовуваних для товару упаковок викликається з панелі навігації форми номенклатури.

Для кожної упаковки можна задати інформацію про ту тару, яка для неї використовується. Тара буде автоматично підставлятися при виборі упаковки в документі. У тому випадку, якщо використовується дерево упаковок, то буде поставлятися інформація про тару всіх тих упаковок, які підпорядковані даній упаковці.

Взуття поставляється в палетах ( на піддонах). На кожному піддоні лежить 10 ящиків, в кожному ящику по 5 пар взуття. Для палети визначена тара - піддон, для ящика - картонний ящик. При відвантаженні взуття в палетах програма запропонує для заповнення в якості зворотної тари не тільки піддон, але і картонні ящики.
