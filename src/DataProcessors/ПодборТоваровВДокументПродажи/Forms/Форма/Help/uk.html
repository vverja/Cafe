Підбір товарів в документ продажу

Робоче місце призначене для підбору товарів в табличну частину документа. У діалоговому вікні підбору можна встановити різні фільтри для обмеження області пошуку товарів: за ієрархією номенклатури, за видом номенклатури, за товарами іншої якості. Відповідно до обраного варіанту відбору буде показаний список груп номенклатури, список видів номенклатури або поле для вибору товару вихідної якості.

У лівій частині діалогового вікна відображається перелік товарів, що належать зазначеній у правій частині групі номенклатури, виду або товару іншої якості.
Товари для яких є товари іншої якості відзначені в списку коричневою рискою.

Особливості робочого місця:

  • При використанні фільтра за товарами іншої якості в полі товар можна вибрати номенклатуру, для якої ведуться товари з якістю відмінним від нового. Також встановити фільтр і вихідний товар можна по кнопці контекстного меню Товари іншої якості.
  • При підборі товарів можна використовувати швидкий відбір за видом і властивостями номенклатури. Список параметрів, які використовуються для швидкого відбору налаштовується для кожного виду номенклатури. Для реалізації такого відбору необхідно встановити фільтр за: видом і властивостями і вибрати потрібний вид номенклатури. У діалоговому вікні буде показаний список тих параметрів, які вказані в настройках швидкого відбору для обраного виду номенклатури. Для відбору потрібного товару потрібно встановити прапорці для потрібних значень параметрів.
  • Для кожного товару показуються ціни по тому виду ціни, який вказаний в документі і залишки товарів на тому складі, який вказаний в документі. Для перегляду інших цін по товару використовуйте кнопку Ціни.
  • Передбачена можливість установки відбору за ціною, задавши діапазон ціни від і до.
  • Для підбору товарів які є в наявності потрібно активізувати прапор Тільки в наявності на складі з документа.
  • Для установки відбору по сегменту номенклатури виберіть сегмент зі списку (підбір по сегменту можливий тільки при включеній функціональної опції Сегменти номенклатури в розділі НДІ і адміністрування- CRM і маркетинг - Маркетинг).
  • При підборі товарів можна показувати список залишків товарів на складах або відключити цей список (гіперпосилання Детальна інформація про залишки(приховати)). Інформацію по залишкам можна деталізувати по датах і підбирати товари прямо з цієї таблиці.
  • Підбір товарів може бути здійснено з запитом кількості і ціни або без запиту кількості і цін (кнопка Запитувати кількість і ціну). При підборі товарів можна показувати список підібраних позицій або відключити цей список (гіперпосилання Всього підібрано... (приховати.)). Можна перетягувати товари зі списку товарів і залишків в список підібраних позицій. Для перенесення підібраних позицій в документ використовується кнопка Перенести в документ.


Гарячі клавіші при підборі товарів:

  • Alt + 1-перехід до строки пошуку.
  • Alt+2 – перейти до фільтрів.
  • Alt+3 – перейти до списку товарів.
  • Alt+4 – перейти до списку підібраних товарів.

Переходи також можна здійснювати за допомогою клавіші Tab.

Варіанти відбору при підборі позицій

У діалоговому вікні підбору можна встановити різні варіанти відбору товарів за допомогою команди Фільтр за. Для кожного товару показуються ціни по тому типу цін, який вказаний в документі, і залишки товарів на тому складі, який вказаний в документі.

  • Фільтр по ієрархії. Буде показаний ієрархічний список груп номенклатури. При виборі конкретної групи номенклатури в лівій частині буде показаний список номенклатурних позицій, що належить даній групі.
  • Фільтр за видами і властивостями. Буде показаний список видів номенклатури. При виборі конкретного виду номенклатури в лівій частині буде показаний список позицій, що належить даному виду номенклатури. При використанні даного варіанту підбору можна встановити додатковий фільтр за властивостями виду номенклатури (наприклад, відібрати товари, що мають певний розмір, колір тощо). Список полів для відбору вказується у формі виду номенклатури (Настройка панелі швидкого відбору номенклатури). Перейти до фільтру за видом і властивостями і встановити додаткові фільтр за властивостями виду номенклатури можна по команді контекстного меню Номенклатура зі схожими властивостями на позиціонованій номенклатурі.
  • Фільтр по товарах іншої якості . При виборі номенклатури вихідної якості в лівій частині буде показаний список неякісних товарів вихідної номенклатури. При використанні даного варіанту підбору можна встановити додатковий фільтр за якістю. Перейти до фільтру по товарах іншої якості та встановити номенклатуру вихідної якості можна по команді контекстного меню Товари іншої якості на спозицінованій номенклатурі. Товари, для яких є товари іншої якості відзначені коричневою картинкою.
  • Ціна від ... до .... Відбираються товари з ціною в заданому діапазоні. Відбір за ціною доступний при включенні фільтра за видами і властивостями або по ієрархії.
  • Сегмент. При виборі конкретного сегмента номенклатури в лівій частині буде показаний список позицій, що належить даному сегменту номенклатури.
  • Тільки в наявності на складі.... При встановленні цього прапорця будуть показані тільки ті товари, які є в наявності на зазначеному складі.

У діалоговому вікні підбору можна встановити додатковий відбір за довільними словами пошуку (Знайти). Для кожного товару по умовчанню показуються залишки товарів на тому складі, який вказаний в документі.
Встановити/відключити відображення залишків по всіх складах можна по гіперпосилання Детальна інформація про залишки (показати)/(приховати)

Відбір товарів за властивостями видів номенклатури
  • При установці фільтра за видами і властивостями і вибору виду номенклатури буде показаний список всіх тих значень, які визначені для властивостей, обраних в якості параметрів швидкого відбору у вигляді номенклатури.
  • Якщо для товару ведеться облік за характеристиками, то в список будуть додані параметри швидкого відбору за характеристиками.
  • Позначте прапорцями ті значення властивостей, за якими ви хочете зробити відбір товарів (наприклад, розмір: 35, 36. Колір: Білий.)
  • У списку залишаться тільки ті товари, які мають дані значення властивостей.
  • Виберіть потрібний товар.
  • Якщо для товару ведеться облік за характеристиками, то буде показаний список товарів, деталізований до рівня конкретних характеристик (значень властивостей виду номенклатури).
  • Виберіть потрібний товар з характеристиками.
  • Для повернення до списку товарів натисніть на кнопку До номенклатури.
  • Виконайте аналогічні дії з підбору наступного товару з відібраною характеристикою.

Висновок інформації про ціни призначених на номенклатурну позицію
  • Встановіть курсор на товар, для якого треба подивитися ціни.
  • Виберіть команду Ціни.
  • Будуть показані всі ціни, призначені на товар.

У списку будуть показані тільки ті ціни, які присутні в прайс-листі торгового підприємства (встановлений прапор Використовувати при продажу клієнтам). Список цін також може бути обмежений відповідно до встановлених прав доступу (видами доступу за видами цін) конкретного користувача.

Підбір і перенесення товарів в документ
  • Підбір товарів проводиться установкою курсору на товар і натисканням клавіші ENTER, подвійним кліком по позиції товару або перетягуванням в список підібраних позицій. Якщо для товару використовуються характеристики, то відкриється список характеристик, де необхідно відібрати конкретну характеристику.
  • Підбір товарів може бути здійснено з запитом кількості і ціни або без запиту (при встановленій настройці Ще - Настройка - Запитувати кількість і ціну).
  • При підборі товарів можна показати список підібраних позицій або відключити цей список (гіперпосилання Всього підібрано позицій).
  • Для перенесення підібраних позицій в документ використовується кнопка Перенести в документ.

Підбір товарів можна здійснити зі списку, який показує залишки товарів. Такий варіант зручно використовувати, якщо склади вказуються в табличній частині документа. При активізації строки із зазначеним складом і датою передбачуваного відвантаження, інформація про склад і дату відвантаження переноситься в табличну частину документа.

Підбір товарів у кошик покупця

Для того що б почати підбір товарів в кошик покупця необхідно скористатися командою форми Підібрати товари. Буде відкрита форма підбору товарів в корзину. Детальніше про роботу з формою див. довідку форми.

Зміна варіанту пошуку номенклатури

Змінити варіант пошуку номенклатури можна за допомогою спеціальної форми налаштування пошуку. (Ще - Настройки - Настроїти пошук).
