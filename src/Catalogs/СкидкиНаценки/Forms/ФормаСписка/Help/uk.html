Знижки (націнки)

Довідник Знижки та націнки призначений для опису всіх можливих знижок/ націнок, що застосовуються на підприємстві та визначення правил їх взаємного застосування. Кожна група довідника є категорією і визначає правила спільного застосування знижок всередині категорії.

Можливість використання ручних (автоматичних) знижок визначається відповідними функціональними опціями в розділі НДІ і адміністрування - CRM і маркетинг - Маркетинг. При цьому доступно незалежне використання ручних, автоматичних знижок і знижок по картах лояльності.

Настройка пріоритету знижок (націнок)

Спільне застосування для груп знижок (націнок)

Додавання нової групи
  • У списку знижок (націнок) натисніть на кнопку введення нової групи. Для введення нової групи можна також застосовувати поєднання клавіш CTRL + F9.
  • З'явиться картка нової групи.
  • У картку нової групи треба ввести найменування групи і умови спільного застосування знижок, які будуть відноситися до даної групи.

Розподіл знижок по групах


Система дозволяє провести групову операцію по віднесенню списку знижок (націнок) до однієї групи.

  • У списку знижок (націнок) відзначте ті знижки (націнки), які необхідно віднести до однієї групи, використовуючи стандартні можливості платформи.
  • У контекстному меню (по правій клавіші миші) виберіть дію Перемістити до групи.
  • У списку груп встановіть курсор на потрібну групу і натисніть на кнопку Вибрати.
  • Зазначені знижки (націнки) будуть перенесені в зазначену групу.
  • Аналогічним чином виконайте дії з розподілу знижок (націнок) по інших групах.

Настройка пріоритету знижок (націнок)


Пріоритет дії знижок (націнок) настроюється для кожної групи. Найвищим пріоритетом буде володіти та знижка (націнка), яка розташована вгорі. Для настройки пріоритету використовуються клавіші переміщення, розташовані в командній панелі списку.

Спільне застосування для груп знижок (націнок)


Правило спільного застосування можна вказати не тільки для знижок (націнок), що відносяться до однієї групи, але і для декількох груп знижок (націнок). Для груп також можна призначити пріоритет і варіант спільного використання. Таким чином можна побудувати багаторівневу систему спільної дії знижок.

Приклад. При призначенні знижки повинна застосовуватися максимальна знижка з тієї групи, яка має найвищий пріоритет. У списку знижок вводимо дві групи: Група № 1 і Група № 2, в яких в якості варіанту спільного застосування відносяться до них знижок вказуємо Максимум. Ці групи об'єднуємо в одну групу Загальна група, в якій вказуємо варіант спільного використання Витіснення. Для Групи №1 встановлюємо найвищий пріоритет (поміщаємо її вгорі списку групи Загальна група). При спільному застосуванні знижок буде обрана максимальна знижка з групи Група №1.
