Список регламентованих звітів

Список «Регламентована звітність» містить список всіх введених форм звітності. Список відкривається по кнопці «Журнал» з форми списку довідника «Регламентовані звіти».
При відкритті раніше введених звітів відразу відкривається форма звіту, без виклику «стартової» форми.
