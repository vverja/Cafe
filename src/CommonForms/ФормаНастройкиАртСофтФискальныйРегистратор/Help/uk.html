Форма настройки обробника драйвера "ArtSoft: Універсальний драйвер реєстраторів

Форма призначена для настройки параметрів роботи фіскальних реєстраторів, підтримуваних драйвером "АртСофт: Універсальний драйвер фіскальних реєстраторів". Форма дозволяє налаштовувати наступні параметри пристрою:

  • Модель пристрою
     Модель підключається фіскального пристрою і тип протоколу підключення.
  • Порт
    Номер послідовного порту, до якого підключений фіскальний реєстратор.
  • Швидкість
    Швидкість обміну даними з фіскальним реєстратором.
  • IP адрес
     ІР адреса для протоколу підключення TCP/IP.
  • Порт ФР
    Порт підключення для протоколу підключення TCP/IP.
  • Податкові ставки
    Дана настройка призначена для зазначення номера використовуваної податкової ставки фіскального реєстратора для типу податку, заданого для номенклатури. Перед використанням податкова ставка повинна бути запрограмована Центром сервісного обслуговування у фіскальному реєстраторі. Кількість податкових ставок залежить від моделі і версії реєстратора і описано в документації до фіскального пристрою.

Після зміни значень параметрів слід натиснути кнопку "ОК" для того, щоб зберегти внесені зміни, або кнопку "Скасувати" для того, щоб відмовитись від їх збереження.
