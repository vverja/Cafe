Призначений для вибору об'єктів програми.

Викликається з інших об'єктів програми за командою Вибрати або за відповідним посиланням.

Список об'єктів програми являє собою дерево, що згруповане за типами об'єктів програми.

Вибір об'єктів програми

Для вибору потрібних об'єктів увімкніть відповідні прапорці.

  • Для того щоб вибрати всі об'єкти одного типу, увімкніть прапорець, що відноситься до групи об'єктів, наприклад, для того щоб вибрати всі довідники, увімкніть прапорець Довідники.
  • Для того щоб вибрати всі перераховані в списку об'єкти програми, увімкніть прапорець, що відповідає кореню.

Натисніть кнопку Вибрати.
