Как вводится информация о кодах для НН и номерах ГТД?


В форме редактирования выбирается код УКТВЭД из справочника Классификатор кодов для НН и номер ГТД из справочника Номера грузовых таможенных деклараций.

Обязательно ли указывать номер ГТД для импортых товаров?

Можно как указывать, так и не указывать номер ГТД для импортных товаров. В налоговых документах необходимо указывать только коды УКТВЭД (ДКПП). Учет по номерам ГТД целесообразно вести импортерам с целью сравнения цены приобретения и последующей продажи.

  • Если при приобретении товаров не указывается номер ГТД, в информационной базе ведется учет остатков товаров в разрезе кодов УКТВЭД. Товары поступают и выбывают по пустому номеру ГТД. 
  • Если при приобретении товаров указывается номер ГТД, ведется учет остатков товаров в разрезе кодов для НН и номеров ГТД. При выбытии товаров автоматически определяются номера ГТД. Бухгалтеру предприятия известны остатки импортных товаров на любой момент времени; покупателю по его запросу могут быть сообщены номера ГТД поставленных ему товаров (форма Реестр кодов УКТВЭД и номеров ГТД документа Реализация товаров и услуг). В печатных формах  налоговых документов наличие номеров ГТД регулируется настройкой учетной политики
