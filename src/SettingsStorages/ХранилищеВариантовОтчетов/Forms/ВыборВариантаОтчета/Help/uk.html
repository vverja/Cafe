Призначений для вибору варіантів звітів.

Відкривається з форми варіанту звіту по команді Ще - Інше - Вибрати варіант.

Для вибору доступні:

  • стандартні, передбачені в програмі напередвизначені варіанти звітів. По умовчанню є спільними і в списку відображаються зі значком ;
  • спільні варіанти звітів, надані іншими користувачами в загальний доступ. У списку вони відображаються зі значком . Для вибору доступні варіанти всіх авторів;
  • особисті варіанти звітів, які збережені раніше. Кольором відображаються варіанти звітів, автором яких є поточний користувач.

Список вибору може містити три групи варіантів звітів:

  • використовувані варіанти звітів (не виділяються в окрему групу, розташовані на початку списку);
  • Приховані в панелях звітів - у цю групу входять напередвизначені загальні звіти, у яких вимкнений прапорець По умовчанню видний в панелі звітів;
  • Відмічені для вилучення. Група завжди розміщена внизу списку вибору. При виборі відміченого для вилучення варіанта програма виводить попередження.

У списку виводиться:

  • Найменування - повне найменування варіанту звіту, що виводиться у списках.
  • Картинка Доступний всім користувачам:
    • видно - варіант звіту доступний всім користувачам,
    • не видно - варіант звіту доступний тільки автору.
  • Автор - повне ім'я користувача, який є автором варіанту звіту.
  • У нижній частині списку виводиться Опис - детальна інформація про вибраний варіант звіту.

Вибір варіанту звіту

  • Для того щоб відкрити варіант звіту, виділіть його в списку, натисніть Вибрати.
  • При виборі відміченого для вилучення варіанту звіту програма виводить попередження.
  • Також можна скористатися подвійним натисненням кнопки миші.

Відбір

  • Якщо список варіантів звітів великий, то за допомогою відбору по автору можна швидко знайти потрібний варіант звіту.
    • У полі Автор виберіть автора варіанта звіту зі списку користувачів.

Настройка варіанту звіту

  • Виберіть варіант звіту, натисніть Ще - Змінити, відредагуйте необхідні поля.

Вилучення варіанту звіту

  • Користувацький варіант звіту за допомогою команди меню Ще можна Відмітити для вилучення. Також можна скористатися клавішею Del.
  • Повторно виконайте цю команду, для того щоб Зняти відмітку вилучення.
  • Ця можливість недоступна для напередвизначених варіантів звітів.

Адміністратору/Відповідальному за звіти:

  • Зазвичай особисті варіанти звітів інших авторів приховані. Але в рідкісних випадках може знадобитися прямо з цього списку вибору зробити один з особистих варіантів звітів загальнодоступним або призначити його іншому користувачу.
  • Для того щоб побачити всі варіанти звітів без винятку, установіть прапорець Показувати особисті варіанти звітів інших авторів меню Ще.
