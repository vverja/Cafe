План-фактный анализ работы торговых представителей

Отчет предназначен для план-фактного анализа количественных показателей по торговым представителям.

Особенности отчета:

  • детализация по заданиям,
  • отбор по клиентам, с которыми работает торговый представитель.


Для отчета доступны следующие варианты:

  • Выполнение заданий (план-факт по количеству) по номенклатуре;
  • Выполнение заданий (план-факт по количеству) по торговым представителям.


Формы отчетов зависят от выбранной детализации в настройках параметров учета.
