Предназначена для ввода населенного пункта по детальному шаблону.

Открывается из шаблона ввода адреса в поле Город, населенный пункт по кнопке  Выбрать.

Поля Область, Район, Город, Нас.пункт предназначены для ввода соответствующих частей адреса. Поля, которые должны быть заполнены, зависят от адреса. Например, в некоторых адресах может не быть района, в других - города или населенного пункта. Если выбрано устаревшее наименование, программа выводит сообщение. Подтвердите свой выбор, нажав кнопку Да для продолжения ввода.

После окончания ввода города или населенного пункта нажмите ОК для продолжения ввода адреса.
