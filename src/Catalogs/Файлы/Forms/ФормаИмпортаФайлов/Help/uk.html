Призначена для завантаження файлів з диска на комп'ютері.

Відкривається зі списку файлів за командою Завантажити файли контекстного меню по правій кнопці миші або меню Ще після вибору файлу (файлів) у вікні, що відкрилось, Провідника Windows. У провіднику виділіть один або кілька файлів, натисніть Відкрити.

Виводиться список файлів, що завантажуються. Можна замінити весь список за допомогою кнопки Вибрати (доповнити список неможливо, при виборі нових файлів старі вилучаються) - відкрити Провідник Windows і вибрати інші файли.

Заповніть необхідні для завантаження файлів відомості:

  • У полі Папка автоматично проставляється обрана папка в списку файлів, в яку будуть завантажені файли з диска на комп'ютері. При необхідності можна вибрати іншу папку;
  • Для економії місця на диску увімкніть прапорець Вилучати файли після додавання (по умовчанню вимкнено);
  • Напишіть Опис для файлів, що створюються;
  • Увімкніть прапорець, для того щоб Зберігати версії (рекомендується, по умовчанню прапорець включений);
  • Для текстових файлів можна Вибрати кодування зі списку.
  • Для підтвердження натисніть Завантажити файл.
Також можна використовувати перетягування мишею для завантаження файлів з жорсткого або мережевого диска - просто перетягніть файли мишею з Провідника Windows в потрібну папку списку файлів програми.
