Пересчет товаров

Документ Пересчет товаров используется для подбора товаров к пересчету, ввода фактического количества товарных остатков и работы с печатными формами. По результатам пересчета оформляются соответствующие складские акты Оприходование излишков товаров, Списание недостач товаров, Пересортица товаров, Порча товаров. Если для склада установлен флаг использования статусов пересчетов товаров, то оформление складских актов возможно для документов Пересчет товаров в статусе Выполнено.

Складские акты могут быть созданы вводом на основании документа пересчета или с использованием Помощника оформления складских актов по команде Оформить из формы списка (команда доступна, если есть расхождения, по которым не оформлены складские акты, выявленные пересчётами и склад является ордерным при отражении излишков, недостач и порчи).
