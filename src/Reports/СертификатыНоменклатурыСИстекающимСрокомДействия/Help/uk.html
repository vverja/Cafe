Сертифікати номенклатури зі строком дії, що закінчується

Звіт призначений для аналізу реєстру сертифікатів, що діють для обраної області дії.

Звіт Сертифікати номенклатури викликається зі звіту Товари з сертифікатами, які закінчуються. Подвійне натискання по строці ініціює формування нового звіту, при цьому настройки області дій зберігаються.

Для кожного сертифіката зазначаються:

  • дата закінчення строку дії;
  • стан.


Сертифікату можуть бути присвоєні стани:

  • Діє- для діючих сертифікатів;
  • Призупинено- для сертифікатів, які виведені з дії достроково;
  • Прострочено- для сертифікатів, чий строк дії закінчився сьогодні.


Якщо звіт викликаний по групуванню: номенклатура + тип сертифіката, то у звіті також будуть присутні сертифікати, що належать всім характеристикам і серіям.
