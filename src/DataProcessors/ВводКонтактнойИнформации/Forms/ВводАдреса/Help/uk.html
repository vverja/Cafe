Призначена для введення адреси за шаблоном.

Відкривається по кнопці Вибрати з поля Адреса в елементах списків.

  • У полі Країна виберіть країну адреси. Країна "Україна" обрана програмою по умовчанню, при цьому поле може бути недоступним в залежності від настройок. Для того, щоб змінити країну, натисніть Вибрати.
  • Поле Індекс призначене для введення поштового індексу, що відповідає даній адресі.
  • Поле Місто, населений пункт призначено для введення регіону, міста і т. п. При необхідності можна ввести частині адреси детально, натиснувши Вибрати. У відкритому шаблоні введіть дані населеного пункту.
  • Поле Вулиця призначене для введення вулиці адреси.
  • Поля Будинок, Корпус, Квартира дозволяють ввести відповідні частини адреси. Можна змінити ці варіанти, вибравши додаткові значення зі списку.
  • Поле Представлення адреси при заповненні адреси за шаблоном недоступне, заповнюється програмою автоматично в міру заповнення полів адреси.
  • Поле Коментар дозволяє вказати довільний коментар до адреси, який не буде включатися в її представлення і виводитися на друк.

Історія змін

  • У деяких списках можна переглянути історію зміни адреси за допомогою команди Ще - Історія змін.

Адреса на Google Maps

  • За допомогою цих команд будь-яку адресу, введену в програмі, можна подивитися на картах.

Введення адреси у довільній формі

  • Увімкніть прапорець Ввести адресу у вільній формі меню Ще, для того щоб при введенні адреси за шаблоном ввести свідомо неправильну адресу (наприклад, з товаром на накладній неправильну адресу постачальника, її треба тимчасово зафіксувати для друку, але надалі її використовувати не можна). Така адреса використовується для зберігання, тому можна рекомендувати використовувати її тільки для одиничних випадків.
  • Виводиться шаблон для введення адреси, але всі його поля, крім поля Представлення адреси і Коментар, стають неактивними. Заповніть поле Представлення адреси. При переході до поля Коментар або натисканні кнопки ОК заповнюються ті поля шаблону, які програмі вдалося розпізнати в рядку адреси (тобто в цих полях немає помилок). Перевірка заповнення адреси в даному випадку відключена.
  • Для підтвердження адреси натисніть ОК. Програмою буде зафіксовано, що адреса заповнена у вільній формі і на перегляд адреса буде виводитися в такому вигляді.

Очищення адреси

  • З допомогою команди меню Ще - Очистити адресу можна стерти неправильно введену адресу.
