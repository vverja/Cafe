Форма сегментов запрета отгрузки

Отгрузку партнерам можно запретить вручную и автоматически. Запрет отгрузки настраивается в разрезе сегментов клиентов с установленным признаком Сегмент используется для запрета отгрузки. Если партнер входит в такой сегмент, то в документах продажи появится сообщение о том, что партнеру запрещена отгрузка и документ не проведется. Для отгрузки таким клиентам у пользователя должно быть установлено право Реализация партнерам, входящим в сегменты запрета отгрузки.

Запрет отгрузки возможен только в том случае, если в разделе НСИ и администрирование - Продажи - Оптовые продажи - Использование заказорв включена функциональная опция Запрет отгрузки.

Ручной запрет


Для того чтобы вручную запретить отгрузку партнеру, необходимо создать сегмент клиентов с признаком Сегмент используется для запрета отгрузки и способом формирования Формировать вручную, а затем добавить партнера в состав данного сегмента.

Автоматический запрет


Для того, чтобы запретить отгрузку партнеру автоматически, в соответствии с определенными правилами (например, превышения суммы кредита или за количество дней просроченной оплаты), необходимо создать сегмент клиентов с признаком Сегмент используется для запрета отгрузки, способом формирования Периодически обновлять и настроить правила попадания партнера в состав сегмента в схеме компоновки данных данного сегмента.

Запрет/разрешение отгрузки в карточке партнера


В карточке партнера показывается состояние запрета на отгрузка клиенту:

  • Если показана гиперссылка Клиенту разрешена отгрузка, то клиент не входит ни в один сегмент с признаком Сегмент используется для запрета отгрузки.
  • Если показана гиперссылка Клиенту запрещена отгрузка, то клиент входит хотя бы в один из сегментов с признаком Сегмент используется для запрета отгрузки.


Для изменения состояния выполните следующие действия:

  • В карточке партнера нажмите гиперссылку Клиенту запрещена отгрузка/Клиенту разрешена отгрузка.
  • Будет открыт список сегментов, в которых установлен флаг Сегмент используется для запрета отгрузки.
  • Для каждого сегмента будет показано, разрешена или запрещена отгрузка данному клиенту в соответствии с правилами формирования сегмента.
  • Если сегмент формируется вручную, то можно запретить/ разрешить отгрузку клиенту непосредственно из формы списка.


Для разрешения отгрузки клиент должен быть исключен из всех сегментов запрета отгрузки.
