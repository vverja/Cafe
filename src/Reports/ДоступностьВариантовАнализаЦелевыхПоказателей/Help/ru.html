Доступность вариантов анализа целевых показателей

Отчет предназначен для проверки доступности вариантов анализа и установленых для них ответственных.

Особенности отчета:

  • анализ по ответственным,
  • анализ по сотрудникам подразделений.


Примеры:

Контроль списка вариантов анализа, по которым не определен ответственный

Ответственный за разработку вариантов анализа по заданному целевому показателю

Доступность вариантов анализа для сотрудника

Контроль списка вариантов анализа, по которым не определен ответственный


Установите флажок в поле Ответственный, поле выбора оставьте пустым. Непосредственно из отчета можно открыть карточку варианта анализа и назначить ответственного.

Ответственный за разработку вариантов анализа по заданному целевому показателю


Установите отбор по конкретному целевому показателю в поле Цель.

Доступность вариантов анализа для сотрудника


Установите отбор по конкретному ответственному. В отчете будет показана информация о вариантах анализа, по которым пользователь является ответственным. Проверьте доступность этих вариантов анализа.
