Выбор различных статей расходов для сумм, распределенных на расходы будущих периодов

Выбор различных статей расходов для сумм, распределенных на расходы будущих периодов

При распределении расходов будущих расходов можно указать различные статьи затрат.

  • Если необходимо изменить статью затрат для всех распределенных сумм, то необходимо по команде Заполнить - Заполнить статью расходов выбрать нужную статью расходов.
  • Если необходимо указать различные статьи расходов для разных строк документа с распределенными суммами расходов, то используется команда Заполнить - Заполнить статью расходов для выделенных строк.
