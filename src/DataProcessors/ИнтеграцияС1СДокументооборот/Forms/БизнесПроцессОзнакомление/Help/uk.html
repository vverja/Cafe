Цей бізнес-процес використовується для передачі колегам будь-яких документів для обов'язкового ознайомлення.

Цей бізнес-процес можна створювати безпосередньо з картки документа за допомогою меню "Створити бізнес-процес".

При старті цього бізнес-процесу всі виконавці за списком автоматично отримають задачу "Ознайомитися".

Для перегляду поточного стану бізнес-процесу (списку виконаних і невиконаних задач) використовуйте команду "Задачі" в панелі навігації документа, за яким було запущено Ознайомлення.

Якщо не був виконаний запуск бізнес-процесу, заповнення форми може бути виконано на основі шаблону бізнес-процесу. Для цього необхідно виконати команду "Заповнити за шаблоном" на командній панелі форми. Якщо для даного типу бізнес-процесу і виду предмета бізнес-процесу настроєні шаблони бізнес-процесів, то система запропонує вибрати один із шаблонів для заповнення форми. Якщо був налстроєний тільки один шаблон, то заповнення форми відбудеться автоматично
