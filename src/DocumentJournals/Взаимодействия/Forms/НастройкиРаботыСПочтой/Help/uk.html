Призначена для настройки роботи з поштою.

Перед тим як почати працювати з поштовими повідомленнями, в програмі необхідно налаштувати роботу з поштою. Для цього в списку взаємодій виконайте команду Ще- Персональні настройки.

Як налаштувати підпис до листа?

Як вказати, що для всіх відправлених листів по умовчанню необхідно повідомлення про доставку та прочитання?

Як налаштувати порядок відповідей на запити повідомлень про прочитання?

Як вказати, що в документі вихідний лист при відповіді або пересиланні необхідно відображати тіло листа підстави?

Як вказати, що в тіло вихідного листа при відповіді або пересиланні необхідно включати тіло листа підстави?

Як вказати в файловому режимі, що листи повинні відправлятися відразу?

Настройка підпису до листа

При настройці роботи з поштою можна створити шаблони підписів для вихідних електронних листів. Також шаблони підписів можна створити при настройці облікових записів електронної пошти.

  • Можна створити підпис двох видів:
    • для нових повідомлень- на вкладці Підпис для нового.
    • для повідомлень при відповіді або пересиланні- на вкладці Підпис при відповіді.
  • Встановіть прапорець Включати для потрібного виду підпису. Виберіть Формат підпису:
    • Звичайний текст;
    • HTML. Створення підпису у форматі HTML дозволить форматувати текст і вставляти в нього картинки.
      • Створення підписів у форматі HTML доступно, якщо це передбачено настройками програми. Цю настройку можна змінити в розділі НДІ і адміністрування- Органайзер, відредагувавши значення прапорця Вихідні листи у форматі HTML.
  • Напишіть підпис у відповідному полі. При виборі формату HTML можна вибирати різні способи форматування тексту, вставляти картинки і гіперпосилання. Текст можна форматувати за допомогою відповідних кнопок, а також команд меню Ще.
Важливо. Якщо для облікового запису налаштовано підпис, то під час створення листа підставляється підпис облікового запису.

Повідомлення про доставку та прочитання

На вкладці Відстеження листів можна включити можливість запиту повідомлень від одержувача про прочитання і доставку відправлених листів:

  • Увімкніть прапорці Сповіщення про прочитання та Сповіщення про доставку.
  • Ці значення будуть використовуватися для кожного вихідного листа як значення по умовчанню, при цьому їх можна буде змінити.

Настройка порядку відповідей на запити повідомлень про прочитання

Дані настройки можна виконати на закладці Відстеження листів. Доступні наступні варіанти поведінки програми при отриманні запитів сповіщення про прочитання листів (виберіть один з варіантів за допомогою перемикача):

  • Завжди відправляти повідомлення- після установки у вхідному електронному листі прапорця Розглянуто буде надіслано повідомлення про прочитання.
  • Ніколи не відправляти повідомлення- запити на відправку повідомлення про прочитання будуть ігноруватися.
  • Запитувати перед тим, як відправити повідомлення- при установці в листі прапорця Розглянуто буде поставлено питання про відправку повідомлення про прочитання.

Відображення тіла листа-підстави в документі Вихідний лист при відповіді або пересилання
  • Увімкніть прапорець Відображати тіло вихідного листа під час відповіді або пересиланні на вкладці Інше.
  • Тіло листа-підстави в цьому випадку відображається тільки в документі взаємодій Вихідний лист, в лист, який надсилається, воно включено не буде.
  • Це значення буде використовуватися для кожного вихідного листа як значення по умовчанню при цьому в самому листі його можна буде змінити.

Включення тіла листа-підстави в тіло вихідного листа при відповіді або пересиланні
  • Встановіть прапорець Включати тіло вихідного листа при відповіді або пересиланні на вкладці Інше.
  • Це значення буде використовуватися для кожного вихідного листа як значення по умовчанню при цьому в самому листі його можна буде змінити.

Настройка відправлення повідомлень відразу в файловому режимі

  • У файловому режимі роботи програми передбачена можливість Відправляти поштові повідомлення відразу після натискання кнопки Відправити (по умовчанню листи відправляються за розкладом регламентного завдання Отримання і відправлення електронних листів).
  • У файловому режимі листа можна відправити також за допомогою команди Відправити і отримати списку взаємодій.
