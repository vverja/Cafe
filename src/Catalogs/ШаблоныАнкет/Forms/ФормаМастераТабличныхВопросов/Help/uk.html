Майстер призначений для введення і редагування табличних питань анкети.

Відкривається по команді Додати - Табличне питання з шаблону анкети.

Вибір типу табличного питання

  • На першій сторінці майстра напишіть Формулювання питання, наприклад, "Вкажіть близьких родичів".

  • За допомогою перемикача виберіть тип табличного питання. Табличні питання можуть бути чотирьох типів:

    • Складене табличне питання;
    • Табличне питання з напередвизначеними відповідями в рядках;
    • Табличне питання з напередвизначеними відповідями в колонках;
    • Табличне питання з напередвизначеними відповідями в рядках і колонках.
  • У правій частині вікна розташовується малюнок з прикладом обраного типу табличного питання.
  • Натисніть кнопку Далі для підбору питань.
  • Для роботи з питаннями використовуйте команди меню Ще або контекстне меню за допомогою правої кнопки миші.
  • Якщо заздалегідь підготувати питання типу Вибір одного або декількох варіантів відповіді із запропонованих, то можна буде використовувати напередвизначені відповіді в табличних питаннях, крім цього, можна швидко додати варіанти відповідей.

Складене табличне питання

  • Для складеного табличного питання задається склад питань. Задані питання будуть відображатися в якості заголовків колонок таблиці.
  • Респондент відповідає на питання анкети, зможе додати довільну кількість рядків у таблицю при відповіді.
  • За допомогою команди Додати виберіть необхідну кількість питань зі списку.
  • Натисніть Далі для перегляду таблиці, що вийшла.

Табличне питання з напередвизначеними відповідями в рядках;

  • Для даного типу табличного питання на другій сторінці помічника задається склад питань.
  • Натисніть Додати, виберіть як мінімум два питання зі списку.
  • Задані питання будуть відображатися в якості заголовків колонок таблиці в анкеті респондента.
  • Число рядків в таблиці фіксовано, при відповіді респондент не зможе додати або вилучити рядки з таблиці. Кількість рядків таблиці дорівнюватиме кількості напередвизначених відповідей.
  • Для першого питання на третій сторінці помічника задайте напередвизначені відповіді, які будуть відображатися у рядках першої колонки таблиці. Натисніть Додати. Для додавання скористайтеся одним із способів:
    • Напишіть варіанти відповідей для першого питання вручну.
    • Якщо був обраний питання типу Вибір одного або декількох варіантів відповіді із запропонованих, то для того щоб швидко перерахувати можливі варіанти відповідей для першого введеного питання, можна скористатися кнопкою Заповнити варіанти відповідей.
  • Натисніть Далі для перегляду таблиці, що вийшла.

Табличне питання з напередвизначеними відповідями в колонках;

  • Цей тип табличного питання є варіантом попереднього.
  • Для даного типу табличного питання задається склад питань. Задані питання будуть виведені в рядках першої колонки таблиці. За допомогою кнопки Додати введіть як мінімум два питання.
  • На третій сторінці помічника укладач анкети вказує напередвизначені відповіді на перше питання для колонок таблиці. Для цього натисніть Додати. Для додавання скористайтеся одним із способів:
    • Напишіть варіанти відповідей для першого питання вручну.
    • Якщо був обраний питання типу Вибір одного або декількох варіантів відповіді із запропонованих, то для того щоб швидко перерахувати можливі варіанти відповідей для першого введеного питання, можна скористатися кнопкою Заповнити варіанти відповідей.
  • Дані напередвизначені відповіді будуть відображатися в якості заголовків колонок таблиці. При відповіді респондент не зможе додати або вилучити рядки з таблиці. Кількість рядків таблиці дорівнюватиме кількості питань, зменшеній на одне (відповіді на перше питання - це заголовки колонок, при цьому перше питання в таблиці не відображається).
  • Натисніть Далі для перегляду таблиці, що вийшла.

Табличне питання з напередвизначеними відповідями в рядках і колонках.

  • Для даного типу табличного питання задається три питання. Натисніть Додати, виберіть питання зі списку.
  • На третій сторінці помічника необхідно задати напередвизначені відповіді на перші два питання.
    • Напередвизначені відповіді на перше питання будуть відображатися в якості заголовків рядків таблиці.
    • Напередвизначені відповіді на друге питання будуть відображатися в якості заголовків колонок.
  • Відповіді на третє запитання респондент повинен буде вказувати в комірках таблиці.
  • Натисніть кнопку Додати. Для додавання напередвизначених варіантів відповідей на перші два питання скористайтеся одним із способів:
    • Напишіть варіанти відповідей вручну.
    • Якщо був обраний питання типу Вибір одного або декількох варіантів відповіді із запропонованих, то для того щоб швидко перерахувати всі відповіді, можна скористатися кнопкою Заповнити варіанти відповідей.
  • При відповіді респондент не зможе додати або вилучити рядки з таблиці. Кількість рядків таблиці дорівнюватиме кількості напередвизначених відповідей на перше запитання.

    Перегляд готового табличного питання

    • На останній сторінці помічника для перегляду виводиться таблиця в тому вигляді, в якому вона буде виглядати в анкеті респондента. Якщо необхідно поміняти склад питань або їх розташування в таблиці, можна повернутися на попередні кроки за допомогою кнопки Назад.
    • Натисніть Готово для закінчення редагування.
