Виконання плану продажів

Звіт призначений для контролю виконання плану продажів по товарах і оплаті.

Варіанти звіту

Виконання плану продажу (по клієнтам, підрозділам, складам і менеджерам)

Виконання плану оплат і відвантажень по клієнтам

Виконання плану продажу (по клієнтам, підрозділам, складам і менеджерам)

Звіт дозволяє оцінити фактичний стан продажу товарів за планом і замовленням клієнтів, які ще не відвантажені зі складу.
Особливості звіту:

  • виведення показників (план, факт, виконання плану і не відвантажених замовлення) по періодичності зазначеної в сценарії за вказаний період;
  • швидкий відбір нерелевантних строк з урахуванням допустимого відхилення відсотка виконання плану за кількістю і сумою.

Для звіту передбачено чотири варіанти:

  • Виконання плану продажів по підрозділах;
  • Виконання плану продажів по клієнтам;
  • Виконання плану продажів по складах;
  • Виконання плану продажів по менеджерах.

У кожному варіанті можна встановити відбір по відповідному розрізу планування (підрозділ, клієнт, склад, менеджер).

Виконання плану оплат і відвантажень по клієнтам

Звіт дозволяє перевірити виконання плану оплат по фактичному приходу грошових коштів, порівняти з сумою плану і факту відвантажень клієнту за період.
Особливості звіту:

  • всі показники плану, факту, виконання плану з оплати і відвантаження клієнтам можна вивести з наростаючим підсумком, який підсумовує показник на заданий період;
  • аналіз виконання плану ведеться в розрізі сценарію і клієнтів;
  • швидкі відбори по допустимому відхиленню суми на відсоток виконання плану дозволяють відібрати рядки, в яких відсоток виконання перевищує або не досягає встановленого.
