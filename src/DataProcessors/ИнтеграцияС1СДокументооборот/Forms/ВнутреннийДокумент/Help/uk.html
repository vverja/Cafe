Довідник "Внутрішні документи" служить для обліку офіційних інформаційно-довідкових або організаційно-розпорядчих документів.

До кожного внутрішнього документа можна приєднати один або кілька файлів. Облік файлів можливий, якщо настройка програми "Використовувати файли у внутрішніх документів" встановлена.

Для документа слід задати:

  • папку внутрішніх документів, в якій буде розміщено документ;
  • користувача, який підготував документ (при створенні нового документа заповнюється програмою по поточному користувачу);
  • підрозділ, який випускає документ;
  • керівника (користувача), який затвердив (підписав) документ;
  • кореспондента, з яким пов'язаний документ, показується, якщо для зазначеного виду документа в настройках встановлено "Вести облік за кореспондентами";
  • контактна особа обраного кореспондента, показується, якщо для зазначеного виду документа в настройках встановлено "Вести облік за кореспондентами";
  • строк виконання - задається дата виконання документа, показується, якщо для зазначеного виду документа в настройках встановлено "Використовувати строк виконання";
  • гриф доступу - якщо встановлена настройка "Використовувати грифи доступу", то внутрішньому документу потрібно обов'язково присвоїти гриф доступу, вибравши його з числа заздалегідь настроєних адміністратором грифів. Гриф визначає права доступу користувачів до внутрішнього документу;
  • вид документа - показується, якщо настройка програми "Використовувати види внутрішніх документів" встановлена;
  • стан документа - показується, якщо настройка програми "Використовувати стан документів" встановлена;
  • номенклатура справ - показується, якщо для зазначеного виду документа в настройках встановлено "Вести облік за номенклатурою справ";
  • організація - у разі ведення документообігу для декількох організацій (показується, якщо настройка програми "Використовувати облік по організаціях" встановлена);


Документ може нумеруватися вручну або автоматично (в залежності від настройок виду документа).
Якщо задана автоматична нумерація, то для того, щоб присвоїти реєстраційний номер необхідно натиснути кнопку "Зареєструвати".
Якщо задано ручну нумерацію, то достатньо заповнити поля "Реєстраційний номер" та "Дата реєстрації" та записати документ.

Якщо настройка програми "Використовувати електронно цифровий підпис" включена, то в картці документа додаються група команд "ЕЦП" (містить команду "Підписати", за допомогою якої виконується посвідчення документа електронним підписом) і закладка "ЕЦП", в якій редагуються і перевіряються електронно-цифрові підписи.
Якщо документ підписаний, то його редагування неможливо. Для редагування потрібно спочатку вилучити всі підписи. Поточний користувач може вилучити тільки свої підписи. Право вилучення всіх підписів належить адміністратору.
Підпис можна зберегти у файл (наприклад, для пересилання електронною поштою), а також переглянути сертифікат підпису.
