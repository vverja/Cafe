Основний робочий каталог - спеціальний каталог на Вашому комп'ютері, в якому програма розміщує популярні і редаговані Вами файли. Список призначений для управління файлами в робочому каталозі.

Викликається з персональних настройок програми (закладка Робота з файлами) за командою Настройка основного робочого каталогу - Відкрити список файлів.

У списку виводиться:

  • Файл зайнятий для редагування - прапорець включений, якщо файл редагується.
  • Файл - формат (виводиться у вигляді значка) і найменування файлу;
  • Розмір (Кб);
  • Дата зміни.

З допомогою команди Ще - Змінити форму можна додати в список додаткові поля.

Доступні дії

  • Для видалення непотрібних файлів з робочого каталогу натисніть Видалити. При цьому будуть вилучені тільки ті файли, які Вами не вносилися будь-які зміни.
  • Для того щоб зберегти зміни у файлі і звільнити його для інших користувачів, натисніть Закінчити редагування.
  • Для того щоб звільнити зайняті Вами файли, використовуйте команду Скасувати редагування.
  • Натисніть Відкрити каталог файлу, для того щоб переглянути теку, в якій зберігається редагований файл.
  • Виконайте відповідну команду контекстного меню по правій кнопці миші або меню Ще, для того щоб Відкрити картку файлу.
