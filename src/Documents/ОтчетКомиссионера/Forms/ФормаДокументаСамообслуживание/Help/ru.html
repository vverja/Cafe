Заполнение реквизитов
  • Период отчета - указывается за какой период комиссионер отчитывается о продажах.
  • Соглашение - указывает на то, в рамках какого соглашения были взяты на комиссию товары
  • Договор - доступен, если указание договора требуется условиями соглашения.
  • Контрагент - юридическое лицо, от имени которого оформляется отчет.
  • Контактное лицо - определяет, с кем менеджер будет взаимодействовать по отчету.
  • Форма оплаты -доступна для указания, если не определена в соглашении.
  • Номер по данным комиссионера - номер отчета по вашим данным.
  • Дата по данным комиссионера - дата отчета по вашим данным.

Подбор товаров


По команде Подобрать по остаткам будет открыта форма в которой можно будет подобрать ранее взятые на комиссию товары.

Команда Подобрать будет доступна только после указания Соглашения.

Заполнение табличной части
  • Упаковка - возможно уточнить упаковку товара, например, на комиссию товар передавался в коробках, а отчет производится в штуках.
  • Цена продажи - необходимо указать цену, по которой был продан взятый на комиссию товар.

Уточнение разных цен для одного товара


По команде Разбить строку командной панели списка товаров, укажите количество позиций в новой строке. Количество в исходной строке будет соответственно уменьшено. В результате получится две строки с одним и тем же товаром. В строках можно указать разную цену продажи.

Определение суммы вознаграждения


Информация о рассчитанном вознаграждении выводится в нижней части формы. Она доступна, если в условиях соглашения определен автоматический расчет вознаграждения.
