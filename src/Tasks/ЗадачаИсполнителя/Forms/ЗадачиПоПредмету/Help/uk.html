Призначений для перегляду задач, створених по об'єкту програми.

Відкривається з об'єкта програми по команді Задачі панелі навігації.

Червоним кольором виділяються прострочені задачі. Виконані задачі стають неактивними, виділяються сірим кольором.

У списку виводиться:

  • - важливість, приймає значення Звичайна (по умовчанню), Низька, Висока. Для того щоб виділити бізнес-процес зі списку, або навпаки, вказати, що виконання цього бізнес-процесу може бути відкладено, вибираються відповідні значення цього поля.
  • - зупинені бізнес-процеси в цій колонці позначаються значком. Для того щоб Зупинити або Продовжити бізнес-процес, скористайтеся відповідними кнопками.
  • Задача- повне найменування задачі. Поруч проставляється значок Задача.
  • Виконавець задачі;
  • Строк-дата і час, до якого Завдання має бути виконано.

Перегляд задачі

  • Натисніть Змінити, для того щоб відкрити задачу для перегляду або редагування.
  • Також можна використовувати подвійне клацання лівою кнопкою миші.
