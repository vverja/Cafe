В даному діалозі можна ввести межі періоду розрахованих підсумків для регістрів накопичення або бухгалтерії.

По умовчанню пропонуються наступні дати для установки границі розрахованих підсумків:

  • для регістрів накопичення - закінчення попереднього місяця;
  • для регістрів бухгалтерії - закінчення поточного місяця.

Дати можна написати вручну або заповнити за допомогою кнопки .

Після зміни дата автоматично перетворюється до дати закінчення місяця обраної дати.

Якщо команда установки періоду розрахованих підсумків обрана для одного регістра, то буде доступна зміна тільки одного параметра, відповідного типу регістра.
