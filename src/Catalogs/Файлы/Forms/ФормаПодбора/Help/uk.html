Призначена для підбору файлів.

Список складається з двох частин: зліва список папок, праворуч - список файлів в обраній папці. Список складається з колонок:

  • Найменування - ім'я файлу, також наводиться опис - докладні відомості про файл;
  • Автор - користувач, який додав файл в програму;
  • Дата створення - дата та час додавання файлу в програму.

Виділіть у списку кілька файлів, натисніть Вибрати.

Див. також:

  • Загальні відомості про файли;
  • Робота з формами.
