Призначена для настройки параметрів проксі-сервера.

Програма надає можливості для отримання файлів з Інтернету по протоколах HTTP, HTTPS, FTP і збереження отриманих файлів на клієнті, сервері або у тимчасовому сховищі.

При роботі з програмою для виконання певних операцій (таких, як завантаження курсів валют, завантаження адресного класифікатора, обмін даними через FTP-ресурс, оновлення конфігурації з веб-сайту постачальника тощо) може знадобитися доступ до мережі Інтернет.

У разі якщо доступ в Інтернет здійснюється через проксі-сервер, в програмі необхідно вказати параметри проксі-сервера. При цьому існують клієнтські і серверні настройки проксі-сервера.

У загальному випадку, параметри проксі-сервера потрібні для роботи механізмів, які отримують файли з Інтернету за протоколами HTTP, HTTPS і FTP. Настройка проксі-сервера на сервері потрібна для роботи механізмів на сервері.

А для клієнтських механізмів (наприклад, завантаження курсів валют в тонкому і товстому клієнті) потрібна настройка на робочому місці. Водночас, для роботи в програмі з електронною поштою налаштовувати проксі-сервер не потрібно. Як правило, параметри проксі-сервера зазначаються для кожного робочого місця окремо, оскільки настройки доступу до мережі Інтернет можуть відрізнятися для різних робочих місць.

Але якщо в організації використовується клієнт-серверна версія продукту, параметри проксі-сервера треба вказати і на сервері (з розділу Адміністрування - Загальні настройки). При цьому параметри проксі-сервера для робочого місця і сервера можуть розрізнятися.

Для доступу в Інтернет з клієнтських комп'ютерів параметри проксі‑сервера задаються на кожному клієнтському робочому місці (по команді Персональні настройки в розділі Адміністрування).

Увага! при роботі з включеними профілями безпеки для настройки параметрів проксі-сервера можна використовувати тільки конфігураційний файл inetcfg.xml.

Настройка параметрів проксі-сервера

  • Виберіть варіант використання проксі-сервера за допомогою перемикача:
    • Не використовувати проксі-сервер - значення за замовчуванням при першому запуску програми;
    • Використовувати системні настройки проксі-сервера - настройки заповнюються автоматично значеннями з налаштувань проксі-сервера операційної системи. У цьому випадку заповнені настройки недоступні для редагування.
    • Використовувати інші настройки проксі-сервера - настройки задаються вручну:
      • Напишіть Адреса сервера і Порт сервера (однаковий параметр для сервера і робочих місць).
      • Для кожного робочого місця і сервера вкажіть Користувача проксі-сервера і його Пароль (в загальному випадку можуть бути різними, що залежить від мережевої політики в організації).
      • Установка прапорця Не використовувати проксі для локальних адрес дозволить знизити навантаження на проксі-сервер при роботі в локальній мережі організації.
  • Для підтвердження параметрів проксі-сервера натисніть ОК.
