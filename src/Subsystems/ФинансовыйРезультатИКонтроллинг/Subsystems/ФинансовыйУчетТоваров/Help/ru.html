Финансовый учет товаров

Назначение подсистемы


Подсистема предназначена для отражения и контроля операций с комиссионным товаром и последующего расчета себестоимости (предварительной и фактической) в рамках организаций входящих в состав предприятия.

Настройка подсистемы


Для настройки подсистемы в разделе НСИ и администрирование необходимо предварительно установить следующие флажки:

  • Комиссионные продажи - в разделе Продажи - Оптовые продажи позволяет осуществлять продажи товаров по комиссионной схеме.
  • Комиссионные закупки - в разделе Закупки - Документы закупок позволяет существлять закупку товаров по комиссионной схеме.
  • Обособленный учет себестоимости товаров по видам запасов - в разделе Финансовый результат и контроллинг - Учет товаров позволяет использовать обособленный учет товаров в разрезе сделок, менеджеров или подразделений.

Работа подсистемы


Передача товаров между организациями может быть оформлен в рамках рабочего места Передачи и возвраты товаров между организациями в двух режимах:

  • о продажи товаров конечному клиенту – на закладке Оформлено;
  • по результатам продажи товаров другой организации – на закладке К оформлению.


В ситуации, когда необходимо оформить передачу товаров до момента продажи, новый документ можно заполнить вручную. Отличить документы, оформленные по результатам продажи, можно по установленному признаку По товарам к оформлению. Особенность документа Передача товаров между организациями состоит в том, что в нем выбираются две организации: продавец и покупатель. Для организации-продавца движения по регистрам и заполнение печатных форм выполняются аналогично документу Реализация товаров и услуг, для организации-покупателя – документу Поступление товаров и услуг.
Если по результатам возврата от клиента необходимо отразить межфирменный возврат первоначальному владельцу, в настройке передачи следует выбрать вариант Продажи и возврат или Передача на комиссию и возврат. В первом случае будет создан документ с видом операции Возврат товаров между организациями, во втором – с видом операции Возврат по комиссии между организациями.

Отчет по комиссии между организациями


Документ Отчет по комиссии между организациями оформляется после продажи товара, полученного на комиссию от другой организации. Если настройка межфирменной передачи предусматривает комиссионную схему, продавец (организация-комиссионер) после реализации должен не только оформить передачу товаров на комиссию, но и отчитаться перед владельцем товара (организацией-комитентом).
При формировании нового документа список проданных или возвращенных клиентом товаров и прочие реквизиты заполняются автоматически. Если предполагается использовать комиссионное вознаграждение, эти данные заносятся вручную.
При установке в документах признака Расчеты через посредника становится доступной дополнительная закладка, где указываются партнер и контрагент для непрямых расчетов. В этом случае кредиторская и дебиторская задолженности регистрируется между организацией и контрагентом-посредником, а не между двумя организациями.

Расчет себестоимости товаров

Расчет себестоимости выполняется по данным оперативного учета и отражается в информационной системе документами Расчет себестоимости товаров. Документ выполняет один из двух видов расчета:

  • Предварительный расчет – предназначен для использования торговыми организациями в целях определения оценочной стоимости закупаемых материальных ценностей в течение отчетного периода. Выполняется способом средней взвешенной оценки. Рассчитанные значения используются для определения валовой прибыли организации при условии выполнения плана продаж;
  • Фактический расчет – выполняется по итогам месячного отчетного периода с полным расчетом стоимости партий движения номенклатурных затрат, что позволяет выбрать метод определения стоимости списания материальных ценностей.


При создании нового документа Расчет себестоимости метод расчета будет выбираться согласно данным настройки, но при желании его можно изменить вручную. Если расчет себестоимости производится для группы организаций, то метод оценки стоимости является общим для всех организаций. Для организаций, использующих механизм межфирменных продаж, расчет себестоимости должен производиться совместно.

Корректировки обособленного учета запасов


Документ предназначен для корректировки данных при ведении обособленного учета по сделкам, менеджерам или подразделениям. Ведение подобного учета позволяет детализировать данные о поступлениях и расходах в рамках объектов обособленного учета (сделок, менеджеров и т.д.).
Документ позволяет "перекидывать" остатки по видам запасов между конкретными значениями аналитик.

Анализ данных


Анализ данный осуществляется при помощи отчетов:

  • Продажи между организациями;
  • Расчеты между организациями;
  • Товары на комиссии;
  • Себестоимость товаров.
