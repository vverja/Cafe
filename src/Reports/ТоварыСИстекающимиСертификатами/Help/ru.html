Товары с истекающими сертификатами

Отчет предназначен для контроля сроков действия использующихся сертификатов по выбранной номенклатуре.

Сертификату могут быть присвоены состояния:

  • Действует - для действующих сертификатов;
  • Приостановлено - для сертификатов, которые выведены из действия досрочно;
  • Просрочено - для сертификатов, чей срок действия закончился сегодня.

Особенности отчета:

  • область действия сертификатов определяется по типу сертификата, номенклатуре, виду номенклатуры, товарам в наличии, складу и детализируется по характеристикам и сериям.
  • строки с просроченными сертификатами подсвечиваются красным цветом,
  • двойное нажатие по строке формирует отчет Сертификаты номенклатуры с истекшим сроком действия.
