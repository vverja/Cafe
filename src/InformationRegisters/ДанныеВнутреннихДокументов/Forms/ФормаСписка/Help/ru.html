Форма списка

Рабочее место предназначено для отображения и оформления (корректировки) документов в рамках перечня документов доступных на соответствующих закладках (Оформлено, К оформлению).

Отображение необходимого типа документа осуществляется при помощи отборов по операциям. При этом в зависимости от выбранной операции часть функционала рабочего места может быть недоступна (например подменю Печать пропадет при отборе документов которые не имеют печатных форм).

Переход в данное рабочее место может быть произведен по следующим командам:

  • Внутреннее товародвижение
  • Внутренние потребления товаров
  • Перемещения товаров
  • Сборки (разборки) товаров

При использовании отбора по операции, следующие списки команд будут перезаполнены в соответствии с выбранными типами документов:

  • Создать
  • Установить статус
  • Печать
  • Ввод на основании

Закладка К оформлению

Распоряжения к оформлению

На закладке К оформлению отображаются заказы являющиеся распоряжениями на оформление накладных, и документы (накладные) созданные на ордерных складах независимо и являющиеся распоряжениями на отгрузку со склада:

  • Заказ на сборку
  • Заказ на внутреннее потребление
  • Заказ на ремонт
  • Заказ на перемещение
  • Заказ материалов в производство
  • Заказ на производство
  • Сборка товаров
  • Внутреннее потребление товаров
  • Перемещение товаров
  • Передача материалов в производство

Отгрузка на ордерном складе оформляется документом Расходный ордер на товары

Имеется возможность отобрать распоряжения по состоянию оформления накладной, а также по состоянию расходных и приходных ордеров. В указанных ниже полях отбора приводится расшифровка вариантов отбора:

  • Оформление накладной
  • Расходные ордера
  • Приходные ордера

Команды Оформить по заказам и Оформить по отгрузке (приемке) могут создать одну накладную типа Перемещение товаров или Внутреннее потребление товаров по разным распоряжениям при соблюдении следующих условий:

  • Совпадают поля шапки документа;
  • Распоряжения не являются накладными и на их основании не были введены накладные;
  • Включена соответствующая функциональная опция (НСИ и администрирование - Склад и доставка - Внутреннее товародвижение - Перемещение по нескольким заказам или НСИ и администрирование - Склад и доставка - Внутреннее товародвижение - Внутреннее потребление по нескольким заказам);

Оформление по заказам

Распоряжения к оформлению отображаются в списке документов К оформлению, если накладные по данному распоряжению:

  • не оформлялись.
  • оформлены частично (не все запланированное заказом количество товаров оформлено накладными).
  • оформлены полностью, но товары не полностью отгружены со склада (не все расходные или приходные ордера на товары оформлены).

При выполнении команды Оформить по заказам будет оформлена накладная, в которую будет включено все неоформленные товары по выбранному в списке заказу (заказам).

Оформление по отгрузке

Для склада с ордерной схемой при отгрузке в списке документов К оформлению отображается:

  • накладная созданная независимо, отгрузка на складе по которой не завершена (не все расходные ордера на товары оформлены).
  • заказ, данные о собираемых, собранных и отгруженных со склада товарах которого не соответствуют количеству товаров, оформленному накладными по заказу.

Команда Оформить по отгрузке (приемке) позволяет перезаполнить накладную выбранного в списке распоряжения по фактической отгрузке. В этом случае товары которые не были фактически отгружены со склада будут удалены из накладной. Для позиций накладной, отгруженных со склада в меньшем количестве будет скорректировано (уменьшено) количество.

Для выбранных в списке К оформлению заказов команда позволяет создать накладную по заказам и заполнить ее всем фактически отгруженным количеством. Для заказов по которым уже оформлены накладные, система предлагает сопоставить данные фактической отгрузки накладным в отдельной форме Переоформление накладных по распоряжениям и решить создавать новую накладную или перезаполнить имеющиеся.
