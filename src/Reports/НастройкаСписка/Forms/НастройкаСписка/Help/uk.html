Настройка списку

Форма призначена для настройки списку (установки відборів, сортувань, умовного оформлення і групування). Форма відкривається по команді Ще - Настройка списку.

Настройка звіту:

  • Для настройки відбору в поле Відбір клацніть кнопку вибору. У формі, що відкрилася, вкажіть поля і умови відбору.
  • Вибрані умови відбору показуються в полі Відбір у виді представлення відбору, а у формі настройки нижче поля Групування додаються поля для управлінням відбором.
  • Для настройки сортування в полі Сортування клацніть кнопку вибору. У формі, що відкрилася вкажіть поля, за якими буде виконано сортування, і напрямок сортування.
  • Для застосування настройки списку, натисніть кнопку Завершити редагування.
  • Для скидання будь-якої настройки у відповідному полі натисніть кнопку Очистити.
  • Для відновлення стандартних настройок, заданих при конфігуруванні, виконайте команду Ще - Встановити стандартні настройки.
  • Якщо встановлені настройки передбачається використовувати в подальшому, то слід використовувати команду Зберегти настройки. Для використання збереженої настройки слід виконати команду Вибрати настройки.
