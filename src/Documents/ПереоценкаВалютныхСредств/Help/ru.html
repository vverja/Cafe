Расчет курсовых разниц

Документ предназначен для проведения переоценки валютных средств и расчетов с партнерами связанных с курсовыми разницами. Переоцениваются:

  • расчеты с клиентами,
  • расчеты с поставщиками,
  • денежные средства
    • безналичные валютные стредства
    • наличные валютные стредства
    • валютные средства у подотчетных лиц
    • денежные документы
    • переводы в пути.
  • расчеты по кредитам, депозитам, займам


Переоценка осуществляется независимо в валютах регламентированного и управленческого учета, в связи счем и могут возникать курсовые разницы по гривневым операциям в управленческой валюте, отличной от регламентированного учета.

Для возможности использования мультивалютного учета необходимо предварительное включение функциональнлй опции Несколько валют в разделе НСИ и администрирование - Предприятие - Валюты.

Для проведения переоценки доступны два режима:

  • На дату оплаты и на конец месяца.
  • На каждый день месяца.

Отчеты по отражению изменений валютных активов/пассивов

Отчеты по отражению изменений валютных активов/пассивов
  • Движение денежных средств;
  • Наличные денежные средства;
  • Остатки денежных средств;
  • Безналичные денежные средства;
  • Денежные средства у подотчетных лиц;
  • Расчеты между организациями (по продажам);
  • Расчеты между организациями (по закупкам);
  • Анализ денежных потоков;
  • Ведомость расчетов с партнерами.
