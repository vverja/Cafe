Для відкриття картки файлу використовуйте клавішу F2 або кнопку ("Змінити поточний елемент") командної панелі списку файлів.

Картка файлу містить різні відомості про файл і кнопки для виконання основних дій з файлом. Додаткові команди для роботи з файлом знаходяться в підменю "Ще" командної панелі форми файлу.

Команда Опис
Відкрити Відкриває файл зовнішнім додатком для перегляду.
Зберегти як Зберігає файл у вказаний каталог на жорсткому або мережевому диску.
Оновити з файлу на диску Оновлює файл в інформаційній базі обраним файлом на жорсткому диску.
Підписати Відкриває діалог для вибору сертифіката та встановлення підпису.
Додати ЕЦП з файлу Додає вибраний файл підпису до підписів файлу.
Зберегти разом з ЕЦП Зберігає файл у вказаний каталог на жорсткому або мережевому диску, а також зберігає підписи файлу - всі або вибрані, залежно від настройки "Дії при збереженні з ЕЦП"
Відкрити підпис Відкриває форму для перегляду інформації про підпис: кому видано сертифікат, хто встановив підпис, дату встановлення підпису, коментар до підпису
Перевірити Перевіряє істинність виділених підписів
Перевірити всі Перевіряє істинність всіх підписів
Зберегти Зберігає вибраний підпис у файл
Вилучити Вилучає виділені підписи
