(не використовується) Доступні для продажу товари

Звіт призначений для контролю станів запасів товарів та аналізу загального та доступного залишку товарів.

Особливості звіту:

  • В наявності на складі - загальна кількість товарів.
  • Відвантажується - кількість товарів, вже запланованих до відвантаження за документами Реалізація товарів.
  • Резерв-кількість зарезервована за замовленнями. При варіанті контролю залишків З урахуванням графіка відображається кількість запланована до відвантаження до межі контролю графіка.
  • Відокремлений резерв-кількість товарів по відокремлено забезпечуваних замовленнях.
  • Вільно для продажу-залишок на складі за мінусом кількості з граф Недоступне для продажу.
