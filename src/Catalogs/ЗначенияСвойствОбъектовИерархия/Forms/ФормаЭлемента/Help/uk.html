Призначена для вводу значення додаткового реквізиту (відомості).

У програмі передбачена можливість використовувати додаткові реквізити (відомості) декількох типів. Тип даних вибирається при настройці додаткових реквізитів (відомостей). При виборі типу даних Додаткове значення (ієрархія) програма пропонує ввести значення. При цьому для кожного додаткового реквізиту або відомості даного типу використовується свій список значень.

Введення значень

  • Вкажіть Найменування значення додаткового реквізиту (відомості).
  • За допомогою поля Входить в групу можна підпорядкувати новий елемент одному з наявних в списку. Для введення нового елемента вищого рівня натисніть Створити (клавішу F8). Для вибору з наявних елементів натисніть Показати всі (клавішу F4), виберіть потрібний елемент зі списку.
  • У разі якщо для додаткового реквізиту (відомості) включений прапорець Значення додатково характеризуються ваговим коефіцієнтом, введіть числове значення в поле Ваговий коефіцієнт.
    • Це поле надалі можна використовувати для аналізу у звітах. Логіку використання вагового коефіцієнта необхідно продумувати самостійно. Наприклад, для значень властивості "Надійність" можна задати вагу наступним чином: для значень від одиниці до трійки поставити вагу рівним нулю, а для значень чотири і п'ять рівним одиниці. При таких вагах у звіт можна швидко вивести надійних клієнтів (у яких вага значення більше нуля).
