Коммерческое предложение клиенту

Документ предназначен для регистрации в системе формального предложения (условий продаж) клиенту о покупке товаров.
Коммерческое предложение может быть создано на основе анализа первичного спроса, который регистрируется при первичном контакте с клиентом. Первичный спрос клиента может быть зарегистрирован в сделке с клиентом.

Использование коммерческого предложения регулируется функциональной опцией Коммерческое предложение в разделе НСИ и администирирование - Продажи - Оптовые продажи. Коммерческое предложение обязательно используется при включенной функциональной опции Управление сделками.

Создание коммерческого предложения


Новое коммерческое предложение может быть введено следующими способами.

  • Созданием на основании сделки, зарегистрированной с клиентом.
  • Добавлением нового коммерческого предложения в список коммерческих предложений.
  • Созданием на основании ранее оформленного коммерческого предложения.
  • Используя помощник продаж.

Создание коммерческого предложения на основании оформленного ранее
  • В форме коммерческого предложения нажмите на кнопку Создать на основании.
  • В появившемся списке выберите Коммерческое предложение клиенту.
  • Будет создано новое коммерческое предложение клиенту.


В новое коммерческое предложение будут скопированы все данные из ранее оформленного предложения. Ранее оформленное коммерческое предложение будет указано в качестве основания во вновь созданном коммерческом предложении.

Цепочку оформленных коммерческих предложений можно посмотреть в форме Структура подчиненности. Форма структуры подчиненности вызывается с помощью одноименной кнопки на панели навигации формы Коммерческое предложение клиента.

Анализ цен конкурентов


В командной панели формы коммерческого предложения выберите пункт Анализ цен. Будет сформирован отчет Анализ цен. В отчете будут автоматически отображены те позиции, которые указаны в коммерческом предложении. В отчете будет произведено сравнение цен, указанных в коммерческом предложении с зарегистрированными ценами конкурентов и поставщиков.

Цены поставщиков и конкурентов регистрируются в системе с помощью документа Регистрация цен поставщиков.

Согласование коммерческого предложения


Использование процесса согласования коммерческого предложения включается/отключается функциональной опцией Согласование коммерческих предложений в разделе НСИ и администрирование - Продажи - Оптовые продажи.

  • В форме коммерческого предложения нажмите на кнопку Создать на основании.
  • В появившемся списке выберите пункт Согласование коммерческого предложения клиенту.


В появившейся форме бизнес-процесса Согласование продажи укажите срок, до которого должно быть согласовано коммерческое предложение и введите текстовое описание обоснования отклонений от типовых условий продаж (важный клиент, большие объемы поставок и т.д.). Для запуска бизнес-процесса нажмите на кнопку Старт. Программа определит какие условия продажи (ценовые, финансовые или логистические) необходимо согласовывать и направит задачи тем исполнителям, которые отвечают за эти процессы.

Контроль состояния результатов согласования
  • Для просмотра процесса согласования используйте команду Согласование в панели навигации формы коммерческого предложения. В форме Согласование будет показан список направленных исполнителям задач и состояние их выполнения. Предусмотрена возможность просмотра просроченных задач, а также отбор задач по их статусам ( в процессе согласования, согласовано, несогласовано).
  • Для просмотра результатов согласования используйте отчет Результаты согласования. Отчет открывается с помощью одноименной команды в командной панели формы коммерческого предложения.

Создание заказа на основании коммерческого предложения
  • В карточке коммерческого предложения нажмите на кнопку Создать на основании.
  • В появившемся списке выберите Заказ клиента.

Оформление электронного документа


Если с контрагентом существует договоренность об электронном взаимодействии, то документ учетной системы может быть оформлен и отправлен получателю в электронном виде (группа команд ЭД).
Группа команд доступна при условии включенной функциональной опции Обмен электронными документами в разделе НСИ и администрирование - Обмен электронными документами - Управление обменом электронными документами.
