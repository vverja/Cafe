Призначена для вибору сертифікатів при шифруванні даних.

Шифрування даних

  • За допомогою посилання у верхній частині вікна можна переглянути об'єкт, який буде підписаний. Посилання приймає значення найменування об'єкта.
  • Особистий сертифікат - вказується обов'язково для можливості подальшої розшифровки;
    • виберіть особистий сертифікат зі списку;
  • Сертифікати тих, кому дозволено розшифровувати дані, крім Вас - список інших сертифікатів,
    які можуть бути використані для розшифровки.
    • За допомогою кнопок Підібрати (виберіть зі списку один або кілька сертифікатів) або Вилучити можна сформувати список для вибору.
  • Можуть використовуватися кілька сертифікатів, наприклад, якщо необхідно зашифрувати дані для декількох користувачів.
  • Для підтвердження вибору натисніть кнопку Зашифрувати.

Див. також:

  • Настройка електронного підпису та шифрування.
