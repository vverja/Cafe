Архівування та вилучення чеків ККМ

Форма призначена для вилучення або архівування чеків ККМ в ручному режимі. Для автоматичного вилучення чеків ККМ можна скористатися регламентним завданням Вилучення чеків ККМ.

  • Для заповнення списку кас ККМ, за якими будуть вилучатися/архівуватися чеки ККМ необхідно вказати період, за який відбуватиметься вилучення/архівування. У табличній частині відобразиться кількість чеків ККМ за вказаний період по кожній касі ККМ.
  • Для вилучення/архівації чеків необхідно вибрати каси ККМ, за якими буде відбуватися вилучення або архівування, вказати дію Вилучення чеків ККМ або Архівування чеків ККМ і натиснути кнопку Виконати.
