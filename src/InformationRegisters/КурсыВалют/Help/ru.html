Предназначен для хранения и просмотра курсов валют.

Открывается по соответствующей команде в панели навигации валюты.

Если нет возможности загружать курсы из сети Интернет, их можно ввести вручную.

В списке выводится:

  • Дата курса;
  • Валюта - символьный код валюты, принятый международным стандартом;
  • Курс;
  • Кратность.

Ввод курса валюты вручную

  • Нажмите Создать, 
  • выберите из списка Валюту (соответствует списку Валюты).
  • введите Дату курса или выберите с помощью календаря, 
  • введите Курс,
  • введите Кратность - по умолчанию равна 1. Если необходимо задать точность курса валюты больше четырех знаков дробной части, то необходимо указать количество котировочных единиц.
