Оформление заказа с помощью помощника


На первой странице помощника можно выполнить следующие действия:

  • Если соглашением допускается, то необходимо выбрать контрагента партнера на которого будет оформлен заказ. Если по соглашению допускается работа только с одним контрагентом, то поле Контрагент будет заполнено и недоступно для редактирования.
  • Если соглашением допускается, то необходимо уточнить форму оплаты заказа: Наличная, Безналичная, Платежная карта. Если соглашением определена форма оплаты, то поле Форма оплаты будет заполнено и недоступно для редактирования.
  • Указать Приоритет заказа.
  • Далее необходимо указать Адрес доставки - поле можно заполнить из выпадающего списка адресом партнера.
  • Можно указать желаемую дату отгрузки и установить флажок Не отгружать частями. Если флажок установлен, то отгрузка заказа произойдет, только если весь товар из заказа будет в наличии.
  • В поле Дополнительная информация можно указать произвольную информацию.
  • Командой Далее можно перейти на следующую страницу помощника.
  • Командой Отмена можно прервать работу помощника и закрыть его.


На второй странице помощника можно проверить товарный состав заказа (номенклатура, количество, цена, суммы и т.д.) и ознакомиться с рассчитанными скидками. Если все в порядке, то с помощью команды Далее можно перейти на следующую страницу помощника. В противном случае с помощью команды Назад можно вернуться на предыдущую страницу помощника.

На третьей странице помощника можно проверить условия, на которых будет оформлен заказа (адрес и дата доставки, форма оплаты и т.д.). Если все в порядке, то с помощью команды Оформить можно зарегистрировать в системе заказ. В противном случае с помощью команды Назад можно вернуться на предыдущую страницу помощника.

При оформлении заказа доступна возможность выбора: "сохранить как черновик" или "отправить на согласование менеджеру" (по умолчанию). В обоих случаях возможно дальнейшее редактирование заказа. Заказ становится недоступным для редактирования только в случае согласования менеджером, а к согласованию менеджер приступит только в случае если заказ отправлен ему на согласование.
