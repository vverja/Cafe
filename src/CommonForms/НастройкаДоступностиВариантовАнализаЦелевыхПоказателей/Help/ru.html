Настройка доступности вариантов анализа сотрудникам подразделения

В форме настроек установите курсор на то подразделение, сотрудникам которого надо обеспечить доступ к вариантам анализа целевого показателя. В правой части списка будет показан список целевых показателей и вариантов их анализа. Будет показано текущее состояние доступности вариантов анализа.

  • Если флажок установлен, то вариант анализа доступен для всех сотрудников подразделения.
  • Если флажок установлен и закрашен цветом, то вариант анализа доступен не всем сотрудникам подразделения. Доступность установлена хотя бы по одному сотруднику.
  • Если флажок не установлен, то вариант анализа не доступен ни одному сотруднику подразделения.

Администратор может установить/изменить доступность сотрудников подразделения как к конкретному варианту анализа, так и ко всем вариантам анализа целевого показетеля. Можно предложить следующий порядок действий.

  • Установите доступность для целевого показателя.
  • Уточните доступность для вариантов анализа. Снимите флажки у тех вариантов анализа, которые не должны быть доступны сотрудникам данного подразделения.
  • Если по целевому показателю доступны не все варианты анализа, то информация о доступности показывается закрашенным флажком.

Для каждого пользователя можно уточнить доступность по каждому варианту анализа. Для этого необходимо в списке сотрудников подразделения выбрать нужного сотрудника и уточнить его доступ к вариантам анализа целевых показателей (снять флажки у тех вариантов, которые ему недоступны). Аналогичным образом следует настроить доступность вариантов анализа для сотрудников других подразделений. После проведения всех настроек следует нажать на кнопку Записать и проверить правильность установки параметров доступа с помощью отчетов.

Контроль доступности вариантов анализа для сотрудников

В командной панели формы настройки выберите Отчет по доступу (сотрудникам подразделения).
Информация в отчете сгруппирована по целевым показателям, вариантам анализа, подразделениям и сотрудникам. Для каждого варианта анализа показывается список сотрудников подразделения, которым он доступен. Используя этот отчет можно получить ответы на следующие вопросы.

  • Варианты анализа доступные конкретному сотруднику - установите отбор по конкретному сотруднику в поле Пользователь.
  • Доступность анализа целевого показателя для сотрудника - установите отбор по целевому показателю в поле Цель.
  • Доступность варианта анализа сотруднику - установите отбор по варианту анализа. Информация о подразделениях, которым будет доступен данный вариант анализа выводится в виде группировки колонок в отчете.

Сохранение настроек

Пользователь может сохранить настройки двумя способами.

  • Записать и закрыть форму (Ctrl+Enter).
  • Записать без закрытия формы (Ctrl+S).

Пользователь также может отказаться от сохранения настроек, выполнив команду Отмена. Если настройки производились, то система предупредит пользователя, о том, что при отмене произведенные настройки будут утеряны. Пользователь может отказаться от сохранения настроек (настройки не сохранятся, форма будет закрыта) или продолжить редактирование и сохранение настроек.
