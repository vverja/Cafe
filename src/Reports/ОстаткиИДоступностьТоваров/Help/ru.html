Остатки и доступность товаров

Отчет предназначен для контроля состояний запасов товаров и анализа общего и доступного остатка товаров.

Особенности отчета:

  • В наличии - общее количество товаров.
  • В резерве - количество товаров, зарезервированных по заказам на отгрузку или уже запланированных к отгрузке по документам реализации товаров.
  • Доступно сейчас - количество товаров, свободное для продажи, равно общему количество товаров за вычетом количества в резерве.
  • Приход - количество товаров ожидаемое к поступлению на склад по заказам на поставку.
  • Расход - количество товаров ожидаемое к отгрузке со склада по заказам к отгрузке.
  • Остаток - ожидаемый остаток товаров, равно общему количеству товаров в наличии за вычетом ожидаемого прихода и ожидаемого расхода товаров.
  • Доступно по датам - количество товаров, доступное для продажи, начиная с определенной даты, рассчитано с учетом общего остатка и графика ожидаемых поступлений и отгрузок.

Доступное количество товаров может увеличиваться с даты очередного ожидаемого поступления товаров.

Расшифровка по документам-основаниям

Графы Приход и Расход могут быть расшифрованы по документам-основаниям.

Заказы на отгрузку с вариантами обеспечения Не обеспечивать, К обеспечению не ограничивают доступность товаров и не учитываются в ожидаемом расходе.
