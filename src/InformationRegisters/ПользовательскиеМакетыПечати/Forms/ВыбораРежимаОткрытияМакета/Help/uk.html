Призначена для вибору режиму відкриття макета.

Для відкриття макету списку Макети друкованих форм:

  • натиснути клавішу Enter - необхідно вибрати режим відкриття макета;
  • використовувати подвійне натисненням лівою кнопкою миші необхідно вибрати режим відкриття макета;
  • натиснути Відкрити;
  • натиснути Змінити.

Варіанти відкриття макета:

За допомогою перемикача виберіть режим відкриття макета за подвійним натисканням кнопки миші (клавіші Enter):

  • Тільки для перегляду - макет не буде помічений, як зайнятий Вами для редагування і відкриється зовнішнім додатком в режимі "тільки перегляд".
  • Для перегляду та редагування - макет буде помічений, як зайнятий Вами для редагування і відкриється зовнішнім додатком з можливістю внесення і збереження змін.

Припинення виведення повідомлення

  • Для того щоб більше не показувати це повідомлення, увімкніть прапорець Більше не питати.
  • Змінити цю настройку потім можна за допомогою команди Ще-Задати дію при виборі макета друкованої форми в списку макетів друкованих форм або з персональних настройок, вкладка Головне, посилання Задати дію при виборі макета друкованої форми.
