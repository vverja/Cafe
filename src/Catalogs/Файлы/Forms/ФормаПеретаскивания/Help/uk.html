Призначена для завантаження файлів і папок в список файлів за допомогою перетягування мишею.

Завантаження файлів і папок за допомогою перетягування мишею

  • Якщо потрібно завантажити один або кілька файлів (папок), можна використовувати перетягування мишею - просто перетягніть файл (папку) або кілька обраних файлів (папок) мишкою з Провідника Windows в потрібну папку списку файлів програми.
  • Один файл відразу міститься в список файлів, автоматично створюється картка файлу.
  • При перетягуванні декількох файлів, папки або декількох папок необхідно заповнити відомості:
    • Файли, які перетягувалися мишею для завантаження, виводяться у вигляді списку. Папки, які перетягувались мишею для завантаження, виводяться у вигляді дерева. При цьому можна розкрити кожну папку і переглянути файли, які в неї входять.
      • За допомогою прапорців можна вказати папки і файли, які будуть завантажені в програму.
    • У полі Папка автоматично проставляється обрана папка в списку файлів, в яку будуть завантажені файли з диска на комп'ютері. При необхідності можна вибрати зі списку іншу папку;
    • Для економії місця на диску увімкніть прапорець Вилучати файли після додавання (по умовчанню вимкнено);
    • Напишіть Опис для файлів, що створюються;
    • Увімкніть прапорець, для того щоб Зберігати версії (рекомендується, по умовчанню прапорець включений);
    • Для текстових файлів можна Вибрати кодування зі списку.
  • Для підтвердження натисніть Завантажити файли.
