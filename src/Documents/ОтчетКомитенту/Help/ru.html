Отчет комитенту о продажах

Документ Отчет комитенту о продажах регистрирует факт продажи товаров комитента. Документ может быть оформлен только по тому соглашению, в котором указан вид операции Прием на комиссию.

Информация в журнале разделена на закладки:

  • Отчеты комитентам - на закладке отображается информация о всех оформленных отчетах комитентам.
  • Распоряжение на оформление - отображается информация о тех продажах комиссионного товара, по которым, еще не отчитались за проданные товары перед комитентом (не оформили документ Отчет комитенту о продажах).
