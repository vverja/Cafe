Документ предназначен для корректировки назначения товаров, поступающих на склад под конкретный заказ, либо требуемых для конкретного заказа (заказ клиента, заказ поставщику, заказ на перемещение, заказ на сборку разборку).

Документ может быть оформлен только в том случае, если в разделе НСИ и администрирование - Склад и доставка - Обеспечение потребностей включена функциональная опция Обособленное обеспечение заказов.

Документ может быть оформлен вводом на основании из заказа, либо командами Зарезервировать под назначение или Снять резерв по назначению. Документ корректировка назначения товаров заполняется товаровами которые поступили, либо которые требуются под данный заказ, в зависимости от поля действие документа.

Для корректировки назначения необходимо выполнить действия:

  • Если поступивший под заказ товар должен поступить как свободный товар на склад, то необходимо из заказа выполнить команду Обеспечивать - Снять резерв по назначению, либо создать на основании заказа документ корректировка назначения товаров, выбрать действие Снятие резерва и заполнить документ с помощью команды заполнения.
  • Если заказ необходимо обеспечить со свободных остатков, то необходимо из заказа выполнить команду Обеспечивать - Зарезервировать под назначение, либо создать на основании заказа документ корректировка назначения товаров, выбрать действие Резервирование и заполнить документ с помощью команды заполнения.
  • Если заказ необходимо обеспечить не только со свободных остатков, но еще и из других заказов (переместить обособленный резерв другого заказа на текущий), то необходимо из заказа выполнить команду Обеспечивать - Зарезервировать под назначение и установить флаг Использовать резервы других обособленных заказов, либо создать на основании заказа документ корректировка назначения товаров, выбрать действие Резервирование и корректировка и заполнить документ с помощью команды заполнения.


Оформление документа возможно при условии, что заказ имеет статус К выполнению и в поле Действия хотя бы в одной строке указано Обеспечивать обособленно.
