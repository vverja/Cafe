Форма подбора товаров

Форма предназначена для быстрого отбора товаров в документ корректировки налогового назначения запасов.

  • Если в документе предварительно установлено налоговое назначение, то при открытии формы автоматически будут заполнены соответствующие фильтры и будет установлен отбор товара.
  • По отобранному товару показывается доступное количество по выбранному налоговому назначению.
  • После отбора необходимо отметить флажками те позиции которые будут перенесены в документ, а так же установить требуемое для корректировки количество товара.
  • По команде Перенести в документ отмеченные позиции будут добавлены в документ корректировки.
