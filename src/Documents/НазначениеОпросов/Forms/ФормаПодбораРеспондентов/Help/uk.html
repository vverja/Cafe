Призначена для підбору респондентів з обраного списку програми.

Відкривається по кнопці Підібрати з документа Призначення опитувань.

Вибір респондентів

  • Виділіть один або кілька елементів списку, натисніть Вибрати.
  • Для вибору одного елемента також можна скористатися подвійним натисненням кнопки миші.

Див. також:

  • Робота з формами.
