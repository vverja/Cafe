У даній панелі здійснюється настройка електронної пошти, заміток, нагадувань і бізнес-процесів. Доступні команди:

  • Поштовий клієнт - увімкніть прапорець, для того щоб використовувати можливості вбудованого в програму поштового клієнта для взаємодій з допомогою електронних листів (e-mail).
  • Вихідні листи у форматі HTML - увімкніть прапорець, для того щоб редагувати вихідні листи у форматі HTML.
  • Дзвінки, SMS, зустрічі і заплановані взаємодії- увімкніть прапорець, для того щоб реєструвати в програмі дзвінки, зустрічі, повідомлення SMS і заплановані взаємодії.
  • Ознаки "Розглянуто" і "На контролі" - увімкніть прапорець, для того щоб можна було ставити позначки про розгляд вхідних листів, зустрічей, телефонних дзвінків і запланованих взаємодій, а також для постановки на контроль вихідного листа та повідомлення SMS. Цей прапорець корисно використовувати для організації спільної роботи в програмі.
  • Адреса публікації інформаційної бази в Інтернеті- адреса інформаційної бази на веб-сервері. Використовується при формуванні навігаційних посилань на об'єкти інформаційної бази в електронних листах.
  • Настройка системного облікового запису - перехід до настройки основного облікового запису електронної пошти для відправки повідомлень з програми.
  • Облікові записи електронної пошти - адміністрування всіх облікових записів електронної пошти.
  • Настройка відправки SMS- перехід до настройки параметрів відправки SMS-повідомлень з програми.
  • Замітки - увімкніть цей прапорець, для того щоб можна було використовувати замітки - довільну текстову інформацію, для якої не виходить підібрати якого-небудь більш відповідного довідника або документа, і яка більше нікому не повинна бути доступна. Замітки можуть бути довільними або по предмету, посилаючись, наприклад, на елемент довідника, документ, або інший об'єкт програми.
  • Нагадування - увімкніть цей прапорець, для того щоб можна було вводити нагадування, які спрацьовують в призначений час. Вкажіть час і предмет, з приводу якого в програмі в зазначений час буде виведений список нагадувань. Нагадування можуть бути довільними або по предмету, посилаючись, наприклад, на елемент довідника, документ, або інший об'єкт програми.
  • Анкетування - увімкніть цей прапорець, для того щоб став доступним відповідний розділ програми, у якому можна готувати опитування, складати анкети, проводити опитування серед респондентів, а також і аналізувати результати опитувань.
  • Бізнес-процеси та задачі - увімкніть прапорець, для того щоб став доступним розділ програми Бізнес-процеси і задачі, який передбачає використання бізнес-процесів, що запускаються, і дозволяє роботу з задачами. Також стає доступним ряд інших команд, за допомогою яких можна запустити бізнес-процеси і використовувати задачі:
    • Ролі і виконавці задач- настройка ролей для адресації задач групам виконавців.
    • Підпорядковані бізнес-процеси і задачі - увімкніть прапорець для обліку бізнес-процесів, які можна створювати в рамках задач інших бізнес-процесів.
    • Змінювати запущені бізнес-процеси - увімкніть прапорець, для того щоб можна було вносити зміни в задачі запущених бізнес-процесів.
    • Дата початку задач - увімкніть прапорець, для того щоб дозволити виконавцям заповнювати поле Дата початку та сортувати по ньому список своїх задач.
    • Дата і час в строках задач - увімкніть прапорець, для того щоб можна було вводити строки задач з точністю до хвилин.
    • Повідомляти про прострочені задачі поштою - увімкніть прапорець, для того щоб сповіщати виконавців про прострочені задачі за розкладом, після цього стає доступною команда
      • Настроїти розклад - виконайте команду для настройки розкладу розсилки повідомлень.
    • Повідомляти виконавців про нові задачі поштою - виконайте команду для настройки розкладу розсилки повідомлень, після цього стає доступною команда
      • Настроїти розклад - виконайте команду для настройки розкладу розсилки повідомлень.
