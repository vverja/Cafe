Схема забезпечення

Схема забезпечення призначена для встановлення основного способу забезпечення потреб на одному чи декількох складах, для цілей планування забезпечена. Настроєна схема забезпечення може бути призначена номенклатурній позиції або виду номенклатури.

Можливість використання схем забезпечення з'являється при включеній функціональної опції Розширене в розділі НДІ і адміністрування – Склад і доставка - Забезпечення потреб.

Якщо процес забезпечення потреб включає декілька послідовних етапів, коли номенклатурні позиції надходять на один склад, потім переміщуються на інші склади і т.д., доречно настроїти схему забезпечення. Така схема забезпечення буде включати в себе інформацію про способи забезпечення потреб кожного етапу (кожного задіяного складу).

При включеній функціональної опції Формати магазинів у розділі НДІ і адміністрування – CRM і маркетинг - Маркетинг є можливість встановити основний спосіб забезпечення потреб на всі магазини (склади) одного формату.

Якщо спосіб забезпечення потреб в окремій номенклатурній позиції або у її характеристиці на складі відрізняється від способу, встановленого за схемою забезпечення, то такий спосіб можна встановити для позиції безпосередньо на робочому місці Параметри забезпечення потреб,

Для встановлення способів забезпечення потреб передбачено два режими відображення складів: за форматами і по ієрархії. Список За форматами необхідно використовувати для установки способу забезпечення для всіх магазинів (складів), які належать одному формату.

Передбачено індивідуальне зазначення способу забезпечення для вибраного рядка списку, а також групове заповнення способу забезпечення по команді Заповнити спосіб забезпечення.

Якщо склад не задіяний в схемі забезпечення, виключити його зі схеми забезпечення дозволяє команда Очистити спосіб забезпечення.
