Призначена для зазначення параметрів файлової або клієнт-серверної бази при створенні початкового образу РІБ. При настройці синхронізації РІБ (зовнішні файли) необхідно вказати, де буде створено початковий образ РІБ. Якщо файли зберігаються в томах, також буде потрібно вказати шлях до файлу, в який буде записаний архів (.zip) з усіма файлами в томах.

Відкривається по команді Створити початковий образ з Помічника створення синхронізації даних при настройці синхронізації РІБ (зовнішні файли). Також можна в списку Синхронізація даних за допомогою кнопки відкрити настройку синхронізації РІБ (зовнішні файли, натиснути кнопку Створення початкового образу підпорядкованого вузла розподіленої інформаційної бази.

Параметри інформаційної бази

Вкажіть за допомогою перемикача, де буде створюватися початковий образ інформаційної бази:

  • На даному комп'ютері або на комп'ютері в локальній мережі.
    • Задайте Параметри інформаційної бази:
    • У полі Мова (Країна) виберіть потрібне значення зі списку. По умовчанню пропонуються параметри, які були задані при створенні синхронізації даних;
    • Задайте Повне ім'я файлової бази:
      • Для сервера платформи під управлінням Microsoft Windows - за допомогою кнопки вкажіть шлях до папки, в якій буде розміщений файл.

або

      • Для сервера платформи під управлінням Linux - напишіть шлях до папки, в якій буде розміщений файл початкового образу.
  • На сервері.
    • Задайте Параметри інформаційної бази:
      • Вкажіть Кластер серверів;
      • Напишіть Ім'я інформаційної бази в кластері.
      • У полі Захищене з'єднання виберіть значення зі списку (Вимкнено, Встановлення з'єднання, Постійно);
      • Виберіть зі списку Тип СУБД (Microcoft SQL Server, PostgreSQL, IBM DB2, Oracle Database);
      • Вкажіть Сервер бази даних.
      • Напишіть Ім'я бази даних;
      • Напишіть ім'я Користувача бази даних і Пароль користувача;
      • Виберіть Зсув дат (по умовчанню 0);
      • У полі Мова (Країна) виберіть потрібне значення зі списку. По умовчанню пропонуються параметри, які були задані при створенні синхронізації даних;
      • При необхідності увімкніть прапорець Встановити блокування регламентних завдань (по умовчанню вимкнено).

Шлях до архіву з файлами томів

  • Вкажіть Повне ім'я архіву з файлами томів:
    • Для сервера платформи під управлінням Microsoft Windows- вкажіть за допомогою кнопки шлях до каталогу

або

    • Для сервера платформи під управлінням Linux- напишіть шлях до папки, в якій буде розміщений архів з файлами томів.

Для продовження натисніть Створити початковий образ.

Через деякий час програма виводить повідомлення про успішне створення початкового образу підпорядкованого вузла розподіленої інформаційної бази. Натисніть ОК для повернення в програму.

Для того щоб завантажити файли в томах в базі філії, при першому запуску необхідно виконати команду Розмістити файли початкового образу панелі дій розділу Органайзер, вказавши шлях до файлу архіву з файлами в томах.
