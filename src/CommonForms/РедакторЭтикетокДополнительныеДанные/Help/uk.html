Форма додаткових даних

Форма призначена редагування списку додаткових даних етикеток і цінників.

Для додавання нового поля з типом "Додатково" служить кнопка "Додати". Для редагування служить команда "Змінити". Для видалення поля необхідно натиснути кнопку "Вилучити". Для прийняття змін у списку додаткових необхідно натиснути "Записати і закрити".
