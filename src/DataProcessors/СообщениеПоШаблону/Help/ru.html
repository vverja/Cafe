Сообщение по шаблону

Помощник предназначена для отпраки электронных писем и SMS сообщений с использованием шаблонов.
Процедура отправки осуществляется в пошаговом режиме:

Выбор шаблона

Настройка параметров шаблона

Предварительный просмотр

Настройка адресатов и редактирование сообщений

Выбор шаблона

На первой странице помощника выбирается шаблон для сообщения. Причем, если сообщение создается на основании документа или справочника, выбор будет ограничен теми шаблонами, для которых установлен признак того, что они вводятся на основании данного типа документа или справочника.
В нижней части окна отображается текст шаблона, на котором выполнено позиционирование в списке.

Настройка параметров шаблона

На второй странице помощника заполняются параметры шаблона. В случае, если сообщение создается на основании документа или справочника, параметр соответствующего типа будет автоматически заполнен данным документом или справочником.

Пример. Сообщение создается по шаблону, предназначенному для оповещения клиентов о измненении статуса заказа. В качестве параметра сообщения будет предложено указать документ Заказ клиента, у которого изменился статус. После указания данного параметра в текст сообщения автоматически будут подставлены ФИО контактного лица заказа, его номер и текущий статус заказа.

Предварительный просмотр

На третьей странице помощника можно ознакомится с подготовленным сообщением. Для создаваемого письма будут показаны тело письма, тема письма и прикладываемые печатные формы. Для сообщения SMS будет показан текст сообщения.

Настройка адресатов и редактирование сообщений

На четвертой странице помощника осуществляется уточнение учетной записи из доступных пользователю, от лица которой будет отправлено письмо/сообщение. Также на данной странице производится уточнение адресатов, выбор которых будет предложен исходя из параметров выбранного шаблона сообщения.

Если в предложенном списке нет нужно адресата, то его можно добавить, воспользовавшись одноименной командой панели списка адресатов (Подобрать из адресной книги).

Пример. Если одним из параметров является документ Заказ клиента, то в качестве возможный адресатов будет предложены контактное лицо, контрагент и клиент заказа, а также контактные лица всех родительских документов. Контактное лицо заказа будет автоматически помечено как выбранный адресат.

При необходимости сообщение может быть дополнительно отредактировано непосредственно перед отправкой, если установлено значение переключателя Отредактировать сообщение перед отправкой (установлено по умолчанию).
При выполнении команды Готово будет открыта форма документа Электронное письмо исходящее или Сообщение SMS.
Если редактирование не требуется, то необходимо установить значение переключателя Отправить сразу. В этом случае при выполнении команды Готово будет выполнена отправка сформированного сообщения.
