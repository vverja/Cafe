Помічник інтерактивної синхронізації даних призначений для синхронізації даних з можливістю попереднього перегляду і зіставлення даних, що завантажуються. Помічник дозволяє виконувати зіставлення об'єктів двох інформаційних баз, між якими настроена синхронізація даних. Після виконання зіставлення виконується завантаження даних і подальше вивантаження даних.

Режим інтерактивної синхронізації даних слід використовувати у випадку, коли до початку синхронізації один і той же об'єкт міг бути введений незалежно в обох обмінюються ІБ. Іншими словами, інтерактивний режим синхронізації даних слід вибирати при першій синхронізації даних між ІБ, облік в яких вже ведеться. Оскільки ймовірність існування в таких ІБ одних і тих же об'єктів дуже висока. У цьому випадку інтерактивний режим дозволить такі об'єкти зіставити вручну.

Відкрити помічник можна, натиснувши кнопку Синхронізувати дані, задавши додаткові параметри в списку Синхронізація даних.

Параметри підключення

  • На першій сторінці помічника слід вибрати вид транспорту повідомлень обміну (можна змінити вже наявний варіант підключення, він вказаний на цій сторінці), для цього натисніть Настроїти, при необхідності можна перейти в каталог обміну інформацією і переконатися у фізичному існуванні необхідних файлів або перевірити параметри транспорту повідомлень обміну.
  • Для переходу до наступної сторінки натисніть кнопку Далі.

Зіставлення даних

  • На другій сторінці помічника відображається статистика зіставлення об'єктів. Перед відкриттям сторінки програма виконує спробу зіставити дані автоматично.
  • Автоматичне зіставлення виконується за заданими по умовчанню полями зіставлення, при цьому дотримуються найсуворіші правила зіставлення. Рекомендується виконати перевірку результатів автоматичного зіставлення даних. У списку виводиться:
    • Дані, Всього прийнято, шт.
    • - є питання по зіставленню об'єктів;
    • У цій програмі, шт.
    • Залишилося зіставити, шт.
    • Зіставлення, %
  • По команді Виконати зіставлення відкривається вікно для перегляду завантажених даних і зіставлення об'єктів вручну. Зіставлені вручну об'єкти двох ІБ запам'ятовуються і при виконанні наступної синхронізації даних зіставляються автоматично. При використанні інтерактивного режиму завантаження рекомендується спочатку виконувати автоматичне зіставлення, а потім виконувати зіставлення вручну для об'єктів, які не були зіставлені автоматично.
  • Список не є динамічним, тому для того щоб побачити нові дані, отримані і зіставлені після відкриття списку, натисніть кнопку Оновити.
  • По команді Ще - Отримати дані виконується завантаження даних для виділених рядків таблиці. Завантаження даних треба використовувати в рідкісних випадках, коли потрібно зіставити взаємопов'язані таблиці даних.
  • Перераховані команди доступні для виклику з контекстного меню таблиці статистики зіставлення. При виклику з контекстного меню ці команди актуальні для безлічі виділених рядків таблиці.
  • Після того як пройшло зіставлення, програма запам'ятовує правила і ця сторінка вже не виводиться.
  • Натисніть кнопку Далі для переходу до синхронізації даних.

Дані для відправки

  • На третій сторінці помічника слід вказати дані для відправки в іншу програму. Якщо дані не вказувати, то по умовчанню будуть відправлені дані, змінені з моменту останньої сесії синхронізації.
  • По умовчанню перемикач встановлений в положення положенні Не додавати-в цьому випадку будуть відправлені дані згідно із загальними настройками.
  • Для додаткового вивантаження документів натисніть перемикач Додати документи.
    • У полі За період: Виберіть період, за який їх необхідно відправити.
    • У полі З відбором: за замовчуванням встановлено, що можна відправити Всі документи (посилання). Натисніть на посилання, для того щоб встановити умови відбору: введіть поле для відбору, встановіть Вид порівняння, виберіть Значення для порівняння. Умов може бути кілька, в цьому випадку вони групуються за допомогою логічних операцій і, або, Не.
  • Для того щоб відправити додаткові дані, увімкніть перемикач Додати дані.
    • Їх слід відібрати, для цього перейдіть за посиланням, яка по умовчанню приймає значення Додаткові дані не обрані. Підберіть Дані, що відправляються з відповідних списків, натисніть ОК. Після цього посилання приймає значення перерахування обраних даних.
  • Статистику за надісланими даними можна переглянути, натиснувши на посилання Звіт про склад даних, що відправляються.
  • Натисніть Настройка, для того щоб скористатися раніше збереженими настройками або зберегти знову введені.
  • Натисніть Далі, для того щоб почати синхронізацію даних з обраними параметрами.

Синхронізація даних завершена

  • На завершальній сторінці помічника програма виводить повідомлення про стан синхронізації:
    • Не вдалося отримати дані або Дані успішно отримані і т. д.
    • Не вдалося відправити дані або Дані успішно відправлені і т. д.
    • За посиланням Докладно можна перейти в журнал реєстрації і переглянути, як проходив процес синхронізації, дізнатися про помилки.
  • При необхідності можна настроїти розклад для синхронізації даних в автоматичному режимі (в тому випадку, якщо поточна синхронізація даних ще не включена ні в один з існуючих розкладів виконання синхронізації). Натисніть Налаштувати.
  • Для завершення роботи з помічником натисніть Готово.
