Отчеты по комиссии между организациями

Журнал предназначен для отображения документов, которые отражают состояние отчетов по комиссии о продажах и списании между организацими, входящими в состав предприятия в заданном отчетном периоде.

Оформление отчетов по комиссии


Для оформления нового отчета необходимо перейти на вкладку К оформлению и выбрать тип создаваемого документа Отчет по комиссии между организациями или Отчет по комиссии между организациями о списании.
В новом документе дата документа устанавливается равной текущей дате. При необходимости дату документа можно изменить.
В документе заполните список товаров, по которым необходимо отчитаться перед комитентом и цены, по которым товар передавался.
В поле Комитент укажите ту организацию, которая является владельцем товаров, а в поле Комиссионер укажите ту организацию, которая отчитывается за проданный (списанный) товар.

Печать нескольких документов передачи товаров из списка
  • В списке документов выделите те документы, которые Вы хотите распечатать, используя стандартные возможности платформы.
  • Нажмите на кнопку Печать и выберите форму печати документа.
  • Будут распечатаны все отмеченные документы. Каждый документ будет распечатан на отдельной странице.
