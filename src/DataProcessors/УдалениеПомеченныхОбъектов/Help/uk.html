Призначена для остаточного вилучення відмічених об'єктів.

В процесі роботи з програмою можна відмітити для вилучення об'єкти (наприклад, помилково введені). Відмічені для вилучення об'єкти можуть бути безповоротно вилучені користувачем з адміністративними правами.

Команда Вилучення відмічених об'єктів знаходиться в розділі Адміністрування - Підтримка та обслуговування.

Вилучення виконується з контролем посилальної цілісності. Це означає, що якщо на той чи інший об'єкт програми (документ, елемент довідника і т. д.) посилаються інші об'єкти, то вилучення об'єкта не буде виконано. Тим не менш, слід пам'ятати, що вилучення відмічених об'єктів - операція необоротна.

Вилучення раніше відмічених об'єктів можна виконати двома способами, які можна вибрати за допомогою перемикача:

  • Автоматичне вилучення всіх об'єктів - проводиться спроба автоматичного вилучення всіх об'єктів, відмічених на вилучення.
  • Вибіркове вилучення об'єктів - цей режим дозволяє попередньо вибрати об'єкти для вилучення.

Автоматичне вилучення всіх відмічених об'єктів

  • Натисніть Вилучити.
  • При виборі режиму програма здійснює пошук і вилучення всіх відмічених об'єктів. Через деякий час програма виводить повідомлення про кількість вилучених об'єктів, при цьому втручання потрібно, тільки якщо вилучення неможливе у разі виявлення посилань на ці об'єкти.

Вибіркове вилучення об'єктів

  • Натисніть Далі.
  • При виборі режиму програма формує список об'єктів, відмічених для вилучення.
    • У списку об'єкти згруповані за видом, по умовчанню усі об'єкти відмічені прапорцями для вилучення.
    • Знявши прапорці в списку, можна залишити в програмі ті об'єкти, які були відмічені на вилучення.
    • Подвійне натискання кнопки миші по об'єкту або за допомогою кнопки Змінити можна відкрити його і прийняти рішення про правомірність його вилучення.
  • Для остаточного вилучення об'єктів з програми натисніть Вилучити.
  • По закінченні фізичного вилучення об'єктів виводиться повідомлення про кількість вилучених об'єктів.
  • Якщо на об'єкти є посилання, то програма видає попередження і виводить перелік об'єктів, які не вдалося вилучити, а також об'єкти, які перешкоджають вилученню.

Дії при неможливості вилучення об'єктів

Список невилучених об'єктів складається з двох частин: зліва перелік об'єктів, праворуч список Місця використання.

  • Для остаточного вилучення об'єктів можна:
    • Подвійним клацанням у списку або за допомогою команди Ще - Змінити можна відкрити об'єкт, який перешкоджає вилученню, наприклад, для того щоб вручну вилучити в ньому посилання на потрібний об'єкт.
    • вказати посилання на інший елемент зі списку. Натисніть кнопку Замінити у всіх місцях використання, щоб перейти до заміни елементів. Для заміни посилань виберіть інший елемент замість того, що вилучяється.
    • Відмітити для вилучення всі пов'язані з елементом, що вилучається, об'єкти (в правій частині списку - Місця використання),
    • Зняти відмітку на вилучення з об'єкта, відміченого на вилучення.
  • Після цього натисніть Повторити вилучення, для того, щоб спробувати вилучити об'єкти ще раз.
  • Натисніть На початок, для того щоб вибрати інший спосіб вилучення.
  • Натисніть кнопку Закрити, для того щоб повернутися в програму без продовження спроб вилучення.
