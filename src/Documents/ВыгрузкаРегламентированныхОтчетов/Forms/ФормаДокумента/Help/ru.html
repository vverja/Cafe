Документ «Выгрузка регламентированных отчетов»

Документ «Выгрузка регламентированных отчетов» предназначен для выгрузки данных форм отчетности на электронный носитель (USB-флешдиск, дискету) или жесткий диск компьютера в форматах, утвержденными государственными органами для подачи отчетности в электронном виде.

Документ может быть создан двумя способами:

  • вводом вручную нового документа в журнале «Выгрузка регламентированных отчетов»
  • автоматически - при нажатии кнопка «Выгрузить» в форме регламентированного отчета.

Независимо от способа создания документа, в дальнейшем работа с документом выполняется одинаковым образом.

Номер и дата документа заполняются автоматически, и, в общем случае, изменять их нет необходимости.

В поле "Отчет" указывается выгружаемый регламентированный отчет.

Выгружаемые отчетные формы

В список «Выгружаемые отчетные формы» включаются отдельные отчетные формы, содержащиеся в регламентированном отчете (например, декларация и набор приложений к декларации как отдельные отчетные формы). Список заполняется автоматически при выборе (изменении) регламентированного отчета, указанного в шапке документа, либо при нажатии на кнопку "Заполнить список" командной панели списка.

Слева от наименования отчетной формы можно поставить отметку-флажок, и таким образом, указать, что эта отчетная форма будет выгружаться. При выборе регламентированного отчета в шапке документа (или при автоматическом создании документа выгрузки) флажки будут установлены автоматически исходя из заполненных данных регламентированного отчета.

«Установить отметки»/«Снять отметки» - кнопки позволяют установить/снять отметки сразу для всех отчетных форм.

Порядок выгрузки отчетных форм

Выгрузка данных регламентированной отчетности выполняется в 2 этапа:

  • формируются файлы выгрузки;
  • файлы выгрузки записываются на электронный носитель.

Для успешной выгрузки необходимо, чтобы регламентированный отчет был заполнен данными, а сами данные - сохранены в информационной базе.

Проверить заполнение регламентированного отчета можно, открыв его прямо из документа выгрузки.

Формирование файлов выгрузки

Формирование файлов выгрузки выполняется при нажатии кнопки «Сформировать файлы выгрузки» командной панели документа «Выгрузка регламентированных отчетов».

В процессе выгрузки будут извлекаться сохраненные данные регламентированного отчета. Если регламентированный отчет в момент выполнения выгрузки открыт, и в него внесены какие-либо изменения, для выгрузки будут извлекаться данные, сохраненные в информационной базе, а не те данные, которые отредактировны в открытом отчете. По этой причине перед выгрузкой рекомендуется сохранить отредактированный регламентированный отчет.

Если файлы выгрузки сформированы успешно - будет выдано соответствующее сообщение в окно сообщений

В случае, если в процессе выгрузки обнаружены ошибки в данных отчета, которые не позволяют выполнить выгрузку корректно, на экран будет выдан запрос вида «Не удалось выгрузить <наименование отчетной формы>. Прервать выгрузку?».

Если ответить «Да» - процесс выгрузки будет прерван и в окно сообщений будет выдана информация о том, какую отчетную форму не удалось выгрузить.

Если ответить «Нет» или вообще не отвечать на вопрос - в этом случае процесс выгрузки будет продолжен. По окончании процесса выгрузки в окно сообщений будет выдана информация об отчетных формах, которые не удалось выгрузить, и будет выдано дополнительное прикрепленное окно «Навигация по ошибкам», которое содержит список ошибок, обнаруженных в процессе выгрузки.

В процессе выгрузки документ «Выгрузка регламентированных отчетов» будет сохранен.

Навигация по ошибкам

Окно «Навигация по ошибкам» содержит таблицу, состоящую из двух колонок: наименование ошибочного поля и описание ошибки.

При выделении какой-либо строки в таблице, в правой части окна выдаются «координаты» ошибочного поля отчета.

После того, как все обнаруженные ошибки будут исправлены, можно выполнить выгрузку повторно.

Внимание! После исправления ошибок отчет следует сохранить в информационной базе!

Запись файлов выгрузки на электронный носитель

После того, как файлов выгрузки успешно сформированы, их можно будет записать на магнитный носитель - дискету или жесткий диск.

Для записи предназначена кнопка «Сохранить» командной панели документа «Выгрузка регламентированных отчетов».

При ее нажатии будет выдан диалог выбора места сохранения файлов выгрузки. После указания места сохранения необходимо нажать кнопку «Сохранить».
