Групова зміна реквізитів

Групова зміна реквізитів передбачена у двох варіантах:

  • в більшості списків програми передбачена команда Змінити виділені.
  • тільки для адміністратора: у розділі програми НДІ і Адміністрування - Підтримка та обслуговування - Коригування даних за командою Групова зміна реквізитів або у разі підключення в якості зовнішньої обробки - у відповідному розділі програми за командою Додаткові обробки - Групова зміна реквізитів.

Зміна виділених елементів списку 

Зміна значень реквізитів (а також додаткових реквізитів і відомостей) у обраних елементів списку.

Введення нових значень реквізитів вибраних елементів списку

  • Доступні для зміни поля вибраних елементів списку відображаються в рядках колонки Реквізит. Можна змінити відразу кілька реквізитів.
  • Для того щоб змінити значення реквізитів (а також додаткових реквізитів та відомостей), у потрібних рядках необхідно заповнити Нове значення. Для деяких реквізитів при введенні використовується редактор формул, відкрити який можна за допомогою кнопки .
  • Відкрийте рядок для введення за допомогою подвійного натиснення кнопки миші. Можна:
    • ввести значення для пошуку в списку;
    • за допомогою команди Показати всі (клавіші F8) вибрати значення з відповідного списку;
    • ввести нове значення в список за допомогою команди Створити.
  • При введенні нового значення автоматично включається прапорець .
  • Натисніть Змінити реквізити.

Очищення значень полів вибраних елементів списку

  • Для очищення значення реквізиту потрібно включити прапорець у відповідному рядку, а колонку Нове значення не заповнювати.
  • Натисніть Змінити реквізити.

Результат зміни з помилками

  • Список незмінених елементів списку виводиться в окремому вікні. У списку по кожному елементу наводиться причина помилки.
  • Натисніть Відкрити, щоб переглянути елементи, які не вдалося змінити.
  • Натисніть Повторити зміну, щоб спробувати повторно змінити елементи списку з тими ж параметрами.
  • Для виправлення помилки, можливо, доведеться змінити значення реквізитів перерахованих елементів на інші.

Вибір раніше збережених настройок щодо зміни реквізитів

  • Скористайтеся командою Раніше змінені реквізити,
  • З випадаючого списку виберіть потрібну настройку. У списку зберігаються настройки останніх 20 змін реквізитів.

Зміна заблокованих реквізитів

  • У списку можуть бути реквізити, по умовчанню недоступні для зміни і відображаються сірим кольором.
  • Ці реквізити є "ключовими" реквізитами, вони можуть вплинути на інші списки і документи програми. Тому вони захищені від випадкових змін.
  • При спробі введення нових значень програма пропонує їх розблокувати. У разі особливої необхідності їх зміни, можна підтвердити свій вибір, натиснувши на кнопку Так.

Особливості зміни проведених документів

  • При зміні проведених документів виконується їх перепроведення.

Настройка параметрів зміни реквізитів

  • Програма використовує настройки по умовчанню для зміни реквізитів. Натисніть кнопку Додаткові параметри для того щоб задати настройки.

Групова зміна реквізитів (для адміністратора)

  • Адміністратору для зміни доступні реквізити всіх довідників і документів програми, в тому числі табличні частини. Також передбачено більш точний вибір елементів для зміни реквізитів (таблиця Вибір елементів для зміни).

Вибір елементів для зміни

  • У полі Змінювати виберіть документ або довідник програми, в якому необхідно змінити елементи. За замовчуванням можна змінювати Всі елементи довідника або всі документи обраного виду.
  • Для перегляду обраних для зміни елементів скористайтеся посиланням Всі елементи.
  • За допомогою кнопки Додати умову відбору можна відібрати один або кілька елементів за будь-якими ознаками. При налаштованому відборі зміни виконуються тільки в обраних елементах.
  • Після додавання умови відбору посилання показує, скільки елементів за допомогою відборів вибрано для зміни.
  • Після вибору довідника або виду документа в таблиці Змінювані реквізити можна переглянути доступні для зміни реквізити, в тому числі і службові. Для того щоб показувати службові реквізити, натисніть кнопку Додаткові параметри і увімкніть прапорець.
  • Після вибору потрібних реквізитів скористайтеся стандартними командами для їх зміни.
