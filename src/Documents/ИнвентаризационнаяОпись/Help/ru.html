Инвентаризационная опись

Документ предназначен для получения регламентированной печатной формы Инвентаризационная опись (М-21). Генерируется автоматически помощником оформления складских актов для каждой организации, для которой оформляются складские акты.

Печатная форма М-21 заполняется по документам Пересчет товаров за указанный в документе Инвентаризационная опись период.
