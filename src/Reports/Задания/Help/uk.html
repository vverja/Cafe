Призначений для моніторингу бізнес-процесу Завдання.

Відкривається по команді Список завдань за період зі списку Звіти по задачам розділу Бізнес-процеси і задачі.

Варіанти звіту

  • Список завдань за період - призначений для контролю ходу виконання бізнес-процесів Завдання. У рядках звіту виводяться найменування бізнес-процесу, статус його завершення (Так/Ні), автор і дата старту, а також вказується, скільки днів тривав бізнес-процес, і скільки в процесі сталося циклів. Можна вивести всі, завершені або тільки незавершені завдання. Звіт формується за Період.
    • Можливий відбір по Періоду і по Автору.
  • Статистика циклів виконання-перевірки завдань- у звіті в наочній формі (у виді діаграми) представлено порівняння кількості циклів виконання-перевірки завдань по Авторам. У звіті необхідно задати Період.
    • Можна зробити відбір тільки по завершеним завданням і по Автору.
  • Середня тривалість виконання завдань- призначений для контролю строків виконання завдань. У цьому звіті в наочній формі (у виді діаграми) для аналізу представлені самі тривалі бізнес-процеси за певний Період. Звіт видається за завершеними завданнями.
    • Можна відібрати завдання за Період і по Автору.
  • Статистика результатів виконання завдань- в наочній формі (у виді діаграми) показує, скільки завдань виконується і скільки виконано. Звіт формується за певний Період.
    • Можна зробити відбір тільки по завершеним завданням і по Автору.
