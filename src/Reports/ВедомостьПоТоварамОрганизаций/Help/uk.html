Відомість по товарах організацій

Звіт призначений для контролю залишків товарів на складах підприємства в розрізі організацій.

Дані звіту доступні тільки при включеній функціональної опції Контролювати залишки товарів організацій у розділі НДІ і адміністрування - Підприємство - Організації.

Варіанти формування:

Відомість по товарах організацій

Звіт призначений для аналізу рухів товарних залишків на складах підприємства.
Дані звіту можуть бути деталізовані за видами запасів (наприклад, власний або комісійний) та кількістю товарів за одиницями вимірювання.

Доступні для продажу товари організацій

Звіт призначений для аналізу залишків товарів, доступних для продажу від імені обраної організації на складах підприємства.
