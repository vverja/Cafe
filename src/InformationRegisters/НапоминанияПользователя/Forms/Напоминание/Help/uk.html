У програмі можна вводити нагадування, вказавши час і предмет, з приводу якого в програмі в зазначений час буде виведений список нагадувань.

Предметом нагадувань можуть бути різні довідники, документи та інші об'єкти програми. Наприклад, нагадування можна ввести по конкретному партнеру, документу руху товарів і т.д., для того щоб не забути уточнити або дозаповнити будь-які дані.

Відкривається по команді Органайзер - Нагадати з різних об'єктів програми. При цьому можна ввести тільки одне нагадування для одного об'єкта.

Також можна ввести нагадування, пов'язане з будь-якою персональною подією або справою, в списку Мої нагадування по команді Створити. Таке нагадування може бути корисно, наприклад, для того щоб не забути вчасно привітати колегу з днем народження. Спільно з замітками нагадування добре підходять як електронна заміна "стікерів на краях монітора".

Введення нагадування

  • Відкоригуйте або напишіть текст нагадування. Якщо нагадування пов'язане з документом, довідником або іншим об'єктом програми, то в тексті нагадування по умовчанню буде підставлено його найменування.
  • У полі Нагадати виберіть один з варіантів:
    • У зазначений час - для зазначення точної дати і часу нагадування. Введіть дату вручну або виберіть з календаря за допомогою кнопки .
    • Через певний інтервал. Для цього виберіть зі списку для завдання інтервалу, через який нагадування спрацює. Також можна за допомогою клавіатури вказати довільну кількість годин, наприклад, через 10 годин.
    • Щодо предмета - для зазначення часу нагадування щодо дати у предметі нагадування, наприклад, щодо строку виконання у задачі. Це значення поля Нагадати відображається і пропонується програмою по умовчанню, коли у предмета існує щонайменше одне поле з датою в майбутньому часі. Виберіть зі списку, за скільки часу до Строку необхідно вивести нагадування. Якщо така дата в предметі змінюється, автоматично змінюється і строк нагадування. Якщо дата в предметі з якоїсь причини буде скинута, то строк нагадування буде вважатися минулим і нагадування буде виведено достроково.
    • Періодично-для періодичних нагадувань, наприклад, можна вибрати зі списку значення по понеділках в 9-00, або ввести інший розклад. Для введення свого розкладу виберіть зі списку значення за заданим розкладом і зробіть необхідні настройки.

Скасування нагадування

  • Для того щоб скасувати нагадування, натисніть Припинити.
  • Програма виводить повідомлення, в якому вимагає підтвердження. Для підтвердження натисніть Припинити. Для скасування своєї дії натисніть Не припиняти.
