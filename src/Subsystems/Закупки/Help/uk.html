Призначення підсистеми

Підсистема призначена для управління купівельною діяльністю підприємства, таких як замовлення постачальникам, надходження товарів і послуг, комісійні купівлі.

Настройка підсистеми

Перед початком роботи необхідно перевірити встановлення відповідних функціональних опцій у розділі НДІ і адміністрування - Купівлі:

  • Для можливості відстеження етапів оплати постачальникам, контролю стану поставок (погодження умов, авансів постачальникам, підтвердження поставок, оплат після поставок) необхідно включити функціональну опцію - Замовлення постачальникам
  • Для можливості оформлення документа поставки за кількома замовленнями постачальника необхідно включити функціональну опцію- Надходження по декількох замовленнях
  • Для можливості погодження замовлень постачальникам необхідно включити функціональну опцію - Погодження замовлень постачальникам
  • Для можливості оформлення документів поставки з можливістю вказівки кількох складів (без використання ордерної схеми) необхідно включити функціональну опцію - Надходження товарів на кілька складів
  • Для можливості оформлення операцій по роботі з комітентами необхідно включити функціональну опцію - Комісійні купівлі.
  • Для можливості обліку кодів УКТЗЕД та номерів ВМД необхідно включити функціональну опцію Імпортні купівлі
  • Для можливості деталізованого контролю процесів обробки замовлень постачальникам необхідно включити функціональну опцію - Статуси замовлень постачальникам
  • Для можливості відвантаження товарів тільки по тому замовленню, для якого він призначений, необхідно включити функціональну опцію Відокремлене забезпечення замовлень в розділі НДІ і адміністрування - Склад і доставка - Забезпечення потреб.

Робота підсистеми
  • Розділ Розрахунки з постачальниками призначений для вирішення завдань щодо реєстрації цін і асортименту постачальників, контролю термінів поставок товарів, витрат грошових коштів, а також своєчасності оплати постачальникам
  • Розділи Купівлі та Повернення та коригування призначені для безпосереднього оформлення документів поставок і повернень товарів, реєстрацій розбіжностей при прийманні товарів, аналізу причин збоїв поставок (відмов постачальників від поставок), обліку додаткових послуг і витрат при поставках товарів.

Аналіз розрахунків з постачальниками

Передбачена можливість проведення різного аналізу за допомогою звітів:

  • Відомість розрахунків з постачальниками
  • Заборгованість постачальникам
  • Картка розрахунків з постачальниками
  • Звірка розрахунків з партнерами
