План-фактный анализ оплат и начислений

Отчет предназначен для анализа отклонений по фактически произведенным оплатам и начислениям в рамках заключенных кредитных (депозитных) договоров с учетом комиссии.
Отчет доступен при включенной функциональной опции Договоры кредитов и депозитов в разделе НСИ и администрирование - Казначейство - Управление параметрами казначейства.

Отчет вызывается по команде План-фактный анализ:

  • из формы варианта графика;
  • из списка вариантов графиков – строится по указанному в списке варианту;
  • из формы договора – строится по варианту графика с отметкой Используемый;
  • из списка договоров - строится по варианту графика с отметкой Используемый.
