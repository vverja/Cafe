Призначена для настройки на даний сеанс сканування (ці настройки не будуть збережені). Відкривається з Помічника сканування за командою Настройка.

Залежно від можливостей сканера, а також довготривалих настройок сканування, деякі параметри можуть бути приховані. Наприклад, якщо перетворення в PDF вимкнено, то поміняти формат багатосторінкового зображення неможливо, по умовчанню буде обраний формат TIF.

Основне

Параметри стають доступними, якщо відповідні можливості є у сканера.

  • Розрізняюча здатність - дозволяє вибрати розрізняючу здатність сканування (DPI). Список доступних розрізняючих здатностей залежить від обраного сканера.
  • Кольоровість - вибір кольоровісті зображень, які будуть отримані зі сканера (Монохромне, Градації Сірого, Кольорове).
  • Поворот зображення - дозволяє вибрати один з варіантів повороту зображення після сканування. Для деяких моделей сканерів ця властивість недоступна.
  • Розмір паперу - дозволяє вибрати розмір паперу, який поміщена в сканер. Для деяких моделей сканерів ця властивість недоступна.
  • Двостороннє сканування - якщо встановлено, будуть відскановані обидві сторони паперу. Для деяких моделей сканерів ця властивість недоступна.

Для односторінкових зображень

  • Формат - дозволяє вибрати один з форматів, що підтримуються (BMP, JPG, GIF, PNG, TIF, PDF) для поміщення в інформаційну базу.
    • Тип відсканованого зображення - дозволяє вибрати один з форматів, що підтримуються (BMP, JPG, GIF, PNG, TIF) для зображень, які будуть отримані зі сканера. Опція доступна, якщо формат односторінкового зображення - PDF.
    • Якість - якщо вибраний формат JPG, то можна налаштувати його якість (100 - максимальна якість і максимальний розмір, 1 - мінімальна якість і мінімальний розмір.). По умовчанню - 100.
    • Стиснення - якщо вибраний формат TIF, можна налаштувати стиснення. Можливі варіанти: LZW, RLE, CCITT3, CCITT4, Без стиснення. По умовчанню: Без стиснення.

Для багатосторінкових зображень

Прапорець стає доступним для вибору, якщо в настройках сканування включено перетворення в PDF.

  • Формат - дозволяє вибрати один з форматів, що підтримуються (TIF, PDF) для поміщення в інформаційну базу. Застосовується коли обрана команда Прийняти всі як один файл.

Діалог сканера

  • Показувати діалог сканера - якщо прапорець включений, перед скануванням буде показаний діалог для розширеної настройки сканування (наприклад, вибір області сканування на аркуші паперу).
