Обладнання для підключення

Дана форма призначена для підключення і настройки зовнішнього обладнання, підключеного в довіднику Обладнання для підключення. Щоб підключити та настроїти обладнання необхідно виконати описані нижче кроки:

  • За допомогою команди Управління робочими місцями вибрати конкретне робоче місце, до якого буде підключатися торговельне обладнання або вказати список робочих місць.
  • Якщо підключення торговельного обладнання необхідно настроїти для декількох робочих місць, то встановіть прапори Всі робочі місця і Групувати по робочому місцю.
  • Для підключення і настройки торговельного обладнання встановіть прапор відповідного типу торговельного обладнання (Сканери штрихкоду, Фіскальні реєстратори, Термінали збору даних і т.д.).
  • Створіть новий запис для кожного типу торговельного обладнання. У діалоговій формі, що з'явилася, вкажіть драйвер, який використовується при підключенні обладнання.
  • Якщо драйвер обладнання для підключення не завантажений, то програма запропонує встановити драйвер обраного екземпляра торговельного обладнання.
  • Після завантаження драйвера відкриється форма настройки параметрів екземпляра обладнання. Необхідно перевірити правильність встановлених настройок і зберегти настройки.
  • Для перевірки підключеного обладнання настроєного на робочих місцях встановіть прапор Всі типи обладнання і перевірте список підключеного торговельного обладнання.

У будь-який момент можна відключити роботу користувача з конкретним екземпляром торговельного обладнання. Для цього у формі торговельного обладнання, що підключається, слід встановити прапор Не використовувати пристрій.
