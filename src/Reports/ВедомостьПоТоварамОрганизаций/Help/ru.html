Ведомость по товарам организаций

Отчет предназначен для контроля остатков товаров на складах предприятия в разрезе организаций.

Данные отчета доступны только при включенной функциональной опции Контролировать остатки товаров организаций в разделе НСИ и администрирование - Предприятие - Организации.

Варианты формирования:

Ведомость по товарам организаций

Отчет предназначен для анализа движений товарных остатков на складах предприятия.
Данные отчета могут быть детализированы по видам запасов (например, собственный или комиссионный) и количеству товаров по единицам измерения.

Доступные для продажи товары организаций

Отчет предназначен для анализа остатков товаров, доступных для продажи от имени выбранной организации на складах предприятия.
