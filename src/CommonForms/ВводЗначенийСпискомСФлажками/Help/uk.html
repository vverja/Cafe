Призначений для введення декількох значень в список для відбору.

Відкривається з закладки Відбори вікна Настройки звіту подвійним натисненням кнопки миші в стовпці Значення для операцій В списку, Не в списку, В групі зі списку, Не в групі зі списку.

Підбір значень

  • Для додавання декількох значень в список натисніть кнопку Підбір. Виберіть кілька значень з відповідного списку.
  • Для того щоб вибирати значення за одним, виконайте команду Додати меню Ще.
  • Для того щоб елементи списку потрапили в список значень для відбору, необхідно встановити відповідні прапорці (по умовчанню усі прапорці встановлені).

Підбір значень з буфера обміну

  • Натисніть кнопку , виберіть значення, натисніть кнопку Вставити в список.
