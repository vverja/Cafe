Настройка психологического округления с помощью конструктора


С помощью конструктора округления можно настроить и проверить правильность психологического округления.

  • В поле Точность округления вводится информация о применяемой точности округления.
  • В поле Вычитать из результата вводится то значение, которое надо вычесть из результата.
  • Для проверки в поле Пример вводится сумма, для которой хочется проверить применяемое округление.
  • Для переноса данных о психологическом округлении в форму вида цена необходимо нажать на кнопку Сохранить и закрыть.

Пример. Необходимо настроить психологическое округления так, чтобы в конце цены стояло 70 гривен. Точность округления указана до 100 гривен.
  • В поле Точность округления указываем 100.
  • В поле Вычитать из результата указываем 30.
  • В поле Пример вводим 13 865,89. Получаем цифру 13870.
