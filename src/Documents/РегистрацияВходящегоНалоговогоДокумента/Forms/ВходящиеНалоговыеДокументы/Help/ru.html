Форма списка

Документ "Регистрация входящего налогового документа" предназначен для регистрации документов приобретения товаров, работ, услуг в Реестре выданных и полученных налоговых накладных.

Регистрируются документы:

  • Налоговые накладные и Приложения 2 к ним, полученные от поставщиков;
  • Товарные чеки, Проездные документы и др. приобретения, по которым может быть начислен налоговый кредит без подтверждения налоговой накладной;
  • Документы, подтверждающие приобрения у отечественных поставщиков-неплательщиков НДС;
  • Налоговые накладные, выписанные по факту приобретения услуг у нерезидентов.

Регистрация таможенных деклараций, подтверждающих приобретение товаров у нерезидентов, осуществляется документом Таможенная декларация на импорт.

Для автоматизированной регистрации поступивших налоговых документов предназначена закладка К оформлению. На закладке видна информация по всем операциям поставки, которые еще не отражены в Реестре выданных и полученных налоговых накладных. Выбрав операцию, по которой нужно зарегистрировать налоговый документ, нужно нажать кнопку Создать налоговый документ.

Для каких операций приобретения определяется необходимость зарегистрировать налоговые документы?

В соответствии с Порядком ведения реестра выданных и полученных налоговых накладных, регистрации в Реестре подлежат все операции приобретения, в том числе и те, которые не дают права на налоговый кредит.

По операциям приобретения у резидентов-неплательщиков НДС регистрации подлежат сами документы приобретения.

Какая информация используется при определении операций, по которым нужно зарегистрировать налоговые документы?

Используется информация о произведенных оплатах и документах приобретения.

Для определения даты поставки, которая должна быть указана в налоговом документе, учитывается Момент определения базы НДС, который указывается для организации или договора с поставщиком.

Нужно ли выполнять регламентные задания перед регистрацией входящего налогового документа?

Чтобы корректно определить список операций приобретения, по которым  необходимо зарегистрировать входящие налоговые документы, нужно восстановить последовательность расчетов с поставщиками. Однако, вызывать для этого регламентные задания необязательно. Достаточно нажать кнопку Актуализировать расчеты на закладке К оформлению.
