Управленческий баланс

Отчет предназначен для анализа финансового состояния предприятия с контролем баланса активов и пассивов.

Варианты формирования:

Управленческий баланс предприятия

Контроль управленческого баланса

Варианты формирования:

Управленческий баланс предприятия

С помощью параметра Тип анализа можно провести агрегированный анализ по организациям, подразделениям или сводно, в валюте отчета.

Особенности отчета:

  • анализ финансовых показателей по Статьям активов и пассивов;
  • анализ финансовых показателей по Организациям и Подразделениям предприятия;
  • динамика изменений показателей по периодам - месяц, квартал, полугодие, год;
  • анализ фактических изменений (Активов/Пассивов) за отчетный период.

Контроль управленческого баланса

Вариант предназначен для контроля ввода остатков, нарушений баланса в различных периодах отдельно взятыми документами.

Особенности отчета:

  • анализ финансовых показателей по статям активов и пассивов;
  • анализ сформированных финансовых показателей по видам документов.


Особенности отчета:

  • расшифровка показателя Товары в отчет Себестоимость товаров.


Отчет входит в группу отчетов Финансовый результат:

  • Валовая прибыль предприятия,
  • Валовая прибыль по полностью оплаченным отгрузкам,
  • Валовая прибыль по частично оплаченным отгрузкам,
  • Доходы и расходы,
  • Финансовые результаты.
