Призначений для перегляду і введення значень додаткових відомостей. Введення значень додаткових відомостей, як правило, здійснюється в окремому вікні, на відміну від додаткових реквізитів.

Відкривається за допомогою кнопки Додаткові відомості з вікон об'єктів програми, для яких були настроєні додаткові відомості.

У списку виводиться:

  • Найменування - найменування поля додаткової відомості;
  • Поле Значення призначене для перегляду або введення значень додаткової відомості.

Введення значень

  • Для переходу в режим введення або редагування значення додаткової відомості можна двічі натиснути мишею по полях.
  • Спосіб введення значення залежить від типу додаткового відомості (детальніше про типи додаткових реквізитів і відомостей див. тут).
  • Додаткові відомості можуть мати вибір лише з двох значень, наприклад, Да чи Ні. У цьому випадку введення нових значень не передбачено.
  • Якщо для введення передбачений список значень, натисніть Показати всі (клавішу F4). Потім виберіть потрібне значення з запропонованого списку. Залежно від настройок список значень може бути з ієрархією або без ієрархії. При необхідності в ці списки можна додавати значення.
  • Вибір значень (в залежності від настройок) може здійснюватися також з довідників програми.
  • Залежно від настройок додаткових відомостей може знадобитися Вибрати тип даних.

Зміна складу додаткових відомостей

  • При наявності відповідних прав можна настроїти нову додаткову відомість або змінити деякі настройки вже існуючих додаткових відомостей. Для цього виконайте команду Ще - Змінити склад додаткових відомостей.

Вилучені додаткові відомості

  • Додаткова відомость, яку було відмічено на вилучення, зображується зі значком . Для того щоб остаточно вилучити додаткову відомість, можливо, доведеться Очистити значення. Двічі натисніть мишею за значенням, для того щоб перейти в режим редагування.
