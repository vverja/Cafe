Багато документів, довідники та інші об'єкти програми мають свої друковані форми. Друковані форми можуть формуватися у вигляді табличних документів або офісних документів у форматах Microsoft Word, OpenOffice.org Writer, Adobe PDF, HTML, Microsoft Excel, а також в форматі табличного документа платформи (.mxl). При цьому не потрібно, щоб самі ці програми були встановлені на комп'ютері.

У той же час, для формування друкованих форм, які відразу виводяться у форматах Microsoft Word і OpenOffice.org Writer, необхідно, щоб на комп'ютері були встановлені відповідні офісні програми.

Особливості формування друкованих форм у веб-клієнті

При роботі в Інтернеті для формування друкованих форм у веб-клієнта необхідні додаткові настройки браузера. Необхідно скасувати блокування спливаючих вікон для веб-сайту програми.

  • У браузері Internet Explorer виберіть команду Сервіс - блокування спливаючих вікон - Параметри блокування спливаючих вікон.
  • Введіть адресу веб-сайту програми в полі Адреса веб-сайту, що одержує дозвіл, натисніть кнопку Додати, назва сайту з'явиться у полі Веб-вузли, які мають дозвіл. Далі натисніть Закрити. Для отримання більш детальної інформації див. документацію до веб-браузеру.

Також при роботі в Internet Explorer, якщо включена фільтрація активного вмісту, то потрібно натиснути на значок Стан фільтрації активного вмісту і відключити фільтрацію Axtivex тільки для цього сайту.

Друк у форматі PDF

  • При друку у веб-клієнті друковані форми виводяться у форматі PDF (по умовчанню), який можна відразу відправити на принтер або зберегти у файл на комп'ютер. Тому необхідно, щоб на комп'ютері була встановлена програма перегляду документів PDF, наприклад, Adobe Reader.
  • Якщо не встановлена програма перегляду документів PDF або не проведені настройки, зазначені вище, при спробі друку у веб-клієнті в форматі PDF буде виведено інформаційне повідомлення. Якщо на Комп'ютері є програма Adobe Reader, але не виконані налаштування веб-браузера, в нижньому вікні Відкрити або зберегти виберіть Відкрити, потім натисніть Друк або ОК. Після роздрукування закрийте вікно друку, натисніть на кнопку Закрити для повернення в програму.

Друк у форматі HTML

  • Якщо програма Adobe Reader на комп'ютері не встановлена, і немає можливості її встановити, або цей спосіб друку не влаштовує з яких-небудь причин, можна вибрати спосіб друку HTML.
  • Для цього натисніть кнопку Головне меню, виберіть команду Сервіс - Параметри, увімкніть Спосіб друку у вигляді HTML, натисніть OK.

Друковані форми у вигляді документів Microsoft Word або OpenOffice.org Writer

  • Для друку необхідно, щоб на комп'ютері були встановлені офісні програми Microsoft Word (не нижче 2003) та OpenOffice.org Writer (версія не нижче 3.2). Крім того, при роботі через Інтернет необхідний веб-браузер Internet Explorer версії 6.0 і вище, для якого слід попередньо виконати додаткову настройку.
  • Для настройки Internet Explorer виберіть команду Сервіс - Властивості оглядача. На вкладці Безпека натисніть кнопку Інший і налаштуйте рівень безпеки для роботи з веб-клієнта:
    • відключіть блокування спливаючих вікон;
    • дозвольте сценарії;
    • дозвольте використовувати елементи Axtivex.
    • На вкладці Конфіденційність виберіть рівень безпеки не вище середнього.
  • Детальніше про настройкм браузера див. Посібник користувача, глава 13 "Робота з веб-клієнтом" і Керівництво адміністратора, глава 8 "Настройка веб-браузерів для роботи у веб-клієнті".
  • При першій спробі друку без установки розширення роботи з файлами виводиться пропозиція встановити його. Натисніть Встановити, потім Продовжити для початку встановлення.
    • Для початку установки розширення роботи з файлами натисніть Продовжити.
    • В залежності від настройок системи безпеки комп'ютера може знадобитися підтвердити установку програми на комп'ютер. Перевірте видавця програми та натисніть Встановити для підтвердження.
    • Після успішного завершення інсталяції, натисніть ОК. Наступного разу це повідомлення виводитися вже не буде.
  • При першій спробі друку документа програма пропонує створити робочий каталог для зберігання тимчасових файлів друку. Виберіть папку, натисніть ОК. Це повідомлення програма також виводить одноразово. Також для зміни робочого каталогу друку можна скористатися командою в персональних настройках програми (вкладка Друк). На вкладці Друк натисніть Настройка робочого каталогу друку, виберіть папку на комп'ютері.
  • Потім програма пропонує отримати файл макета, необхідний для формування друкованої форми. Натисніть ОК для продовження. Через деякий час сформована друкована форма буде відкрита в програмі Microsoft Word.
  • Якщо які-небудь настройки були пропущені або зроблені неправильно, після виконання команди Друк виводиться повідомлення про помилку. У цьому випадку необхідно ще раз перевірити настройки.

Редагування макетів друкованих форм у веб-клієнті

  • Для зміни і перегляду макетів у виді табличних документів, якщо на комп'ютері не інстальовано повноцінна платформи, програмою пропонується встановити додаткову програму програму Робота з файлами. Її можна завантажити, якщо перейти за посиланням Перейти на сторінку програми "Робота з файлами" для встановлення, або з сайту: https://1c-dn.com/developer_tools/fileworkshop/.
  • Для перегляду макета у веб-клієнті підтвердіть завантаження файлу на комп'ютер за допомогою кнопки Продовжити. При відкритті макета друкованої форми у вигляді табличного документа браузером виводиться пропозиція відкрити або зберегти файл макета на комп'ютер. Виберіть папку для збереження макета. Потім можна відкрити програму Робота з файлами для зміни макета.
  • Збережіть зміни, закрийте файл макета, потім поверніться в програму, натисніть кнопку Завершити зміну. Для розміщення зміненого макета в програму вкажіть шлях до файлу макета з допомогою кнопки Огляд, натисніть ОК. Аналогічно проходить робота з макетами друкованих форм у форматах Microsoft Word і OpenOffice.org Writer.
