Эквайринг

Эквайринговые операции

Подсистема обеспечивает возможность учета поступлений и возвратов денежных средств, поступивших с помощью банковских пластиковых карт различных платежных систем.

Работа с подсистемой Эквайринг состоит из следующих этапов.

  • Настройка эквайринга. Задачами настройки и обеспечением работоспособности подсистемы эквайринга занимается администратор системы.
  • Регистрация оплаты и возврата оплаты по платежным картам (по операциям эквайринга). Операции выполняет кассир.
  • Взаимодействия с банком по операциям эквайринга. Операции выполняет казначей.

Работа с подсистемой эквайринга возможна только в том случае, если в разделе НСИ и администрирование - Казначейство - Управление параметрами казначейства установлен флаг Оплаты платежными картами.

Настройка эквайринга

Перед оформлением эквайринговых платежей необходимо зарегистрировать в программе договора эквайринга, в которых будут определены условия обслуживания банком операций с платежными картами и указать список тех эквайринговых терминалов, с помощью которых будут оформляться эквайринговые операции.

Регистрация оплаты и возврата оплаты по эквайрингу

В программе предусмотрена возможность отражения операций оплаты клиентом за товары, работы, услуги с помощью платежных (пластиковых) карт международных платежных систем, и отражение операций возврата денежных средств на платежную карту.

Регистрация отчетов банков по эквайрингу

В программе предусмотрена возможность регистрации детальных отчетов по операциям эквайринга, полученных из обслуживающих банков. Отчеты представляют собой расшифровку платежей, поступивших на расчетный счет или списанных с расчетного счета, при оформлении платежей с использованием платежных карт.

Контроль расчетов с банком по эквайрингу

Программа обеспечивает контроль расчетов с банком по операциям эквайринга. Эта возможность предназначена для сверки зарегистрированных оплат и возвратов денежных средств по платежным картам и суммы денежных средств, зарегистрированных банком по операциям эквайринга.
Сверка оформленных в программе платежей по платежным картам с информацией, зарегистрированной в банке осуществляется при помощи отчета Отчет банков по эквайрингу
