Оформление ручных скидок


После подбора товаров в документ кассир может указать для товара ручную скидку. Для этого предназначена кнопка Ручные скидки.

Ручная скидка в чеке может быть введена по всем позициям в чеке или по по конкретному товару. Ручная скидка может быть введена как процентная, так и суммовая.

Доступность ручных скидок определяетсмя функциональной опцией Ручные скидки в продажах в разделе НСИ и администрирование - CRM и маркетинг - Маркетинг.

Ручная скидка по всем позициям документа вводится в отдельном диалоговом окне, которое открывается при нажатии на кнопку Назначить ручную скидку. Для отмены ручных скидок используется команда Отменить ручные скидки.

Оформление автоматических скидок


Скидки, предусмотренные для розничного склада рассчитываются автоматически. Для просмотра информации о том, согласно каким условиям была зарегистрирована та или иная скидка, нажмите на гиперссылку с суммой рассчитанной скидки в информационной панели или выберите соответствующую команду в контекстном меню табличной части Товары. Будет показан список тех скидок которые применились для конкретного товара.

Расчет автоматических скидок необходимо проводить обязательно. Если скидки не рассчитаны, то чек не будет проведен.

Пробитие чека


Пробитие чека следует производить после оформления всех операций по оплате чека. Для пробития чека на фискальном регистраторе нажмите на кнопку Пробить чек.

Печать товарного чека


Для печати товарного чека выполните следующие действия:

  • Выберите команду Товарный чек в меню, которое вызывается по кнопке Прочие операции.
  • Появится форма печати товарного чека.
  • Укажите количество копий печати товарного чека.
  • Нажмите на кнопку Печать.
