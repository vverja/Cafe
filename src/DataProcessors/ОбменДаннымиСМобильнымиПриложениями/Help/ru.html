Обмен данными с мобильными приложениями

Рабочее место предназначено для интерактивного выполнения обмена данными с мобильными приложениями в случае, когда обмен данными инициируется не со стороны клиента, а со стороны стационарной ИБ, т.е. из стационарной ИБ дается команда на выполнение обмена данными с мобильным приложением.

Для возможности выполнения обмена данными из формы в текущем сеансе пользователя должна быть запущена компонента обмена данными с мобильными приложениями. При этом для работы компоненты используются ранее заданные настройки компоненты. Для того, чтобы просмотреть эти настройки, или задать новые, можно воспользоваться командой Настройки компоненты.

Для выполнения обмена необходиммо выбрать в поле Настройки обмена данными настройки, соответствующие нужному пользователю и мобильному устройству.

Для выполнения обмена данными необходимо указать параметры обмена:

  • Способ связи - выбор из списка предпочтительного способа связи.
  • Клиент - Сетевое подключение (если используется мобильное устройство, или другой ПК) или Этот компьютер (если клиент запущен на данном компьютере).
  • Адрес подключения и Порт - сетевой адрес подключенного устройства, либо локальный адрес (127.0.0.1) для текущего компьютера, а также порт TCP|IP, через который выполняется обмен данными.
  • Флаг Передавать данные клиенту - требуется установить, если необходимо передать данные из стационарной ИБ клиенту. Если требуется только получить данные клиенту, не передавая ему данных, установка флага не требуется.
  • Флаг Выполнить начальную инициализацию ИБ - необходимо установить, если требуется очистить данные, имеющиеся в мобильной ИБ и передать клиенту все данные, удовлетворяющие настройкам обмена.
