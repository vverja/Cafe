Настройка доступності

У формі відображається список користувачів, згрупований по підрозділах.

Для того, щоб варіант аналізу став доступний користувачеві слід зазначити відповідного користувача. Для надання доступу всім співробітникам підрозділу слід зазначити відповідний підрозділ.

Для швидкого зняття або встановлення позначки можна використовувати відповідні кнопки командної панелі - Відзначити все і Зняти позначки.

Для швидкого відбору користувачів, яким надано доступ до варіанту аналізу слід використовувати кнопку Тільки вибрані користувачі. У списку будуть відображені відмічені користувачі.
