Призначена для перегляду і зміни параметрів вхідного електронного листа або вихідного електронного листа.

Відкривається за відповідною командою меню Ще з електронних листів.

Параметри листа:

  • Створено- для вихідного листа означає дату закінчення редагування листа перед відправленням, для вхідного означає дату відправлення листа.
  • Внутрішній номер- номер, під яким лист зареєстровано в програмі.
  • Відправлено- дата відправлення листа, відображається тільки для вихідного листа.
  • Отримано- дата отримання листа, відображається тільки для вхідного листа.
  • Включати тіло вихідного листа- можна змінити для вихідного листа в статусі Чернетка або Вихідне. Якщо даний прапорець включений, то до тіла листа при відправленні буде прикладений текст листа-підстави.
  • Папка- поточна папка листа, доступна для зміни. Натисніть Показати все, виберіть потрібну папку зі списку.
  • Повідомити про доставку- ознака того, що відправник запитує повідомлення про доставку. Доступний для зміни, якщо статус вихідного листа Чернетка або Вихідне.
  • Повідомити про прочитання- ознака того, що відправник запитує повідомлення про прочитання. Доступний для зміни, якщо статус вихідного листа Чернетка або Вихідне.
  • Заголовки Інтернету- службова інформація про вхідний лист.
  • Ідентифікатори- службова інформація про вихідний лист.
