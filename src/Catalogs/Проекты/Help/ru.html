Проекты

Справочник предназначен для регистрации списка долгосрочных проектов.

Для проектов указываются тема, плановые и фактические сроки работы по проекту, ответственный и прочие привлеченных к проекту участники. Например, с помощью этого справочника можно вести проект по обучению сотрудников или проект работы с аутсорсинговой компанией (аудит, консалтинг). В рамках проекта можно вести список взаимодействий с участниками проекта, оформлять и контролировать задания сотрудникам на различных этапах проекта, хранить файлы документации по проекту.
