Список призначений для ведення валют, що використовуються в програмі. Необхідні для роботи валюти можна підібрати з Класифікатора валют (рекомендується) або додати вручну.

Відкрити список валют можна з розділу Довідники.

У списку виводяться:

  • Найменування валюти - повне найменування валюти.
  • Цифр. код - унікальний числовий код, прийнятий міжнародним стандартом, служить для ідентифікації валюти при завантаженні курсів з мережі інтернет.
  • Симв. код - прийнятий міжнародним стандартом, використовується як основне представлення валюти.
  • Курс - значення курсу на поточну дату.
  • Кратність - по умовчанню заповнене одиницею і не виводиться. Якщо потрібна вказівка курсу точністю більше чотирьох знаків дробової частини, то в цьому полі вказується кількість котирувальних одиниць.

Додавання валюти з класифікатора валют

  • Натисніть Підібрати з класифікатора (рекомендується для коректного введення валюти).
  • Наступні поля нової валюти будуть заповнені з класифікатора автоматично: Найменування, Символьний код, Цифровий код, положення перемикача Курс валюти (буде встановлено положення завантажується з інтернету, якщо є можливість завантажувати курс цієї валюти з сайту finance.ua).

Введення валюти не з класифікатора (вручну)

  • Натисніть Створити;
  • У відповідь на питання програми про підбір з класифікатора натисніть Створити.
  • Заповніть відомості про валюту.
