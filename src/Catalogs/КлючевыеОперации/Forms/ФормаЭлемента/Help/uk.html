Опис властивостей ключових операцій.

У програмі є напередвизначені ключові операції. За необхідності можна додати нові. В якості ключових операцій можуть виступати різні операції програми, наприклад, проведення документа, формування звіту і т. п. Для того щоб оцінювати продуктивність програми, необхідно на початку роботи програми заповнити параметри ключових операцій в профілі ключових операцій (після установки програми параметри ключових операцій не заповнені).

Відомості про ключові операції

  • Найменування - для напередвизначених операцій заповнене по умовчанню, для новостворених операцій необхідно ввести. Поле є обов'язковим;
  • Пріоритет - напишіть пріоритет кожної операції - ціле число, при цьому пріоритет не повинен повторюватися. Поле є обов'язковим;
  • Цільовий час - напишіть час в секундах. Поле є обов'язковим;
  • Мінімально допустимий рівень - виберіть зі списку, наприклад, Задовільно.

Вибрані параметри ключових операцій будуть застосовані при вимірах продуктивності.

Контроль вимірів часу

  • В ході оцінки продуктивності роботи програми проводяться виміри часу за кожною ключовою операцією. Для перегляду історії вимірів за кожною ключовою операцією натисніть Заміри часу на панелі навігації.
