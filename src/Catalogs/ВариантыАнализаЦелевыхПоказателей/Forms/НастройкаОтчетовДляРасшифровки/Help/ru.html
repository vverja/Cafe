Настройка отчетов для расшифровки

Для расшифровки варианта анализа следует выбирать следующие варианты отчетов:

  • в общем случае следует выбирать отчеты, в которых есть показатели идентичные целевому показателю;
  • при анализе динамики желательно выбирать варианты отчетов, в которых есть группировка по периоду и она соотвествует периодичности контроля;
  • при покомпонентном сравнении для расшифровки желательно выбирать варианты отчетов, в которых есть группировки соответствующие объекту анализа.


Для расшифровки доступны все варианты отчетов, зарегистрированные в системе - поставляемые и пользовательские.
