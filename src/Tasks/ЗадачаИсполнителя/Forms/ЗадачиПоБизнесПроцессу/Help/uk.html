Призначений для перегляду задач за бізнес-процесом.

Відкривається з бізнес-процесу (наприклад, Завдання) по команді Задачі панелі навігації.

Червоним кольором виділяються прострочені задачі. Виконані задачі стають неактивними, виділяються сірим кольором. Список задач нестартовавшего бізнес-процесу порожній.

У списку виводиться:

  • - важливість, приймає значення Звичайна (по умовчанню), Низька, Висока. Для того щоб виділити бізнес-процес зі списку, або навпаки, вказати, що виконання цього бізнес-процесу може бути відкладено, вибираються відповідні значення цього поля.
  • - зупинені бізнес-процеси в цій колонці позначаються значком. Для того щоб Зупинити або Продовжити бізнес-процес, скористайтеся відповідними кнопками.
  • Найменування- повне найменування задачі. Поруч проставляється значок Задача.
  • Виконавець задачі;
  • Строк виконання-дата і час, до якого Завдання має бути виконано.

Зміна режиму перегляду

  • Увімкніть прапорець Показувати виконані внизу списку, для того щоб переглянути всі задачі.

Перегляд задачі

  • Натисніть Змінити, для того щоб відкрити задачу для перегляду або редагування.
  • Також можна використовувати подвійне клацання лівою кнопкою миші.
