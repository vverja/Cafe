Выгрузка первичных документов в «FREDO ДокМен»

Обработка предназначена для пакетной выгрузки исходящих первичных документов (Счетов, Накладных, Актов) в систему "FREDO ДокМен"

1.     Выгружаются первичные документы, оформленные на контрагентов, в карточке которых установлен флаг "Использовать "FREDO ДокМен"

2.     Указываются общие параметры заполнения списка документов: тип документов, организация и период, за который будут выбраны документы из информационной базы продукта. Если требуется, устанавливается дополнительный отбор по контрагенту, ответственному за составление документа.

3.     Заполняется список документов при помощи кнопки «Заполнить».

4.     Отмечаются документы, требующие выгрузки. Для этого напротив документов устанавливаются соответствующие пометки.
