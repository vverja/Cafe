Призначена для копіювання настройок одного користувача іншим користувачам програми. При цьому можна скопіювати настройки в межах одного списку - Користувачі або Зовнішні користувачі.

Зручно використовувати в разі, якщо настроюється кілька нових користувачів в одній групі, в якій вже є користувачі.

Команда для копіювання настройок користувачів знаходиться в розділі Адміністрування - Настройки користувачів і прав.

Копіювання настройок користувачів

  • У полі Скопіювати настройки від користувача виберіть користувача (або зовнішнього користувача), чиї настройки будуть використовуватися для копіювання іншим користувачам (зовнішнім користувачам):
    • натисніть Вибрати, вкажіть список Користувачі (Зовнішні користувачі), в якому буде проводитися копіювання настройок.
    • у списку Вибір користувача (Вибір зовнішнього користувача) виділіть потрібного користувача (зовнішнього користувача), натисніть кнопку Вибрати. Також можна двічі натиснути по потрібному користувачеві лівою кнопкою миші.
  • В полі Кому копіювати виберіть користувачів, яким необхідно скопіювати настройки. Перемикач приймає значення:
    • Обраним користувачам - стає доступний вибір користувачів. Перейдіть за посиланням Вибрати.
      • у вікні "Вибір користувачів для копіювання настройок за допомогою прапорців виберіть потрібних користувачів (зовнішніх користувачів).
      • Після завершення вибору посилання приймає значення: ім'я одного вибраного користувача або кількість вибраних користувачів.
    • Всім користувачам - після вибору значення можна перейти до наступного кроку.
  • У полі Що копіювати виберіть, які настройки копіювати іншому користувачеві або користувачам:
    • Всі настройки - після цього можна перейти до копіювання настройок.
    • При виборі значення Окремі настройки з'являється можливість вибору потрібних настройок зі списку. Перейдіть за посиланням Вибрати.
      • у вікні, що відкрилось, Вибір настройок користувача відзначте прапорцями настройки для копіювання.
      • Після завершення вибору посилання приймає значення: найменування однієї настройки користувача або кількість вибраних настройок.
  • Для переходу до копіювання настройок з вибраними параметрами натисніть Скопіювати і закрити, для того щоб повернутися в програму або Скопіювати, для того щоб після завершення копіювання цих настройок продовжити копіювання.

Обмеження

  • Не всі користувацькі настройки можна скопіювати, навіть якщо був натиснутий перемикач Всі настройки .
  • Користувачі, яким копіюються настройки зовнішнього вигляду програми, повинні завершити свої сеанси роботи. Інакше настройки можуть не застосовуватися.
  • При копіюванні настройок варіантів звітів користувачі повинні мати права на роботу з цими звітами.
  • Особисті варіанти звітів скопіювати іншим користувачам неможливо.

В цих випадках програма видає попередження, що не всі настройки були скопійовані. Натисніть кнопку Показати звіт, для того, щоб подивитися, які настройки були пропущені при копіюванні.
