Мобільні комп'ютери

Довідник призначений для зберігання списку мобільних комп'ютерів, які використовуються при роботі торгових представників.

Доступність довідника визначається функціональною опцією Ведеться робота через торгових представників в розділі НДІ і адміністрування - Продажі - Торгові представники.

При видачі торговому представнику мобільного комп'ютера в список вводиться новий запис про мобільний комп'ютер із зазначенням найменування комп'ютера і його серійного номера.

Для роботи з мобільними комп'ютерами використовується продукт "Підприємство 8. Розширення для кишенькових комп'ютерів".

Дані з ліцензування (реєстраційний номер, реєстраційний код, код активації) можуть бути збережені в картці мобільного комп'ютера для їх подальшого використання.

Дані про мобільний комп'ютер використовуються при настройці правил обміну даними між комп'ютером торгового представника з керуючою системою.
