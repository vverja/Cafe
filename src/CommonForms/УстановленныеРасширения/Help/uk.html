Список призначений для додавання до програми розширень конфігурації. Вони дозволяють додавати нові можливості в програму без її прямої зміни.

Відкривається по команді Розширення в розділі Адміністрування - Друковані форми, звіти та обробки.

У списку виводяться:

  • Найменування - проставляється по умовчанню при підключенні розширення, заповнюється розробником;
  • Версія- проставляється програмою по умовчанню при підключенні розширення, заповнюється розробником;
  • Безпечний режим- у доданого розширення прапорець вмикається автоматично, адміністратор може вимкнути прапорець або вказати ім'я профілю безпеки.

Додавання розширення

  • Натисніть Додати, вкажіть шлях до файлу з розширенням на комп'ютері.
  • Додавання розширення займає деякий час.
  • Після додавання розширення, для того щоб продовжити роботу з урахуванням зроблених змін, натисніть кнопку Перезапустити. В моделі сервісу цієї кнопки немає, бо для неподілених користувачів, наприклад, для адміністраторів сервісу, при вході в область даних розширення області даних не підключаються, а залишається підключеним склад розширень, який був до входу в область даних, тобто при запуску сеансу.
  • Після перезапуску розширення буде підключено до програми, і його функції можна буде використовувати.

Вилучення розширення

  • Якщо розширення більше не потрібно в програмі, виділіть його, натисніть Вилучити.
  • Вилучення розширення не вимикає його. Всі сеанси, запущені після додавання розширення і до його вилучення, продовжать роботу з вилученим розширенням.
  • У нових сеансах віддалене розширення більше не підключиться.

Перезапуск програми

  • Для того щоб зміни, внесені в список розширень, вступили в силу (тобто були враховані при роботі програми), натисніть посилання Ще- Перезапустити програму.
  • Якщо при запуску розширення не вдалося підключити до програми, тоді виводиться повідомлення про помилку.

Збереження розширення в файл

  • Виділіть розширення, скористайтеся командою Зберегти як контекстного меню по правій кнопці миші, вкажіть шлях до файлу на комп'ютері.

Оновлення розширення

  • Виділіть розширення, скористайтеся командою Оновити з файлу на диску контекстного меню по правій кнопці миші, вкажіть шлях до файлу на комп'ютері.
  • Слід використовувати цю кнопку для встановлення нової версії розширення.
  • Також потрібно використовувати цю можливість для повернення до попередньої версії, якщо нова версія містить помилку (однак не рекомендується встановлювати розширення меншої версії, тому що це може привести до втрати зроблених налаштувань).
