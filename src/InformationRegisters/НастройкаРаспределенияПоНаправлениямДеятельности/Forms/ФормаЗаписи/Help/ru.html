Настройка распределения продаж по направлениям деятельности

Настройка распределения продаж по направлениям деятельности определяет на какие направления деятельности будут отнесены суммы выручки и себестоимости продаж.
Распределяются выручка и себестоимость, подходящие под указанные фильтры (пустое значение фильтра означает любое значение). Распределение может происходить на конкретное направление деятельности или на несколько направлений, по заданному способу распределения.

Одновременно могут использоваться несколько настроек распределения. У активных настроек нужно установить флажок Используется. При совместном действии разных настроек – приоритет отдается тем, где задано конкретное значение «клиента», потом правилам с заданным значением «подразделения», потом с заданной «номенклатурой» и в конце – с заданной «организацией».

Пример. Есть два правила с разной аналитикой. Правило 1: Организация = «Организация 1»; Номенклатура = «Товар 1», Подразделение = « »; Клиент = «Клиент 1». Правило 2: Организация = «Организация 1»; Номенклатура = «Товар 1», Подразделение = «Подразделение 2»; Клиент = « ». При распределении дохода, подходящего одновременно под оба правила («Организация 1», «Товар 1», «Подразделение 2», «Клиет 1») – будет применено первое, т. к. в нем указан Клиент.

Продажи между организациями в финансовом результате по направлениям деятельности учитываются отдельно для каждой организации, участвующей во внутренних передачах товаров. Чтобы исключить их из финансового результата от продажи товаров клиентам – следует отнести продажи для клиента «Наше предприятие» на отдельное направление деятельности.

Как определить правила, по которым будет происходить распределение выручки и себестоимости продаж по направлениям деятельности?

Как определить правила, по которым будет происходить распределение выручки и себестоимости продаж по направлениям деятельности?

В карточке " Настройка распределения продаж по направлениям деятельности" введите следующую информацию.

  • В поле "Период" укажите дату начала того месяца, в котором будет использоваться данная настройка распределения.
  • Установите флаг в поле "Используется".
  • Укажите организацию, продажи по которой должны относиться на одно или несколько направлений деятельности.
  • Выберите способ распределения.
  • Если продажи должны относиться на одно направление деятельности, то установите флаг "Распределяется на конкретное направление деятельности" и выберите из списка нужное направление деятельности.
  • Если продажи должны распределяться по нескольким направлениям деятельности, то установите флаг в поле "Распределяется по способу распределения" и выберите способ распределения по направлениям деятельности из справочника.

Аналогичным образом можно задать распределение продаж по направлениям деятельности по подразделениям, конкретным позициям номенклатуры или конкретному клиенту.

Распределение продаж по направлениям деятельности производится при выполнении регламентного документа "Расчет себестоимости товаров". При распределении по направлениям деятельности используется следующий алгоритм распределения.

  • Анализируются все продажи за текущий месяц.
  • Анализируются все заданные настройки распределения продаж.
  • Выделяется список продаж, по которым можно применить распределение с полной детализацией: по конкретной организации, конкретному подразделению, конкретной позиции номенклатуры, конкретному клиенту.
  • Производится распределение этих продаж по указанным в настройках направлениям деятельности.
  • Далее анализируются продажи и настройки с неполной детализацией и происходит их дальнейшее распределение.

При распределении продаж по направлениям деятельности используются следующие приоритеты.

  • Организация – самый низкий приоритет;
  • Номенклатура.
  • Подразделение.
  • Клиент - самый высокий приоритет.

Любое измерение настройки может быть не указано. В этом случае настройка применяется ко всем возможным значениям измерения.

Пример. В торговом предприятии имеется центральный офис и несколько магазинов. Магазины переведены на упрощенную систему налогообложения и работают от имени одной организации "Розница". В центральном офисе осуществляются оптовые продажи, причем выделены раздельные подразделения для работы с оптовыми покупателями и дистрибьюторами. Необходимо распределять продажи по следующим направлениям "Розничные продажи", "Оптовые продажи", "Дистрибьюция".
  • В список настроек распределения продаж по направлениям деятельности добавляем следующие настройки.
  • В первой настройке указываем организацию "Розница" и направление "Розничные продажи".
  • Во второй настройке указываем организацию "Центральный офис", подразделение "Работа с оптовыми покупателями", направление деятельности "Оптовые продажи".
  • В третьей настройке указываем организацию "Центральный офис", подразделение "Работа с дистрибьюторами", направление деятельности "Дистрибьюция".
