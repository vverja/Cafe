Довідник "Вихідні документи" служить для обліку офіційної кореспонденції, яку Ваша організація відправляє зовнішнім адресатам.

До кожного вихідного документа можна приєднати один або кілька файлів. Облік файлів можливий, якщо настройка програми "Використовувати файли у вихідних документів" встановлена.

Для документа слід задати:

  • одержувача (кореспондента, для якого призначений документ).
  • адресат - контактна особа;
  • керівника (користувача), який підписав документ;
  • користувача, що підготував документ;
  • підрозділ, який випускає документ;
  • строк відповіді (якщо потрібно) - показується, якщо настройка програми "Використовувати строк відповіді" встановлена;
  • прапорець "Відправлений" встановлюється, якщо документ вже відправлений, також встановлюється дата відправки;
  • спосіб відправки документа;
  • гриф доступу - якщо встановлена настройка програми "Використовувати грифи доступу", то вхідному документу потрібно обов'язково присвоїти гриф доступу, вибравши його з числа заздалегідь настроєних адміністратором грифів. Гриф визначає права доступу користувачів до вхідного документа;
  • вид документа - показується, якщо настройка програми "Використовувати види вихідних документів" встановлена;
  • стан документа - показується, якщо настройка програми "Використовувати стан документів" встановлена;
  • організацію - у разі ведення документообігу для декількох організацій (показується, якщо настройка програми "Використовувати облік по організаціях" встановлена);
  • у полі "Короткий зміст "(нижче поля" Найменування") записується текст, що пояснює ціль відправки документа;


Документ може нумеруватися вручну або автоматично (в залежності від настройок виду документа).
Якщо задана автоматична нумерація, то для того, щоб присвоїти реєстраційний номер необхідно натиснути кнопку "Зареєструвати".
Якщо задано ручну нумерацію, то достатньо заповнити поля "Реєстраційний номер" та "Дата реєстрації" та записати документ.

Якщо настройка програми "Використовувати електронно цифровий підпис" включена, то в картці документа додаються група команд "ЕЦП" (містить команду "Підписати", за допомогою якої виконується посвідчення документа електронним підписом) і закладка "ЕЦП", в якій редагуються і перевіряються електронно-цифрові підписи.
Якщо документ підписаний, то його редагування неможливо. Для редагування потрібно спочатку вилучити всі підписи. Поточний користувач може вилучити тільки свої підписи. Право вилучення всіх підписів належить адміністратору.
Підпис можна зберегти у файл (наприклад, для пересилання електронною поштою), а також переглянути сертифікат підпису.
