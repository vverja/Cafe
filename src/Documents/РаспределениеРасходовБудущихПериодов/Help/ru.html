Особенности работы с документом


Расходы регистрируются по статье расходов с вариантом распределения на расходы будущих периодов. Примером таких расходов является арендная плата, которая может быть оплачена вперед и должна отразиться в расходах тех месяцев, в которые арендованное помещение будет использоваться.

Документ производит распределение по одной статье расходов будущих периодов. Для распределения необходимо указать дату начала распределения расходов, а также дату окончания либо число месяцев для распределения (вариантивность устанавливается с помощью соответствующего тублера). В табличной части документа для каждого из месяцев распределения следует указать сумму расхода, которая будет распределена на этот месяц, подразделение-получатель расходы, а также статью и аналитику расходов, по которой эта сумма отразится в расходах данного месяца.

Статья расходов по умолчанию для месяцев, на которые распределяется расход, указывается в статье расходов будущих периодов. В документе эта статья расходов может быть изменена произвольным образом.

Контроль за распределением расходов


Данные о распределении расходов будущих периодов можно проанализировать в отчете Контроль распределения доходов и расходов. В отчете показываются суммы, не распределенные в прошлых периодах, распределенные и не распределенные в текущем периоде.
