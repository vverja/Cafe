Довідник "Класифікатор кодів для ПН"

У цьому довіднику зберігається список кодів товарів і послуг для використання в податкових документах: коди УКТЗЕД для товарів і коди ДКПП для послуг, а також коди видів діяльності для сільгосппідприємств.

Підбір кодів з класифікаторів УКТЗЕД і ДКПП здійснюється при натисканні відповідних кнопок. При виборі (ручному введенні) коду УКТЗЕД вибирається Вид: Товар або Імпортований товар. Для імпортованого товару в графі 3.2 Податкової накладної проставляється відмітка Х. Якщо підприємство здійснює постачання імпортних і вітчизняних товарів з одним і тим самим кодом УКТЗЕД, код повинен бути внесений до Класифікатора двічі: як Товар і Імпортований товар. При виборі (ручному введенні) коду ДКПП вказується Вид Послуга.

Сільгосппідприємства, які виписують Податкові накладні по спецрежиму 2, крім кодів УКТЗЕД сільськогосподарської продукції, вказують також Код сільгоспдіяльності. Код сільгоспдіяльності вказується у колонці 11 Податкової накладної по спецрежиму 2.
