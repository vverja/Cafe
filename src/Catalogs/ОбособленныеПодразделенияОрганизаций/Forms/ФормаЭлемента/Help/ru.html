Какая информация указывается для всех обособленных подразделений?

Для каждого обособленного подразделения обязательно указывается Организация, к которой оно относится, и Наименование - рабочее наименование подразделения, которое будет указываться в документах информационной базы.

Какая информация указывается для обособленных подразделений, по которым ведется отдельная кассовая книга?

Указывается Полное наименование обособленного подразделения. Это наименование выводится в печатных формах кассовых ордеров и кассовой книги как найменування підприємства (установи, організації). Полное ниаменование обособленного подразделения долно включать наименование организации.

Какая информация указывается для обособленных подразделений, по которым выписываются налоговые накладные?

Префикс. Указывается числовой номер, присвоенный подразделению. Этот номер будет указываться в составе номера налоговых документов, выписанных подразделением, через дробь.

Дополнение к названию организации. Указывается наименование подразделение, которое будет указываться в налоговых документах в составе наименования продавца (после наименования организации).
