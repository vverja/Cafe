Налаштовуються для механізмів, які працюють з веб-сервісами.

За необхідності вкажіть адреси проксі-серверів для окремих протоколів та адреси інтернет-серверів, до яких повинен використовуватися прямий доступ.

За умовчанням увімкнуто прапорець Використовувати проксі-сервер за умовчанням для всіх протоколів.

Основні параметри настройки (Адреса сервера і Порт) за замовчуванням будуть також використані в якості додаткових параметрів.

Настройка додаткових параметрів

  • Для того щоб вказати адреси проксі-серверів для окремих протоколів, вимкніть прапорець Використовувати проксі-сервер по умовчанню для всіх протоколів;
  • У відповідному полі - HTTP, HTTPS, FTP - відкоригуйте ім'я проксі-сервера, який буде використовуватися для кожного протоколу;
  • За необхідності вкажіть Порт.
  • У списку Не використовувати проксі-сервер для адрес за допомогою кнопки Додати вкажіть адреси, до яких буде здійснюватися прямий доступ.
  • Для підтвердження введених параметрів натисніть ОК.
