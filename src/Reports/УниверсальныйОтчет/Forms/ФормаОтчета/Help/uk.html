Для перегляду звіту необхідно натиснути кнопку Сформувати, попередньо задавши параметри (якщо вони передбачені). У клієнт-серверному варіанті роботи програми передбачено формування звітів у фоновому режимі.

Режим формування

Установіть прапорець Формувати відразу меню Ще, для того щоб звіт формувався автоматично відразу після відкриття, без натискання кнопки Сформувати.

Варіанти звіту

  • У програмі може бути передбачено кілька варіантів звітів, що виконують схожий аналіз. Для їх перегляду передбачена кнопка Варіанти звіту.
    • Виберіть зі списку, що випадає, потрібний варіант звіту. Поточний варіант звіту відзначений прапорцем.
    • для перегляду списку інших варіантів звітів призначена команда Інші звіти.

Настройки

  • По кнопці Настройки можна перейти до настройки параметрів звіту.
  • Натисніть Ще- Інше- Стандартні настройки, для того щоб повернутися до настройок по умовчанню.
  • Натисніть Ще- Інше- Всі настройки, для того щоб перейти до настройок всіх передбачених параметрів звіту.
  • Натисніть Ще- Інше- Зберегти настройки, для того щоб записати поточні настройки звіту.
  • Натисніть Ще- Інше- Вибрати настройки, для того щоб скористатися раніше збереженими настройками звіту.

Інші звіти

У вікні розміщений список схожих за змістом звітів.

  • В списку виводяться найменування та описи звітів.
  • Список звітів згрупований. Передбачені групи, в яких виводяться схожі за змістом інші звіти того ж розділу програми.
  • Для переходу до іншого варіанту звіту натисніть на заголовок лівою кнопкою миші.
  • За допомогою відповідного прапорця можна Закривати вікно після переходу до звіту.

Друк звіту

  • Сформований звіт можна роздрукувати, натиснувши кнопку Друк.
  • Для того щоб перевірити звіт і змінити параметри сторінки, натисніть Попередній перегляд.
  • За допомогою команди Ще-Параметри сторінки можна настроїти друк звіту.

Надсилання звіту електронною поштою

За допомогою кнопки Надіслати звіт електронною поштою можна:

  • Відправити зараз, для цього:
    • за допомогою прапорців виберіть один або кілька форматів, в яких необхідно відправити звіт.
    • при необхідності увімкніть прапорець Додатково упакувати в архів (. zip).
    • Натисніть Продовжити. Звіт буде прикріплений до електронного листа в якості вкладення.
  • Створити розсилку звітів;
  • Включити розсилку звітів, що вже існує. Для цього зі списку розсилок виберіть потрібну.

Вибір варіанту звіту

  • Для завантаження раніше збереженого варіанту звіту скористайтеся командою Ще-Інше- Вибрати варіант звіту.
    • у списку варіантів звіту, натисніть кнопку Вибрати для переходу до потрібного варіанту звіту.

Зміна варіанту звіту

  • Натисніть Ще- Інше- Змінити варіант звіту. Можна додати поля, змінити параметри та настройки звіту.
  • Внесені зміни можна зберегти в інший варіант звіту.

Збереження варіанту звіту

  • Натисніть Варіанти звіту- Зберегти варіант звіту, вкажіть Найменування та інші параметри.
  • Також можна скористатися командою Зберегти варіант звіту меню Ще.

Скидання настройок

  • Натисніть Ще- Інше- Стандартні настройки, для того щоб скинути користувацькі настройки.

Інші доступні дії

  • Для пошуку в звіті скористайтеся командою Знайти.
  • За допомогою кнопок Згорнути всі групи або Розгорнути всі групи можна приховати або показати групування звіту.
  • Для збереження результату звіту в файл натисніть, вкажіть найменування файлу і шлях для збереження.
  • Натисніть, для того щоб порахувати суму чисел у виділених комірках.
