Настройка відкриття форм при початку роботи програми

Регістр призначений для того, щоб адміністратор може визначити для користувача те робоче місце , яке буде відкрито йому по умовчанню при початку роботи.

Приклад. Касиру ККМ має бути визначено робоче місце касира (форма чека ККМ) для оформлення роздрібних продажів, а менеджеру з продажу зручно працювати, використовуючи обробку Помічник продажів.

У списку заповнена інформація про всіх зареєстрованих користувачів програми і робочих місцях (спеціалізованих формах), які можуть бути їм призначені відповідно до встановлених для них правами. Для установки робочого місця по умовчанню необхідно виконати наступні дії.

  • Встановити курсор на потрібний запис у списку.
  • Виконати команду Змінити.
  • У формі, що відкрилася, встановити прапорець Відкривати по умовчанню.
  • Для деяких робочих місць необхідно встановити додаткові параметри. Наприклад, склад і дозвіл екрану для робочого місця працівника складу.

Можливий альтернативний спосіб для відкриття робочого місця користувачеві по умовчанню. Адміністратор може змінити у користувача властивості ярлика запуску продукту, вказавши потрібні параметри запуску відповідно до тих даних, які вказані в списку Настройки відкриття форм при початку роботи з програмою.
