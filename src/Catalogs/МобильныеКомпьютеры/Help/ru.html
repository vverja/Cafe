Мобильные компьютеры

Справочник предназначен для хранения списка мобильных компьютеров, которые используются при работе торговых представителей.

Доступность справочника определяется функциональной опцией Ведется работа через торговых представителей в разделе НСИ и администрирование - Продажи - Торговые представители.

При выдаче торговому представителю мобильного компьютера в список вводится новая запись о мобильном компьютере с указанием наименования компьютера и его серийного номера.

Для работы с мобильными компьютерами используется продукт "Предприятие 8 Расширение для карманных компьютеров".

Данные по лицензированию (регистрационный номер, регистрационный код, код активации) могут быть сохранены в карточке мобильного компьютера для их дальнейшего использования.

Данные о мобильном компьютере используются при настройке правил обмена данными между компьютером торгового представителя с управляющей системой.
