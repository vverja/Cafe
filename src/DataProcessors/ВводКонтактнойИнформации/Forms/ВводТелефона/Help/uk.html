Призначена для введення номера телефону (факсу) з використанням шаблону.

Відкривається з табличної частини документів або елементів списків - носіїв контактної інформації по кнопці Вибрати.

Для того щоб ввести телефон (факс), необхідно заповнити поля:

  • Код країни - програма автоматично додає "+" до коду країни;
  • Код міста - автоматично поміщається в дужки;
    • У шаблоні введення номера телефону для прискорення введення нових телефонних номерів в поля з кодом країни і міста автоматично підставляються значення, введені в попередній раз. Додатково для коду міста запам'ятовуються 10 раніше введених кодів, які можна швидко вибрати з випадаючого списку;
  • Номер телефону - пишеться в довільній формі;
  • Додатковий номер - перед цифрами додаткового номера програма автоматично проставляє доб;
  • Коментар - введіть додаткову інформацію про телефон.

Очищення номера телефону (факсу)

  • Для того щоб вилучити помилково введений номер телефону (факсу) і почати заповнення заново, скористайтеся командою Ще - Очистити номер телефону.
