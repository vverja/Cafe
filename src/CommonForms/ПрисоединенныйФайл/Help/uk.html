Призначена для перегляду відомостей про приєднантй файл.

Відкривається зі списку приєднаних файлів за допомогою команди Ще - Відкрити картку (клавіші F2).

Відомості про файл

  • Найменування - повне ім'я файлу. Автоматично заповнюється програмою під час приєднання файлу і створення картки файлу, можна змінити;
  • Опис - напишіть докладні відомості про файл;
  • Автор - користувач, який додав файл в базу;
  • Створений - дата додавання файлу в базу;
  • Відредагував - останній користувач, який відредагував файл;
  • Змінено - Дата зміни даних файлу (місцевий час);
  • Розміщення - найменування і дата документа, до якого приєднано файл. Можна переглянути документ;
  • Розмір - розмір даних файлу;
  • Розширення - розширення (формат) файлу.

Редагування приєднаного файлу

  • У тонкому клієнті редагування файлів табличних документів (формату .mxl) здійснюється у вбудованій формі редактор табличних документів. У веб-клієнті необхідно завантажити прикладну програму для редагування цих файлів. Детальніше про особливості редагування табличних документів у веб-клієнті див. тут.
  • Для редагування файлів інших форматів на комп'ютері повинні бути встановлені відповідні прикладні програми, наприклад, для редагування файлів у форматі Microsoft Word має бути встановлена ця програма.
  • За допомогою кнопки Переглянути можна відкрити файл зовнішнім додатком для перегляду або редагування, в залежності від того, позначений чи ні файл як зайнятий іншим користувачем для редагування.
  • Команда Ще - Редагувати відкриває файл в режимі редагування, а також виконує захоплення файлу в програмі. Захоплений файл буде недоступний для редагування іншим користувачам програми і позначається в списку зеленим кольором.
  • Команда Ще - Закінчити редагування звільняє файл, зберігає його в програмі і знімає позначку зайнятості для редагування. Після закінчення редагування збережіть файл, закрийте зовнішній додаток.
  • Команда Ще - Скасувати редагування звільняє файл, не зберігаючи вироблені в ньому зміни в програмі, і знімає позначку зайнятості для редагування. Ця команда може бути корисна, якщо необхідно відмовитися від результатів редагування файлу (наприклад, при редагуванні в файл були внесені небажані зміни або помилки).

Збереження файлу

  • За допомогою команди Ще - Зберегти як можна зберегти файл в зазначений каталог на жорсткому або мережевому диску.
  • Відповідною командою меню Ще можна Відкрити каталог файлу в Провіднику Windows;

Оновлення файлу

  • З допомогою команди Ще - Оновити з файлу на диску можна оновити файл з оригінального файлу на диску (з якого спочатку був створений файл у програмі).

Електронний підпис і шифрування

При використанні електронного підпису стає доступним випадаюче меню Ще - Електронний підпис і шифрування, яке містить команди для роботи з електронним підписом файлів:

  • Підписати - створює електронний підпис для файлу. Перед створенням електронного підпису програма пропонує вибрати сертифікат електронного підпису з наявних на комп'ютері. Вибір сертифіката можна зробити і під час персональних настройок програми. Підписаний файл стає недоступним для редагування і відображається у списках зі значком .
  • Додати електронний підпис з файлу - додає підпис до файлу. Команду можна використовувати, наприклад, якщо підпис (файл підпису) був отриманий пізніше, ніж підписаний файл.
  • Зберегти разом з електронним підписом - зберігає підпис у файл на диску. Команду можна використовувати, наприклад, якщо необхідно передати підписаний файл іншій особі.
  • Зашифрувати - по команді відбувається шифрування файлу. На відміну від підписання можуть використовуватися кілька сертифікатів, наприклад, якщо необхідно зашифрувати файл для декількох користувачів. Зашифрований файл у списках відображається зі значком .

  • Розшифрувати - по команді відбувається розшифровка файлу. Для розшифровки потрібно ввести пароль до закритого ключа.

При використанні електронного підпису та шифрування стають доступними закладки:

Електронні підписи

На закладці розміщуються відомості про електронний підпис файлу:

  • Автор підпису - користувач, який підписав файл;
  • Дата підпису - дата підписання файлу;
  • Коментар - напишіть додаткові відомості про підписи;
  • Статус, Дата перевірки - істинність підпису і дата її перевірки.

За допомогою відповідних кнопок можна:

  • Підписати файл;
  • Відкрити - відкрити вікно з основними властивостями сертифіката електронного підпису виділеного файлу: кому і ким виданий, призначення і термін дії сертифіката, на основі якого створено підпис;
  • Перевірити, чи змінювався будь-ким підписаний файл, за допомогою одного або всіх сертифікатів електронного підпису. Для цього виділіть один або кілька підписів. Якщо електронний підпис файлу, що зберігається, не відповідає якому-небудь сертифікату електронного підпису, то такий електронний підпис після перевірки буде виділений в списку червоним кольором, а в колонці Статус у такого електронного підпису буде зазначено Не вірний.
  • Перевірити все - перевірити всі підписи;
  • Зберегти електронний підпис в файл на диск;
  • Видалити у файлу електронний підпис.

Дозволено розшифровувати

  • На закладці розміщуються відомості про шифрування файлу:
  • У списку на цій закладці відображаються сертифікати, за допомогою яких зашифрований файл. Подвійним натисненням кнопки миші або за допомогою відповідної кнопки можна Відкрити сертифікат електронного підпису і переглянути його основні властивості: кому і ким виданий, призначення і термін дії сертифіката.
