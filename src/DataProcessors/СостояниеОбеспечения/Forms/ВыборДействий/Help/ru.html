Действия по обеспечению заказа

Позволяет выполнить действия по обеспечению заказа в целом, за счет выполнения однотипных рекомендаций для всех строк текущего заказа. Выбор действий производится установкой одного или нескольких флагов:

  • Исправить срывы в обеспечении заказа. Установка данного флага возможна, если имеются строки заказа с рекомендацией Исправить обеспечение. Нажатие Выполнить приведет к выполнению всех рекомендаций Исправить обеспечение для текущего заказа.
  • Снять преждевременное обеспечение. Установка данного флага возможна, если имеются строки заказа с рекомендацией Резервировать позже. Нажатие Выполнить приведет к выполнению всех рекомендаций Резервировать позже для текущего заказа.
  • Ускорить обеспечение и отгрузку заказа. Установка данного флага возможна, если имеются строки заказа с рекомендацией Резервировать или Резервировать раньше. Нажатие Выполнить приведет к выполнению всех рекомендаций Резервировать и Резервировать раньше для текущего заказа.
  • Отгружать частями. Установка/сброс данного флага возможны только для заказов в кторых предусмотрена возможность как отгрузки частями так и полной отгрузки одной датой, например для Заказов клиентов.
