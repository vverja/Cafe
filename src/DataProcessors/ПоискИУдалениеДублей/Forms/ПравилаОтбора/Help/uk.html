Призначена для відбору елементів списків.

Відкривається з вікна Пошук і вилучення дублів за посиланням в поле Відбирати.

Введення умов відбору

  • Натисніть Додати новий елемент.
  • Виберіть зі списку Поле, по якому виробляти відбір.
  • У колонці Вид порівняння Виберіть умову порівняння. Залежно від обраного списку можуть бути доступні як стандартні види порівняння (Дорівнює, Не дорівнює, Більше, Менше і т. д.), так і більш складні:
    • За допомогою виду порівняння В списку можна задати список можливих значень для відбору. Вид порівняння Не в списку є запереченням попередньої.
    • За допомогою операції Містить або Не містить можна використовувати відбір по частині поля (поєднання букв).
    • Визначити, є чи в полі значення, можна з допомогою операції Заповнено або Не заповнено.
  • Настройте Значення. Для цього в залежності від обраного поля і виду порівняння Виберіть із запропонованого списку значень потрібне, введіть значення або створіть свій список значень (для операцій У списку або Не в списку).
  • Для вилучення відбору натисніть Видалити.
  • Для підтвердження і повернення в програму, натисніть ОК.

Групування умов

Можна використовувати кілька умов для більш тонкого відбору. Їх можна переставляти місцями за допомогою миші або кнопок і .

  • Якщо у правилі задається кілька умов без групування, то по умовчанню до таких умов застосовується логічна операція І.
  • До виділених умов можна застосувати логічні операції І, Або, Не. Для цього в програмі застосовуються Типи груп: Група І, Група Або, Група Не.
    • Для того щоб використовувати групування умов, виділіть одне або кілька потрібних умов, натисніть Згрупувати умови. По умовчанню умови включаються в Групу І.
    • При необхідності можна змінити операцію групування. Для цього за допомогою подвійного натиснення кнопки миші по Групі відкрийте комірку вибору Типу групи. Виберіть інший Тип групи. Також можна виправити тип групи вручну.
    • За допомогою лівої кнопки миші перетягніть інші умови в групу або створіть їх усередині групи.
    • Для того щоб скасувати групУвання, натисніть Ще - Розгрупувати.
