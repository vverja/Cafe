Призначений для управління файлами, приєднаними до об'єкта програми.

Відкривається по команді Приєднані файли панелі навігації об'єкта програми. За допомогою відповідних команд в список можна додавати нові файли, копіювати, змінювати або вилучати їх.

Зеленим кольором відображаються файли, зайняті Вами для редагування, а сірим кольором - файли, зайняті іншими користувачами програми.

У списку виводяться:

  • - наявність електронного підпису або шифрування.
  • Найменування - повне найменування файлу. Значок поруч з найменуванням вказує тип файлу.
  • Автор - користувач, який додав версію файлу;
  • Дата зміни - дата зміни даних версії файлу (місцевий час);
  • Опис - докладний опис файлу.

За допомогою команди Ще - Додатково - Змінити форму можна додати відомості:

  • Розширення - формат файлу, поруч значок, який також показує формат;
  • Редагує - користувач, який зайняв файл для редагування і т.д.
  • Розмір (КБ) - розмір даних версії файлу;

Завантаження файлів

  • За допомогою команди Ще - Завантажити файли можна розмістити в списку файли з жорсткого або мережевого диска.
    • У Провіднику Windows, який відкрився, вкажіть шлях до файлу, натисніть Відкрити.
    • Заповніть необхідні для завантаження відомості.
  • Також можна перетягнути один або кілька файлів з Провідника Windows за допомогою миші. Якщо був доданий один файл, автоматично створюється і відкривається картка файлу. При додаванні декількох файлів картки файлів автоматично створюються, але не відкриваються.
