Призначений для моніторингу задач.

Відкривається по команді Задачі, що спливають на дату в списку Звіти по задачах розділу Бізнес-процеси.

Для контролю своїх задач виконавець може скористатися цим звітом. По кожному виконавцю виводиться зведення за кількістю задач з строками виконання, що спливає на зазначену в полі Крайній строк дату. Звіт дозволяє виконати розшифровку по Автору, Важливості, Задачі і Строку.

Можна встановити відбір по Важливості і Автору задачі.

По умовчанню в звіт потраплять задачі, крайній строк виконання яких закінчується завтра вранці.

Див. також:

  • Робота зі звітами;
  • Настройки звітів.
