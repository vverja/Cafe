Настройка списка

Форма предназначена для настройки списка (установки отборов, сортировок, условного оформления и группировок). Форма открывается по команде Еще - Настройка списка.

Настройка отчета:

  • Для настройки отбора в поле Отбор щелкните кнопку выбора. В открывшейся форме укажите поля и условия отбора.
  • Выбранные условия отбора показываются в поле Отбор в виде представления отбора, а в форме настройки ниже поля Группировка добавляются поля для управлением отбором.
  • Для настройки сортировки в поле Сортировка щелкните кнопку выбора. В открывшейся форме укажите поля, по которым будет выполнена сортировка, и направление сортировки.
  • Для применения настройки списка нажмите кнопку Завершить редактирование.
  • Для сброса какой-либо настройки в соответствующем поле нажмите кнопку Очистить.
  • Для восстановления стандартных настроек, заданных при конфигурировании, выполните команду Еще - Установить стандартные настройки.
  • Если установленные настройки предполагается использовать в дальнейшем, то следует использовать команду Сохранить настройки. Для использования сохраненной настройки следует выполнить команду Выбрать настройки.
