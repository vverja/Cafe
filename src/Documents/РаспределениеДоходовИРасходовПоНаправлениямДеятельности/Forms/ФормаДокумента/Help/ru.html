Распределение доходов и расходов по направлениям деятельности

Документ предназначен для распределения по направлениям деятельности доходов и расходов, зарегистрированных по статьям доходов и расходов.
Для каждой статьи в документе требуется указать один из способов распределения, указанный в Настройке распределения продаж по направлениям деятельности.
Данные о способе распределения можно изначально указать в самой статье доходов и расходов, тогда этот способ распределения будет подставлен в табличную часть документа автоматически.

Пример. Допустим, для некоторой статьи расходов определен способ распределения пропорционально валовой прибыли. При этом суммы после распределения должны быть отнесены на направление деятельности «НД1» и «НД2». Тогда для построения базы распределения будут выбраны суммы валовой прибыли, у которых направление деятельности равно «НД1» или «НД2», а значение организации совпадает с организацией из документа Распределение доходов и расходов для этой статьи (значение подразделения игнорируется). Пропорционально соотношению валовой прибыли по двум направлениям деятельности будет распределена сумма по статье доходов/расходов.

Если в текущем месяце постатейные доходы и расходы невозможно распределить по указанному способу (например, отсутствует выручка по направлению деятельности), они остаются нераспределенными. Эти расходы будут предложены для распределения в следующем месяце, когда для них может появиться база распределения.

Анализ распределения доходов и расходов на финансовый результат можно увидеть в отчете Доходы и расходы.

Заполнение информации о доходах и расходах


Для заполнения табличной части документа списком доходов и расходов, которые должны быть распределены на направления деятельности воспользуйтесь командой Заполнить по остаткам в реквизитах документа. Предусмотрена также возможность отдельного заполнения доходов и расходов. Для этого предназначены команды Заполнить по остаткам в панелях действий табличных частей документа на соответствующих закладках.

Применение команды Заполнить по остаткам после того как табличные части откорректированы вручную, приведет к потере корректировок (табличные части документа будут очищены и перезаполнены).

  • Заполнение табличной части на закладке Доходы выполняется по остаткам прочих доходов, отраженных в течение месяца.
  • Заполнение табличной части на закладке Расходы выполняется по остаткам прочих расходов, отраженных в течение месяца. Заполнение производится с отбором по организации и по статьям расходов, у которых задан вариант распределения На направления деятельности.


При необходимости можно откорректировать список доходов и расходов: удалить записи о доходах (расходах), изменить способ распределения по направлениям деятельности, указать другую статью расхода и т.д. Пользователь также может дополнить список доходов (расходов) списком произвольных доходов и расходов, которые не были ранее отражены в системе.

В случае возникновения погрешности округления при заполнении табличной части, программа автоматически добавит в табличную часть закладки Расходы, строку с предопределенной статьей расхода Погрешность расчета себестоимости. При этом автору документа необходимо вручную заполнить у такой строки колонку Подразделение. Сумма погрешности округления, после проведения документа, будет отнесена на эту статью и подразделение.

Добавление произвольного списка доходов (расходов)
  • Нажмите на кнопку Добавить на соответствующих закладках Доходы (Расходы) в табличной части документа.
  • Будет добавлена строка в табличной части документа.
  • В строке табличной части укажите сумму дохода (расхода), выберите статью дохода (расхода), укажите аналитику, соответствующую статье дохода (расхода), введите информацию о способе распределения доходов (расходов) по направлениям деятельности.

При проведении документа распределение по этим статьям доходов (расходов) будет производиться аналогично тем расходам, которые были оформлены документально.

Результат распределения по направлениям деятельности


При проведении документа Распределение доходов и расходов распределяются не только те доходи расходы, которые указаны в табличной части документа, но также выручка и себестоимость продаж товаров в течение того месяца, который указан в документе.

Результат распределения выручки, себестоимости продаж и прочих доходов и расходов можно посмотреть в отчете Анализ финансовых результатов.
