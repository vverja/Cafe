Звіт призначений для перегляду інформації про всіх користувачів або зовнішніх користувачів програми, включаючи настройки для входу (властивості користувача інформаційної бази).

Необхідність у звіті виникає, якщо потрібно виконати груповий аналіз настройок для входу (імені для входу, видів аутентифікації та інших).

Звіт відкривається зі списку Користувачі (Зовнішні користувачі) по команді Ще - Відомості про користувачів (про зовнішніх користувачів).

У звіті по умовчанню видаються відомості:

  • Повне ім'я користувача;
  • Вхід в програму дозволений - поле може приймати значення "Так" або залишається порожнім;
  • Ім'я (входу);
  • Аутентифікація стандартна - поле може приймати значення "Так" або залишається порожнім;
  • Аутентифікація ОС - поле може приймати значення "Так" або залишається порожнім;
  • Користувач ОС - ім'я, з яким користувач входить в ОС;
  • Аутентифікація OpenID - поле може приймати значення "Так" або залишається порожнім;
  • Режим запуску.

Для додавання відомостей натисніть кнопку Настройки, змініть настройки звіту.

Відбори користувачів

  • За допомогою кнопки Настройки можна відкрити настройки звіту. Для відбору користувачів за одним або кількома параметрами увімкніть відповідні прапорці.
  • Можливий відбір користувачів за параметрами:
    • Недійсні;
    • Відмічені для вилучення;
    • Без користувача інформаційної бази;
    • Вхід в програму дозволений;
    • Аутентифікація стандартна;
    • Аутентифікація ОС;
    • Аутентифікація OpenID.

Варіанти звіту

  • З допомогою кнопки Варіанти звіту можна відкрити список інших доступних варіантів звіту. Існує три варіанти звіту:
    • Відомості про користувачів - відкривається зі списку Користувачі;
    • Інформація про зовнішніх користувачів - відкривається зі списку Зовнішні користувачі;
    • Відомості про користувачів і зовнішніх користувачів - загальний звіт про всіх користувачів програми, в якому передбачено поле Це зовнішній користувач (приймає значення "Так" або залишається порожнім) для виділення і відбору зовнішніх користувачів.

Див. також:
