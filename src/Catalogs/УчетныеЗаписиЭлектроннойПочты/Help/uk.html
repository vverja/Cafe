Призначений для ведення облікових записів електронної пошти, які використовуються для отримання і надсилання електронних повідомлень.

Список можна відкрити відповідною командою в розділі Адміністрування - Органайзер.

У списку виводяться:

  • Найменування облікового запису;
  • Ім'я користувача - вказується ім'я відправника листів, воно може відрізнятися від імені користувача в програмі;
  • Адреса електронної пошти облікового запису у форматі name@server.net.

Для облікових записів, отримання і надсилання з яких виконується автоматично без участі користувача, повинен бути обов'язково введений пароль. Також для коректної роботи програми обов'язково повинен бути настроєний системний обліковий запис електронної пошти.

Системний обліковий запис електронної пошти

  • Програма поставляється з напередвизначеним обліковим записом - системної обліковим записом електронної пошти. Вона використовується по умовчанню для розсилки різних повідомлень з програми, наприклад, для автоматичної відправки повідомлень адміністратора по завершенню оновлення програми. Перед використанням цих можливостей необхідно настроїти системний обліковий запис.
  • Настройка системного облікового запису провадиться по відповідній команді в розділі НДІ і адміністрування - Органайзер. Також можна Змінити її в списку.

Введення облікового запису електронної пошти

  • При спробі збереження програма перевіряє введені облікові записи електронної пошти (намагається надіслати пробне повідомлення). Облікові записи, що не пройшли перевірку, не зберігаються.
  • Натисніть Створити, введіть необхідні поля:
    • Вкажіть Адресу електронної пошти облікового запису у форматі name@server.net;
    • Введіть Пароль облікового запису. Для системних облікових записів рекомендується запам'ятати пароль, для того щоб програма могла розсилати повідомлення від такої облікового запису без участі користувача. Проте, рішення про зберігання паролів в інформаційній базі приймається в кожному випадку індивідуально.
    • Встановіть з допомогою прапорців, як Використовувати обліковий запис:
      • Для відправки листів;
      • Для отримання листів;
    • Вкажіть Ім'я відправника (Ваше ім'я або назву організації);
    • За допомогою перемикача визначте спосіб створення облікового запису:
      • Визначити параметри підключення автоматично (рекомендується);
        • Натисніть Створити; Програма зробить спробу автоматичної настройки і перевірки облікового запису. Дотримуйтесь вказівок програми. При цьому якщо при автоматичній настройці стався збій, можна спробувати настроїти параметри вручну (див. нижче).
      • Настроїти параметри підключення вручну;

    • Натисніть Далі для заповнення решти параметрів, при цьому програма проставляє рекомендовані параметри для більшості поштових серверів. Їх потрібно тільки перевірити і спробувати зберегти обліковий запис ще раз.

Настройка облікового запису електронної пошти

  • Натисніть Змінити, за допомогою програми зробіть необхідні настройки.

Перевірка облікового запису електронної пошти

  • Виділіть обліковий запис у списку, натисніть Ще - Перевірити обліковий запис для перевірки введених параметрів облікового запису. При цьому надсилається стандартне поштове відправлення і перевіряється можливість з'єднання з сервером вхідних поштових повідомлень (якщо обліковий запис використовується для отримання пошти).
  • Якщо для облікового запису не прописаний пароль, програма його запитує. При необхідності введіть пароль.
