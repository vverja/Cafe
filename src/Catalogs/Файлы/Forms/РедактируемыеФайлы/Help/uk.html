У цьому списку відображаються тільки файли, зайняті Вами для редагування. Редагувати файли можна в списку файлів або в списку приєднаних файлів об'єктів програми.

Відкривається по команді Редаговані файли розділу Органайзер.

У списку виводяться відомості:

  • Файл - найменування файлу
  • Розміщення - папка, в якій знаходиться файл або об'єкт, до якого приєднаний файл;

Доступні дії:

  • Переглянути - відкрити файл для перегляду;
  • Закінчити редагування - зберігає версію файлу і звільняє файл для інших користувачів.
  • Відкрити картку - відкрити для перегляду відомостей про файл.

Див. також:

  • Загальна інформація про файли;
  • Робота з формами.
