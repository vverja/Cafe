Області зберігання

Довідник Області зберігання використовується для поділу складів або приміщень на області зберігання відповідно до різних умов зберігання.

Області зберігання використовуються тільки для тих складів і приміщень, на яких передбачено адресне зберігання товарів в комірках. Для комірок на складі (приміщенні) встановлений варіант використання Використовувати для зберігання залишків номенклатури.

Поділ складів на області зберігання визначається тими умовами зберігання, які необхідні для зберігання певних груп товарів. Наприклад, при зберіганні продуктів повинні бути різні умови зберігання для свіжих овочів, молочних продуктів і заморожених продуктів. Для кожної області зберігання на складі визначається список складських груп номенклатури і складських груп упаковок, які відповідають умовам, передбаченим для даної області зберігання. Для однієї і тієї ж складської групи номенклатури можна визначити кілька областей зберігання, які будуть розрізнятися своїм пріоритетом. Розподіл складських груп номенклатури і упаковок по областях зберігання визначається в регістрі відомостей Області зберігання номенклатури.

Приклад. Для молочних продуктів, ковбасних виробів і овочів передбачені різні області зберігання: Молоко, Ковбаси, Овочі. Для кожної групи найвищий пріоритет встановлюється для тієї області зберігання, де він краще зберігається. Тобто, для молочних продуктів найвищий пріоритет має область зберігання Молоко, а для ковбасних виробів - Ковбаси. У той же час, для цих груп продуктів можуть бути призначені і альтернативні області зберігання. Якщо молочні пакети не вдається розмістити в пріоритетній галузі зберігання, то їх можна покласти на час в область зберігання ковбаси, яка має для групи молочних продуктів більш низький пріоритет.

Для кожної області зберігання визначається список складських комірок, які відносяться до даної області зберігання. Комірка в області зберігання можуть використовуватися для зберігання тільки відокремлених товарів або будь-яких товарів відповідно до стратегій відбору та розміщення.
Ознака Комірки використовуються для зберігання відокремлених товарів, встановлюється для області зберігання, в якій розташовуються товари під конкретне призначення (відокремлене забезпечення).

Розподіл складів (приміщень) по областях зберігання використовується при реєстрації завдань на розміщення або відбір товарів по конкретних комірках, які належать до даної галузі зберігання. При цьому аналізується заповненість комірок в області зберігання, визначеної для груп номенклатури і пріоритетність області зберігання. Товар буде розміщуватися в ту область зберігання, яка йому призначена (найвищий пріоритет), а забиратися з тієї області зберігання, яка має найнижчий пріоритет (товар був покладений туди на час, оскільки іншого вільного місця не було і його необхідно якомога швидше звідти забрати).

Створення нової області зберігання

Створення нової області зберігання
  • У списку областей зберігання конкретного складу або приміщення натисніть на кнопку Створити.
  • Буде створена нова область зберігання.
  • Введіть найменування області зберігання.
  • У полі Опис вкажіть основне призначення даної області зберігання, умови зберігання товарів в даній області зберігання.
  • Вкажіть як визначатиметься монотоварність у сфері зберігання та пріоритети використання області зберігання при розміщенні та відборі товарів.
