Призначений для вибору документа Повідомлення SMS. Документ використовується для відправки повідомлення SMS одному або декільком адресатам.

Відкривається по кнопці після вибору типу даних Повідомлення SMS (наприклад, при заповненні поля Предмет в будь-якому документі взаємодій).

У списку виводиться:

  • Дата- дата створення документу;
  • Номер- номер присвоюється автоматично.

Вибір документа

  • Виділіть документ, натисніть Вибрати.
  • Також можна використовувати подвійне натиснення кнопки миші.

Відбір по періоду

  • За допомогою відповідної команди меню Ще можна Встановити інтервал дат.

Введення документа Повідомлення SMS

  • Натисніть Створити. Заповніть потрібні поля.

Створення взаємодій на підставі повідомлення SMS

  • Виділіть потрібний документ Повідомлення SMS. Натисніть Взаємодії, для того щоб:
    • Запланувати зустріч;
    • Запланувати взаємодію;
    • Написати SMS;
    • Написати електронного листа;
    • Зателефонувати.

Див. також:

  • Робота з формами.
