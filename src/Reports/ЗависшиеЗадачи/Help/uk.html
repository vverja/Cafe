Призначений для моніторингу задач і бізнес-процесів. У звіті виводяться "завислі" задачі, які не можуть бути виконані через те, що не заповнений список ролей і виконавців.

Якщо встановлений відбір по Координатору, то до звіту будуть виводитися тільки ті задачі, які призначені виконавцям в об'єктах адресації, за які відповідає вказаний координатор виконання задач. Можна встановити фільтр за Автором задач. Також передбачено відбір по Періоду.

Варіанти звіту

  • Для того щоб встановити групування списку завислих задач по об'єктах адресації, виберіть варіант звіту Зведення по завислих задачам. У звіті виводяться основний і додатковий об'єкти адресації, відповідальний, задачі. Для скорочення виведеної в звіті інформації зніміть прапорець Докладно, в цьому випадку не виводяться найменування задач.
    • Можна відібрати задачі по Відповідальному і по Автору.
  • Варіант звіту Завислі задачі по об'єктах адресації дозволяє подивитися кількість завислих задач в розрізі ролей виконавців і об'єктів адресації. Виводиться основний і додатковий об'єкт адресації, задача.
    • Можна встановити відбір завислих задач по конкретному виконавцю ролі, наприклад, по Координатору.
  • Варіант звіту Завислі задачі по відповідальним дозволяє включити групування за ролями виконавців, наприклад, за Координатору.
  • Для перегляду одночасно як завислих, так і прострочених задач призначений варіант звіту Прострочені задачі.
    • Завислі задачі в звіті виділяються червоним кольором (для цього увімкніть прапорець Виділяти задачі без призначених виконавців в настройках звіту).
    • Можна встановити відбір по конкретному виконавцю ролі, наприклад, по Координатору.
