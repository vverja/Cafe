Призначений для призначення користувачів на роль виконавця.

Відкривається по команді Призначити виконавців на роль зі списку Ролі і виконавці задач.

У нижній частині списку ведеться підрахунок призначених виконавців.

Додавання виконавця

  • Виконавцями деяких бізнес-процесів можуть бути і зовнішні користувачі (наприклад, партнери або контактні особи партнерів).
  • Для вибору одного виконавця натисніть Додати.
  • З допомогою кнопки виберіть тип списку - Зовнішні користувачі або Користувачі.
  • Виберіть виконавця з відповідного списку.

Підбір виконавців

  • Для підбору одного або декількох виконавців натисніть Підібрати, виберіть потрібних користувачів зі списку.
