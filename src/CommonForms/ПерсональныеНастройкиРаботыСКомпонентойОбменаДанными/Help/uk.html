Настройка компоненти обміну даними з КПК

У формі можна визначити настройки компоненти для поточного користувача.

Підключення компоненти обміну даними

Підключення компоненти обміну даними

Для використання компонента обміну даними вона повинна бути підключена. Підключення можна виконувати один раз під час сеансу роботи. Також можна встановити прапор Підключати компоненту обміну даними при старті системи, в цьому випадку компонента буде автоматично підключатися кожен раз при початку роботи з системою.
