Варіанти аналізу цільових показників

У довіднику зберігається інформація про варіанти аналізу, які використовуються для аналізу цільових показників. Для кожної цілі використовується свій список варіантів аналізу цільових показників.

У формі списку користувач, який має відповідні права, може додавати не тільки нові варіанти аналізу цільових показників, але і вводити інформацію про нові цілі в структуру цілей. Для додаткової деталізації даних можуть бути також використані різні варіанти звітів конфігурації.

Список варіантів аналізу цільових показників розділений на дві частини:

  • В лівій частині списку показується структура цілей.
  • В правій частині для кожної цілі (цільового показника) показується список варіантів, призначених для аналізу вибраного цільового показника.

У варіанті аналізу заповнюється інформація про тип аналізу, період аналізу, періодичності контролю. Результати аналізу можуть бути представлені в короткому виді, у виді діаграми або таблиці. Для кожного варіанту аналізу можуть бути задані цільові значення, з допомогою яких буде здійснюватися контроль цільового показника. Передбачена можливість прогнозування даних, а також порівняння з аналогічними даними минулих періодів. Для показника можна налаштувати список звітів для розшифровки даних, а також визначити список тих користувачів, яким буде доступний перегляд даного показника.
