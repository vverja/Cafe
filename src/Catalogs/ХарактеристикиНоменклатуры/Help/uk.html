Характеристики номенклатури

Довідник призначений для реєстрації переліку характеристик, з допомогою яких може здійснюватися додатковий кількісний облік номенклатури.
Для різних груп (видів) номенклатури може застосовуватися різний набір видів властивостей характеристик (колір, розмір - для взуття; виробник, матеріал обробки - для меблів).

Доступність довідника визначається функціональною опцією Характеристики номенклатури в розділі НДІ і адміністрування - Номенклатура - Розрізи обліку.

Список характеристик може бути призначений як для кожної позиції номенклатури, так і може бути призначений загальний список характеристик для всіх позицій номенклатури, які належать одному виду номенклатури.

Для кожного виду номенклатури може бути визначений свій набір властивостей, в якому будуть вказані ті додаткові властивості і додаткові відомості, які використовуються для характеристик даного виду номенклатури.
