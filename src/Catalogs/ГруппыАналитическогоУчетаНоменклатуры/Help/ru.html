Группы аналитического учета номенклатуры

Справочник предназначен для экономической классификации объектов по однородным группам. Данные группы могут использоваться для объединения номенклатуры, имеющей одинаковые экономические характеристики. Например: покупные товары, материалы, тара и т.п.

Группы аналитического учета номенклатуры могут использоваться:

  • как разрез учета при отражении движения номенклатурных позиций в отчетности;
  • для разделения прямых производственных затрат по группам аналитического учета.
