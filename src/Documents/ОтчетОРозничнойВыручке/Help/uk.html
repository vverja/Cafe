Документ "Звіт про роздрібний виторг (Z-звіт)"

Документ призначений для щоденного відображення інформації про обсяги продажу в автономних (неавтоматизованих) ККМ в сумовому вираженні. Документ фіксує інформацію, аналогічну тій, що міститься в Z-звітах фіскальних реєстраторів.

Ця інформація використовується для відображення доходів платників єдиного податку та для відображення сум податкових зобов'язань з роздрібних продажів для платників ПДВ.
