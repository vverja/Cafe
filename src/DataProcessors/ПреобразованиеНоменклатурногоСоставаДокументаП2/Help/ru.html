Преобразование номенклатурного состава документа "Приложение 2 к налоговой накладной"

Обработка преобразует Приложения 2, выписанные в ИБ автоматически, в соответствии с действующим Порядком выписки НН.

В П2 осуществляется идентификация корректировок следующих видов:

- полный возврат по строке НН: указывается Причина корректировки «Повернення товару або авансових платежів», код причины 103. Номера групп корректировки присваиваются автоматически;

- частичный возврат (уменьшение количества по другой причине без изменения цены) по строке НН: формируется две строки, в первой сторнируется все количество по строке НН, во второй указывается количество, оставшееся в НН после уменьшения; указывается Причина корректировки «Зміна кількості», код причины 102. Номера групп корректировки присваиваются автоматически;

- изменение номенклатуры: сторнирующие строки и строки-замены формируются автоматически на этапе формирования налоговых документов. Пользователь самостоятельно указывает номер группы корректировки для каждой сторнирующей строки и заменящих ее строк. Можно не указывать вручную номера групп корректировки для групп, в которых совпадает номенклатура сторнирующей строки и строк-замен - в случаях изменения цены/цени и количества.

Обработка при необходимости преобразует сторнирующую строку в две строки, по правилам, описанным для частичного возврата, и указывает для всех строк Причину корректировки «Зміна номенклатури», код причины 104.

Как «Зміна номенклатури» классифицируется в том числе изменение цены номенклатуры без изменения количества. Такая классификация не противоречит действующему Порядку заполнения НН, и рекомендуется ее не изменять. Пользователь может вручную указать для соответствующих строк причину 101 «Зміна ціни», однако при этом Д2 отмечается как измененная вручную - переводится в режим автокорректировки «Не изменять». Это может помешать автоматическому преобразованию следующих П2 к той же НН.

- прочие случаи: пользователь может самостоятельно сформировать строки, указать для них коды причины и номера групп корректировки;

В случаях, когда строка исходного П2 разбивается на две строки, новые строки с положительным количеством нумеруются начиная с номера, указанного пользователем. По умолчанию нумерация начинается с номера, следующего за номером последней строки НН.

При формировании П2 принимаются во внимание выписанные ранее П2 к той же НН, при словии, что такие П2

- сформированы по новым правилам

- проведены
