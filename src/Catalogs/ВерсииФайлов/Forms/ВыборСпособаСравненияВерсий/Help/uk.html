Вибір програми для порівняння версій файлів. У програмі передбачена можливість порівнювати версії файлів формату HTML, TXT, RTF, DOC, DOCX, ODT.

Дозволяє задати текстовий редактор, який буде використовуватися для порівняння версій файлів між собою. Викликається при виконанні команди Показати відмінності зі списку версій файлу, якщо текстовий редактор для порівняння не заданий по умовчанню.

Вибір способу порівняння версій

  • У полі Порівнювати версії файлів за допомогою виберіть програму для порівняння файлів зі списку: Для вибору можливі Microcoft Word або OpenOffice.org Writer.
  • Вибір запам'ятовується в персональних настройках програми.

Установка настройки по умовчанню

  • У персональних настройках програми (вкладка Робота з файлами) в поле Порівнювати версії за допомогою необхідно задати текстовий редактор для порівняння версій. Для вибору можливі Microcoft Word або OpenOffice.org Writer. Ця настройка по умовчанню не встановлюється автоматично.
