Предназначена для ввода адреса по шаблону.

Открывается по кнопке  Выбрать из поля Адрес в элементах списков.

  • В поле Страна выберите страну адреса. Страна "Украина" выбрана программой по умолчанию, при этом поле может быть недоступным в зависимости от настроек. Для того чтобы изменить страну, нажмите  Выбрать. 
  • Поле Индекс предназначено для ввода почтового индекса, соответствующего данному адресу.
  • Поле Город, населенный пункт предназначено для ввода региона, города и т.п. При необходимости можно ввести части адреса детально, нажав  Выбрать. В открывшемся шаблоне введите данные населенного пункта.
  • Поле Улица предназначено для ввода улицы адреса.
  • Поля Дом, Корпус, Квартира позволяют ввести соответствующие части адреса. Можно изменить эти варианты, выбрав дополнительные значения из выпадающего списка.
  • Поле Представление адреса при заполнении адреса по шаблону недоступно, заполняется программой автоматически по мере заполнения полей адреса.
  • Поле Комментарий позволяет указать произвольный комментарий к адресу, который не будет включаться в его представление и выводится на печать.

История изменений

  • В некоторых списках можно просмотреть историю изменения адреса с помощью команды Еще - История изменений.

Адрес на Google Maps

  • С помощью этой команды любой адрес, введенный в программе, можно посмотреть на картах.

Ввод адреса в свободной форме

  • Воспользуйтесь командой Ввести адрес в свободной форме меню Еще, для того чтобы при вводе адреса по шаблону ввести заведомо некорректный адрес (например, с товаром на накладной неправильный адрес поставщика, его надо временно зафиксировать для печати, но в дальнейшем его использовать нельзя). Такой адрес используется для хранения, поэтому можно рекомендовать использовать его только для единичных случаев.
  • Выводится шаблон для ввода адреса, но все его поля, кроме полей Представление адреса и Комментарий, становятся неактивными. Заполните поле Представление адреса. При переходе к полю Комментарий или нажатии кнопки ОК заполняются те поля шаблона, которые программе удалось распознать в строке адреса (т.е. в этих полях нет ошибок). Проверка заполнения адреса в данном случае отключена.
  • Для подтверждения адреса нажмите ОК. Программой будет зафиксировано, что адрес заполнен в свободной форме и на просмотр адрес будет выводиться  в таком виде.

Очистка адреса

  • С помощью команды меню Еще - Очистить адрес  можно стереть неправильно введенный адрес.
