Настройка учета НДС

В номенклатурным карточках должна быть указана Ставка НДС по умолчанию.
Для импортных и подакцизных товаров должен быть включен учет По кодам УКТВЭД и номерам ГТД. Предприятиям, не являющимся импортерами и в то же время торгующим импортными товарами, рекомендуется заполнить Код УКТВЭД и номер ГТД по умолчанию, указав основной код УКТВЭД товара и пустой номер ГТД.

Для Контрагентов предприятия долно быть указано, являются ли они плательщиками НДС. Для покупателей-плательщиков НДС рекомендуется также указать ИНН.

В Соглашениях с клиентами и поставщиками выбрать Ставку НДС по умолчанию. Эта ставка будет подставляться в первичные документы продаж и покупок по соглашениям.

  • Возможность учета материальных ценностей в разрезе налоговых назначений определяется функциональной опцией Раздельный учет товаров по налогообложению НДС в разделе Финансовый результат и контроллинг - Учет товаров.
  • В том же разделе должна быть включена функциональная опция Партионный учет.

Текущие операции по учету НДС

Налоговые обязательства по операциям поставки

В документах продажи указываются ставки и суммы НДС.
В документах оплаты ставки и суммы НДС не указываются.

Выписка Налоговых накладных по операциям поставки производится автоматически при выполнении регламентного задания Формирование исходящих налоговых документов до текущей даты по заданному расписанию либо в журнале Регламентированный учет - Исходящие налоговые документы на закладке Формирование налоговых документов.

Автоматически сформированные документы должны быть проверены, для них должен быть установлен статус Проверен.

Налоговый кредит по операциям приобретения

В документах приобретения указываются ставки и суммы НДС.
В документах оплаты ставки и суммы НДС не указываются.

Регистрация входящих налоговых документов поставщиков осуществляется в журнале Регламентированный учет - Входящие налоговые документы.

Учет налоговых назначений запасов

Налоговые назначения указываются при приобретении запасов, работ (услуг) и капитальных инвестиций.
