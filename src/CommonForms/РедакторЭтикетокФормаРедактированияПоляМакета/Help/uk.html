Редактор етикеток форма редагування поля макету

Форма "Редактор етикеток форма редагування поля макету" служить для вказівки значень властивостей конкретного поля. При переході до додавання або редагування поля макету етикетки буде відкрита форма редагування поля, реквізити якої можна розділити на три групи: загальні, приватні, індивідуальні (тільки для одного з типів поля).

До загальних реквізитів входять:

1) Тип поля- тип вмісту поля (текст, штрихкод, картинка або додатково (користувацькі дані)). Залежно від значення реквізиту "Тип поля" склад доступних реквізитів змінюється;

2) Тип заповнення- тип заповнення поля даних ("Параметр"- дані поля конкретної одиниці етикетки задається значенням заповнення, "Постійне значення"- значення однаково для всіх етикеток)

3) Значення- ім'я одного з доступних полів або строкова константа, залежно від значення реквізиту " Тип заповнення"

4) Значення по умовчанню- значення, яке буде підставлено в якості значення поля, якщо воно не було вказано.

Дві інші групи реквізитів впливають на візуальне представлення етикетки, отже, не відносяться до полів з типом "Додатково".

Такі реквізити також відносяться до приватних:

1) Зліва- порядковий номер лівого стовпця, з якого починається поле, щодо лівого краю етикетки. По суті, це відступ зліва в мм- 1 мм;

2) Верх-порядковий номер верхньої строки, з якої починається поле, щодо верхнього краю етикетки. По суті, це відступ зверху в мм-1 мм;

3) Справа- порядковий номер останнього стовпця, в якому закінчується поле, щодо лівого краю етикетки;

4) Низ-порядковий номер останнього рядка, в якому закінчується поле, щодо верхнього краю етикетки;

5) Орієнтація- поворот вмісту поля проти годинникової стрілки. Обробка даного реквізиту подібна властивості поворот тексту в MS Excel.

6) Рамка (не підтримується для типу поля "Штрихкод"):

a) Зверху - наявність рамки зверху;

b) Зліва- наявність рамки зліва;

c) Знизу- наявність рамки знизу;

d) Справа- наявність рамки справа;

e) Тип рамки- тип рамки (точкова, суцільна, подвійна, пунктир);

f) Товщина рамки.

Склад індивідуальних реквізитів залежить від типу поля.

У випадку " Текст":

· Шрифт- ім'я шрифту, розмір, стиль, тощо;

· Багаторядковість- прапор багаторядковості тексту (аналогічно переносу за словами);

· Горизонтальне положення- вирівнювання тексту по горизонталі (зліва, центр, справа);

· Вертикальне положення- вирівнювання тексту по вертикалі (верх, центр, низ);

У разі "Штрихкод":

· Тип штрихкоду- тип штрихкоду (EAN8, EAN13, тощо);

· Наявність підпису до штрихкоду- наявність під штрихкодом рядка-підпису так званої " HRI "(Human Readable Interpretation);

· Виводити контрольний символ-прапор, що відповідає за виведення в підписі до штрихкоду контрольного символу

· Розмір шрифту підпису-власне, розмір шрифту підпису до штрихкоду.

У разі, коли значення реквізиту "Тип заповнення" дорівнює "Параметр", для поля "Значення" стає доступною кнопка вибору параметра зі списку, для чого відкривається відповідна форма.
