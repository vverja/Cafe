Вид номенклатури

Довідник є класифікатором, призначеним для об'єднання номенклатури за загальними ознаками обліку. Фактично види номенклатури - це асортиментні групи товарів, що продаються. При продажу товарів у роздріб (зі складу типу Роздрібний) відповідно до видів номенклатури можуть бути створені товарні категорії та визначені квоти асортименту за кожною товарною категорією відповідно до формату магазину.

Основна інформації для виду номенклатури

У формі виду номенклатури вводиться основна інформація про вид номенклатури.

  • Група видів номенклатури. Група, в яку входить даний вид номенклатури. Дана можливість використовується в тому випадку, якщо на підприємстві використовується багато різних видів номенклатури, які необхідно об'єднувати додатково в групи.
  • Найменування. Назва виду номенклатури.
  • Тип номенклатури. Вказується тип, до якого відноситься номенклатура даного виду. Можна вибрати одне зі значень: Товар, Послуга, Робота, Тара.

Інформація про тару доступна тільки в тому випадку, якщо включена функціональна опція Багатооборотна тара (НДІ та адміністрування - Номенклатура - Розрізи обліку).

Додатково зазначається інформація про використання характеристик для виду номенклатури і вибирається політика обліку серій, якщо для даного виду номенклатури ведеться серійний облік (облік по серійних номерах і строках придатності).

Варіант використання характеристик для виду номенклатури

Варіант використання характеристик для виду номенклатури вказується в полі Використовувати характеристики.

Цю можливість можна використовувати тільки в тому випадку, якщо в розділі НДІ і адміністрування - Номенклатура - Розрізи обліку включена функціональна опція Характеристики номенклатури.

Після запису інформації про новий вид номенклатури змінити варіант використання характеристики не можна.

Для виду номенклатури можна вказати наступні варіанти використання характеристик:

  • Не використовувати. Для даного виду номенклатури не ведеться облік в розрізі характеристик.
  • Використовувати загальні для виду номенклатури. Призначається загальний список характеристик, який використовується всіма позиціями номенклатури, які належать даному виду.
  • Використовувати індивідуальні для номенклатури. Для кожної позиції номенклатури призначається індивідуальний список характеристик.

Визначення загального списку характеристик


Для тих видів номенклатури, для яких поле Використовувати характеристики встановлено у значення Загальні для виду номенклатури, можна вказати список характеристик, який буде використовуватися для всіх тих номенклатурних позицій, які відносяться до даного виду.

Приклад. Товари виду Взуття характеризується розміром і кольором моделі взуття. У список характеристик додається всі можливі поєднання кольору і розміру взуття: Розмір 45, Колір білий, Розмір 37, Колір білий, Розмір 37, Колір чорний і т.д.

Порядок введення списку характеристик для виду номенклатури.

  • Для редагування набору властивостей характеристик для даного виду номенклатури необхідно перейти на закладку Додаткові реквізити.
  • Відкриється вікно для редагування набору властивостей номенклатури і характеристик даного виду номенклатури.
  • Додайте в список Додаткові реквізити характеристик додаткові реквізити, які необхідні для створення характеристики товару даного виду (наприклад, колір, розмір і т.д.).
  • Тут же при натисканні на кнопку Загальні можна відредагувати список реквізитів і відомостей, загальних для всього довідника Характеристики номенклатури.
  • Збережіть зміни за допомогою команди Зберегти.
  • Перейдіть на закладку Основне, натисніть на гіперпосилання Загальні характеристики номенклатури.
  • У списку характеристик натисніть на кнопку Створити.
  • У новій формі характеристики вже будуть додані додаткові реквізити, які описують характеристику даного виду.
  • Вкажіть значення цих реквізитів.
  • Заповніть Робоче найменування та Найменування для друку.
  • Натисніть на кнопку Провести та закрити.
  • Інформація про характеристику запишеться в список характеристик для виду номенклатури.
  • Аналогічним чином додайте інформацію про інші характеристики, що використовуються для номенклатури даного виду.

При введенні інформації про товар, який належить даному виду, його характеристику можна буде вибрати зі списку тих загальних характеристик, які визначені для даного виду.

Додаткові реквізити та відомості для нового виду номенклатури

Додаткові реквізити та відомості створюються на відповідних сторінках. Окремо створюються дод.реквізити та відомості для номенклатури, характеристик і серій.

  • Якщо додатковий реквізит (додаткова відомість) повинен показуватися для всіх позицій номенклатури, то він додається через список по команді Додати - Загальний для всієї номенклатури.
  • Якщо додатковий реквізит (додаткова відомість) повинен показуватися тільки для номенклатурних позицій даного виду, то він додається за допомогою команди Додати - Новий для виду.
  • Додати додатковий реквізит (додаткова відомість) з іншого набору можна за допомогою команди Додати - З іншого набору.

Після додавання додаткових реквізитів натисніть на сторінці Настройки створення з'явиться список додаткових реквізитів. Визначте для кожного реквізиту параметри контролю заповнення та унікальності.

  • Інформація про додаткові реквізити автоматично з'явиться у формі номенклатурної позиції на сторінці Опис.
  • Список додаткових відомостей заповнюється в окремому діалоговому вікні (Ще - Додаткові відомості).

Настройка швидкого відбору в списках і при підборі номенклатури
  • У картці виду номенклатури перейдіть на закладку Фільтр за властивостями.
  • Списки реквізитів номенклатури та реквізитів характеристик, які застосовуються для швидкого відбору.
  • За допомогою кнопки Додати заповніть списки реквізитами, за якими буде здійснюватися швидкий відбір.

В якості параметрів швидкого відбору не можна використовувати додаткові реквізити складеного типу. Такі реквізити не будуть показані в списку для вибору параметрів швидкого відбору.

Шаблони найменувань.

Шаблони найменувань дозволяють створити правило, за яким найменування номенклатури буде формуватися автоматично, з урахуванням її реквізитів. Також за шаблоном може бути перезаповнено найменування раніше створеної номенклатури. При необхідності, можна заборонити відхилення найменування номенклатури від заданого шаблону. Задати шаблони можна на сторінці Шаблони найменувань за допомогою форми Редагування формули, яка відкривається при виконанні команди Змінити.

Для зручного редагування шаблону рекомендується використовувати редактор формул, що запускається по гіперпосиланню Змінити. В редакторі можна вибрати будь-які реквізити (у тому числі і додаткові реквізити) номенклатури, які повинні включатися в її найменування. Також в найменуванні може бути довільний текст, який слід писати в лапках «». Для з'єднання складових частин найменування використовується знак плюс «+». Шаблони найменувань задаються окремо для найменування номенклатури, характеристики і для варіантів найменування, використовуваних при друці документів.

Приклад. Для обліку кабелів слід формувати їх назви за виробником і технічними параметрами, зазначеними в дод.реквізитах. Слід задати наступний шаблон найменування:
"Кабель NYM (" + [Виробник] + ") " + [Число жил] + "х" + [Переріз (кв.мм)]
За цим шаблоном буде формуватися назви кабелів на зразок наступного: Кабель NYM (Севкабель) 3х1,5

Для кожного шаблону можна налаштувати обов'язковість його використання за допомогою прапорців Заборонити редагування робочого найменування в картці номенклатури і Заборонити редагування найменування для друку в картці номенклатури.

  • Якщо прапорець встановлений, то найменування в картці номенклатури або характеристики стає недоступним для редагування і переформовується при кожному записі номенклатури або характеристики.
  • Якщо прапорець не встановлений, то заповнити найменування за шаблоном можна за допомогою команди Заповнити. При цьому якщо найменування не заповнене, то воно формується при записі номенклатури автоматично.

Також можна налаштувати шаблон для робочого найменування серії. Робоче найменування серії буде формуватися автоматично, за вказаним шаблоном, при записі серії.
