Товарные ограничения

Метод обеспечения определяет какие потребности в номенклатуре будут обеспечиваться.

Метод Заказ под заказ означает, что автоматически к заказу будет предложено только количество номенклатуры, необходимое для отгрузки по заказам.

Обеспечиваются не только потребности по заказам клиентов, но и потребности по заказам на перемещение, заказам на сборку/разборку, заказам на внутреннее потребление.

Метод Заказ под заказ может использоваться для обеспечения потребностей в товарах и работах. Прочие методы используются только для обеспечения потребностей в товарах.
