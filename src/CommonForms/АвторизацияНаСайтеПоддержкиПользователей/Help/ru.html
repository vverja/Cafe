Для доступа к онлайн-ресурсам необходимо ввести регистрационные данные, полученные на сайте поддержки пользователей:

  • Введите Логин;
  • Введите Пароль.
  • Для того чтобы больше не вводить эти данные в дальнейшем, включите флажок Запомнить пароль.
  • Для того чтобы получить доступ к онлайн-ресурсам, щелкните по ссылке Перейти к регистрации на сайте поддержки пользователей.
  • Для авторизации на сайте поддержки и возврата в программу нажмите Продолжить.
