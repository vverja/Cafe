Призначений для отримання звіту за різними типами даних програми.

Відкривається з розділу Адміністрування - Друковані форми, звіти та обробки по команді Універсальний звіт.

Дозволяє відображати дані документів, списків, задач, бізнес-процесів, планів видів розрахунку та їх табличних частин, регістрів накопичення, регістрів відомостей, регістрів розрахунку, регістрів бухгалтерії та їх віртуальних таблиць.

Це зручно, якщо необхідно отримати дані, але налаштовувати спеціальний звіт для їх отримання з яких-небудь причин неможливо.

Основні настройки звіту виведені в шапку.

Вибір даних для отримання звіту

  • Виберіть тип даних зі списку, наприклад, Регістр накопичення.
  • Залежно від типу даних, можливо, доведеться вказати додаткові відомості для отримання звіту. Наприклад, для регістру накопичення потрібно вибрати Період.
  • Виберіть зі списку вид даних, наприклад, вид регістру накопичення - Демо: Обороти по рахунках на оплату.
  • Вкажіть, які дані видати, наприклад, для регістру накопичення Демо: Обороти за рахунками на оплату можна включити до звіту Основні дані або Обороти.

Формування звіту

  • У разі потреби можна поміняти настройки звіту за допомогою кнопки Настройки.
  • Після вказівки всіх даних натисніть Сформувати.
