Форма настройки обработчика драйвера "ArtSoft: Универсальный драйвер регистраторов"

Форма предназначена для настройки параметров работы фискальных регистраторов, поддерживаемых драйвером «АртСофт: Универсальный драйвер фискальных регистраторов». Форма позволяет настраивать следующие параметры устройства:

  • Модель устройства
    Модель подключаемого фискального устройства и тип протокола подключения.
  • Порт
    Номер последовательного порта, к которому подключен фискальный регистратор.
  • Скорость
    Скорость обмена данными с фискальным регистратором.
  • IP адрес
    IP адрес для протокола подключения TCP/IP.
  • Порт ФР
    Порт подключения для протокола подключения TCP/IP.
  • Налоговые ставки
    Данная настройка предназначена для указания номера используемой налоговой ставки  фискального регистратора для типа налога, заданного для Номенклатуры. Перед использованием налоговая ставка должна быть запрограммирована Центром сервисного обслуживания в фискальном регистраторе. Количество налоговых ставок зависит от модели и версии регистратора и описано в документации к фискальному устройству.

После изменения значений параметров следует нажать кнопку "ОК" для того, чтобы сохранить внесённые изменения, или кнопку "Отмена" для того, чтобы отказаться от их сохранения.
