Начисление и списание бонусных баллов

Документ предназначен для начисления и списания бонусных баллов в рамках бонусной программы лояльности.

Документ может создаваться:

  • Вручную - при вводе остатков по бонусной программе и прочих случаев единовременного начисления бонусов.
  • Автоматически - при помощи регламентного задания по начислению и списанию бонусных баллов по правилам, зарегистрированным в справочнике Правила начисления и списания бонусных баллов.


Остатки бонусных баллов регистрируются по каждой Бонусной программе лояльности.
