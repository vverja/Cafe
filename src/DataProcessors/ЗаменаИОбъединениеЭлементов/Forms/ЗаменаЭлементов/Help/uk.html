Дозволяє замінити один або кілька елементів на обраний елемент.

Така можливість необхідна при вилученні елементів, наприклад, тому що для остаточного вилучення елемента необхідно позбутися від посилань на нього.

Відкривається по кнопці Замінити і вилучити зі списків програми. Також можна скористатися командою Замінити у всіх місцях використання з вікна Вилучення відмічених об'єктів.

Можна автоматично замінити всі посилання на елементи,що вилучаються, посиланнями на вибраний елемент. Всі зазначені елементи будуть відмічені на вилучення або вилучені безпосередньо, якщо користувач має адміністративні права.

Виводяться всі елементи цього списку, у тому числі відмічені на вилучення.

Перегляд елемента списку

  • Виконайте команду Змінити контекстного меню по правій кнопці миші або меню Ще для перегляду обраного елемента.

Робота з документами

  • Для документів передбачено відбір за періодом. Якщо документів в списку багато, можна скористатися цим відбором. Для цього виконайте команду Встановити період контекстного меню по правій кнопці миші або меню Ще.
  • Також для документів передбачені команди Ще - Провести (Скасування проведення).

Заміна елементів

  • Виберіть елемент, на який потрібно замінити відмічені елементи.
  • Внизу списку в інформаційному рядку вказуються реквізити елемента, який обраний для заміни. Також для користувачів з адміністративними правами надається можливість вибору. Натисніть на посилання, і вкажіть, що замінні елементи будуть:
    • відмічені на вилучення;
    • вилучені безповоротно.
  • Натисніть Замінити для заміни всіх посилань.

У разі виникнення помилок у процесі обробки елементів буде виведено список необроблених об'єктів. У нижній частині списку виводиться опис причини неможливості обробки за відміченим елементом. У списку можна:

  • Відкрити елемент для перегляду, натиснувши Змінити або подвійним натисненням кнопки миші;
  • Спробувати ще раз замінити елемент, натиснувши Повторити заміну;
  • Натиснути Назад і підібрати для заміни інший елемент;
  • Натиснути Закрити для повернення в програму.
