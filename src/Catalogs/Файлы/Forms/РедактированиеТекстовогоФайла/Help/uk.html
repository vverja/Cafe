У вбудованому редакторі текстових файлів можна переглянути і відредагувати файл, змінити кодування, відкрити дерево процесів і завдань, в яких задіяний файл, порівняти з вихідним варіантом в процесі редагування, при необхідності відкрити за допомогою стандартного додатка.

Відкривається зі списку Файли за командою Відкрити або Редагувати файл, в разі якщо в персональних настройках програми є відповідна настройка.

По умовчанню у вбудованому редакторі відкриваються текстові файли з розширенням TXT, INI, XML. Ви можете доповнити список розширень файлів у персональних настройках програми. На закладці Робота з файлами в таблиці Налаштування відкриття файлів можна вказати Розширення файлів для роботи у вбудованому редакторі (перераховуються через пробіл). Також можна вказати Спосіб відкриття - Стандартною прикладною програмою або У вбудованому редакторі (установка по умовчанню).

Редагування файлу

  • Якщо файл відкритий в режимі перегляду, кнопки Записати і закрити і Записати недоступні. Натисніть Редагувати, для того щоб змінити файл.
  • Принцип роботи вбудованого редактора текстових файлів такий же, як і у зовнішнього додатку. Файл копіюється на локальний комп'ютер і позначається як зайнятий в інформаційній базі. До тих пір поки ви не натиснете Ще - Закінчити редагування, створена версія файлу не буде доступна іншим користувачам.
  • Після закінчення редагування файлу натисніть Ще - Закінчити редагування, для того щоб зняти позначку зайнятості для редагування. Залежно від настройок команда також зберігає версію файлу або оновлює файл у сховищі. По умовчанню створюється нова версія файлу. Для зміни настройок під час створення версії зніміть відповідний прапорець.

Перегляд задач

  • З допомогою команди Ще - Задачі можна переглянути список задач, які були створені на підставі даного файлу.

Перегляд картки файлу

Натисніть Ще - Відкрити картку, для того щоб переглянути Відомості про файл.

Порівняння версій

  • Натисніть Ще - Показати відмінності, для того щоб зробити порівняння версій файлів. Якщо програма для порівняння версій не була обрана заздалегідь у персональних настройках програми (вкладка Робота з файлами), пропонується її вибрати. Доступні Microsoft Word або OpenOffice.org Writer.

Збереження файлу

  • Натисніть Ще - Зберегти як, для того щоб зберегти файл в зазначений каталог на жорсткому або мережевому диску.

Відкриття файлу в стандартній програмі

  • Натисніть Ще - Відкрити стандартним додатком, для того щоб перейти в зовнішній додаток (Блокнот Windows) для редагування файлу.
  • Якщо необхідно, щоб текстові файли з певними розширеннями по умовчанню відкривалися в зовнішньому додатку, внесіть зміни в персональні настройки програми. Для цього:
    • У персональних настройках програми перейдіть на закладку Робота з файлами.
    • Змініть Настройки відкриття файлів. У колонці Спосіб відкриття виберіть Стандартним додатком.

Зміна кодування текстового файлу

  • Текстові файли можуть бути збережені в різних кодуваннях. Основними кодуваннями для роботи з текстовими файлами російською мовою є:
    • IBM866 (Кирилиця DOS),
    • KOI8-R (Кирилиця KOI8-R),
    • Windows-1251 (Кирилиця Windows.
  • Якщо після відкриття текстового файлу замість тексту видно набір безглуздих знаків, виконайте команду Ще - Змінити кодування, виберіть інше кодування зі списку.
